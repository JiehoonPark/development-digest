## 핵심 개념

> [!abstract]
> LLM 활용 시 복잡한 언어보다 단순하고 검증된 '지루한' 언어를 선택하는 것이 더 효과적이라는 제안입니다.

## 상세 내용

- LLM의 성능은 언어의 복잡도보다 명확성과 일관성에 더 영향을 받습니다.
- Python, Go 같은 주류 언어가 LLM 학습 데이터가 풍부해 더 나은 결과를 제공합니다.

> [!tip] 왜 중요한가
> LLM을 활용한 개발 시 언어 선택이 코드 생성 품질과 생산성에 직접적인 영향을 미칩니다.
