## 핵심 개념

> [!abstract]
> Kolmogorov-Arnold Network(KAN) 아키텍처를 FPGA에 구현하여 초고속 추론과 온라인 학습을 가능하게 하는 하드웨어 설계에 관한 논문입니다. 고정소수점 양자화와 룩업테이블 신경망(LUT-NN)을 활용하여 나노초 수준의 극저지연 추론을 달성합니다.

## 상세 내용

- FPGA의 룩업테이블(LUT)을 신경망 기본 단위로 사용하여 GPU보다 극저지연(나노초 수준) 성능을 제공합니다.
- 고정소수점 양자화를 통해 실수 연산을 이진 연산으로 변환하여 하드웨어 효율성을 극대화합니다.
- KANELÉ 방식으로 FPGA 2026 최우수논문상을 수상했으며, 하이브리드 방식으로 0.8B LLM으로도 경쟁력 있는 성능을 달성합니다.

> [!tip] 왜 중요한가
> 초저지연이 필요한 엣지 컴퓨팅, 금융거래, 자율주행 등의 실시간 응용에서 GPU를 대체할 새로운 가능성을 제시합니다.

## FPGA에서 머신러닝을 해야 하는 이유

최신 머신러닝 워크로드는 대부분 GPU에서 돌아갑니다. 훈련이든 추론이든 말이죠. GPU는 매우 병렬적인 실행 모델을 지원하는 하드웨어 구조 덕분에, 방대한 데이터에 대해 단순한 연산을 엄청나게 빠르게 처리할 수 있습니다. 따라서 큰 규모의 신경망이나 배치 형식의 훈련·추론 작업에는 GPU가 이상적이죠.

그런데 문제가 있습니다. 초저지연(예: 마이크로초 이하)과 높은 하드웨어 효율을 동시에 요구하는 애플리케이션에서는 GPU 같은 복잡한 프로세서가 제 역할을 하지 못합니다. CPU나 GPU 같은 범용 프로세서는 명령어 스케줄링, 메모리 접근 최적화 등에서 오버헤드가 크거든요. 초저지연(나노초 단위)과 높은 효율이 필수인 특화된 작업에는 커스텀 하드웨어 가속기가 훨씬 낫습니다.

이 지점에서 FPGA(Field-Programmable Gate Array)가 빛을 발합니다. FPGA는 재구성 가능한 디지털 로직 칩으로, 커스텀 하드웨어 가속에 아주 적합합니다. FPGA에는 조회 테이블(LUT, Lookup Table), 플립플롭(FF), 그 외 메모리와 연산 소자들이 있고, 이들과 연결 구조를 재설정해서 원하는 디지털 회로를 만들 수 있습니다. 덕분에 낮은 수준의 하드웨어 아키텍처와 알고리즘을 함께 설계할 수 있고, 이를 통해 초고속 머신러닝이 가능해집니다. 특히 신경망이 프로세서에서 순차적으로 실행되는 명령어가 아니라, 디지털 로직으로 직접 구현되는 점이 중요합니다.

## 배경지식

### 고정소수점 양자화

FPGA 같은 디지털 장치는 근본적으로 비트(bit)를 다룹니다. 연속값이 아니라 말이죠. 하지만 신경망의 산술 연산(덧셈, 곱셈 등)은 보통 실수($\mathbb{R}$) 범위에서 이뤄진다고 생각합니다. 따라서 실수를 비트 문자열로 인코딩해야 하는데, 이 과정을 양자화(quantization)라고 부릅니다. 그러면 덧셈, 곱셈 같은 연산이 이진 함수가 되는 겁니다.

이를 구현하는 한 가지 방법이 고정소수점 양자화입니다.

고정소수점 양자화는 2진법으로 수를 표현하되, 일부 비트(소수 부분 비트)를 소수점 뒤에 배치합니다. 예를 들어 전체 8비트 중 4비트를 소수 부분으로 쓴다면, $(-2^7) / 2^4 = -8$부터 $(2^7 - 1) / 2^4 = 7.9375$까지 $2^8$개의 값을 표현할 수 있습니다. 각 값 사이의 간격은 $1/2^4 = 0.0625$로 균등하죠. 여기서는 표현 가능한 범위가 0을 중심으로 대칭이라고 가정하겠습니다.

고정소수점 양자화 방식에서는 정해진 범위 내에서만 이산적인 값들을 표현할 수 있습니다. 따라서 실제 값을 표현할 때 근사 오차, 즉 양자화 오차가 발생합니다. 자원 효율적인 머신러닝의 핵심 과제 중 하나는 이 양자화 오차를 최소화해서 안정적인 훈련과 추론을 가능하게 하는 것입니다.

### 조회 테이블 신경망(LUT-NN)

FPGA는 주로 조회 테이블(LUT)을 통해 디지털 로직을 구현합니다. LUT는 작은 소자인데, 이진 입력의 모든 조합에 대한 출력값을 저장해서 임의의 이진 함수를 표현합니다. 예를 들어 AND 함수 $\text{AND} : \{0, 1\}^2 \to \{0, 1\}$는 다음과 같은 조회 테이블로 나타낼 수 있습니다.

| 입력 ($x, y$) | $x \text{ AND } y$ |
| --- | --- |
| 00 | 0 |
| 01 | 0 |
| 10 | 0 |
| 11 | 1 |

그렇다면 이런 조회 테이블로 표현된 이진 함수들을 신경망의 기본 단위로 학습하면 어떨까요? 이렇게 만든 신경망을 조회 테이블 신경망(LUT-NN)이라고 부릅니다. 다만 문제가 있습니다. 조회 테이블을 경사하강법 같은 방식으로 직접 학습하기는 매우 어렵거든요.

이 문제를 해결하기 위해 다시 생각해 봅시다. 경사하강법으로 실수 함수 $f: \mathbb{R} \to \mathbb{R}$를 학습할 수 있습니다. 이제 $b_i$개의 입력 비트와 $b_o$개의 출력 비트를 가진 고정소수점 양자화를 적용하면, $f$는 이진 함수 $f_Q : \{0, 1\}^{b_i} \to \{0, 1\}^{b_o}$로 변환됩니다. 그러면 연속 함수 $f$를 학습한 후에, 그것을 양자화된 버전으로 변환할 수 있습니다.
