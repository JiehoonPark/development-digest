## 핵심 개념

> [!abstract]
> 사용자 하드웨어에서 실제로 실행되고 성능이 우수한 로컬 LLM을 찾아주는 도구로, 파라미터 수가 아닌 벤치마크 기반의 실시간 순위를 제공한다.

## 상세 내용

- 로컬 LLM 선택을 파라미터 크기가 아닌 실제 성능 벤치마크를 기준으로 한다.
- 한 명령어로 사용자의 특정 하드웨어에 최적화된 LLM을 빠르게 식별할 수 있다.

> [!tip] 왜 중요한가
> 개발자가 로컬 환경에서 실제 필요에 맞는 LLM을 효율적으로 선택할 수 있게 해주는 실용적인 도구이다.
