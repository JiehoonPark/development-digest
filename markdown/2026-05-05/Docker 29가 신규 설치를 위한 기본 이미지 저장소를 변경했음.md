## 핵심 개념

> [!abstract]
> Docker 29 버전에서 컨테이너 이미지 저장소의 기본값이 변경되었다.

## 상세 내용

- 저장소 기본값 변경: Docker 29는 새로운 설치에서 기본 이미지 저장소 설정을 수정했으며, 이는 containerd 통합과 관련된 변경 사항이다.

> [!tip] 왜 중요한가
> Docker를 업그레이드하거나 새로 설치하는 개발자와 운영자는 저장소 설정 변경을 인식하고 필요시 환경 구성을 조정해야 한다.
