## 핵심 개념

> [!abstract]
> ruflo는 Claude 기반의 에이전트 오케스트레이션 플랫폼으로, 다중 에이전트 스웜 배포, 자율적 워크플로우 조정, 대화형 AI 시스템 구축을 지원한다. 엔터프라이즈급 아키텍처, 자가 학습 스웜 인텔리전스, RAG 통합, Claude Code 및 Codex 네이티브 지원을 포함한다.

## 상세 내용

- 다중 에이전트 스웜(Multi-agent Swarms) 배포 지원으로 여러 AI 에이전트를 조정하여 복잡한 작업을 병렬로 처리할 수 있다.
- 자가 학습 스웜 인텔리전스 기능으로 에이전트들이 상호작용하면서 성능을 지속적으로 개선한다.
- RAG(Retrieval-Augmented Generation) 통합으로 외부 지식 소스를 에이전트에 직접 연결할 수 있다.
- Claude Code와 Codex를 네이티브로 지원하여 코드 생성 및 실행 작업을 자동화한다.
- 엔터프라이즈급 아키텍처로 프로덕션 환경에서 안정적인 워크플로우 구축 및 운영이 가능하다.

> [!tip] 왜 중요한가
> Claude 기반 멀티에이전트 시스템을 빠르게 배포하려는 개발팀에게 필수적인 오케스트레이션 플랫폼으로, 복잡한 자동화 작업을 효율적으로 관리할 수 있게 해준다.
