## 핵심 개념

> [!abstract]
> Anthropic이 금융 서비스 분야의 가장 시간 소모적인 작업(피칭북 구성, KYC 파일 스크리닝, 월말 결산)을 자동화하는 10개의 에이전트 템플릿을 출시했다. 이들은 Claude Cowork, Claude Code 플러그인 또는 Claude Managed Agents 쿡북으로 제공되며, Microsoft 365 애드인(Excel, PowerPoint, Word, Outlook)을 통해 협업 소프트웨어 전반에서 작동한다. Claude Opus 4.7은 Finance Agent 벤치마크에서 64.37%로 업계 최고 성능을 달성했다.

## 상세 내용

- 10개 금융 에이전트 템플릿의 구성: 각 템플릿은 스킬(작업 지침 및 도메인 지식), 커넥터(데이터 거버넌스 접근), 서브에이전트(비교분석 선정, 방법론 검증 등 세부 작업)의 3가지 요소로 구성되어 있으며, 기업이 자신의 모델링 관례, 위험 정책, 승인 절차에 맞게 적응할 수 있다.
- 연구 및 고객 커버리지 5개 에이전트: Pitch Builder(목표 리스트 생성, 비교분석, 피칭북 초안), Meeting Preparer(클라이언트/거래처 브리핑), Earnings Reviewer(실적 공시 검토, 모델 업데이트), Model Builder(공시와 데이터 피드에서 재무모델 생성), Market Researcher(섹터/발행사 개발 추적 및 합성)가 있다.
- 금융 및 운영 5개 에이전트: Valuation Reviewer(비교분석과 방법론 대조 검토), General Ledger Reconciler(계정 조정 및 NAV 계산), Month-end Closer(결산 체크리스트 실행), Statement Auditor(재무제표 일관성/완성도 검토), KYC Screener(엔티티 파일 조립 및 컴플라이언스 검토)로 구성된다.
- 두 가지 배포 모드: 플러그인으로 Claude Cowork/Code 내에서 애널리스트 옆에 실행되거나, Claude Managed Agents로서 플랫폼에서 자율 실행되며, 후자는 장시간 세션, 도구 권한 관리, 자격증명 보관, 감사 로그 등 엔터프라이즈 요구사항을 포함한다.
- Microsoft 365 애드인 통합: Claude는 Excel, PowerPoint, Word, Outlook(곧 출시)을 통해 네이티브로 작동하며, 문맥이 애플리케이션 간에 자동으로 전달되므로 Excel에서 작성한 모델이 PowerPoint 업데이트로 자동 반영된다. Outlook에서는 수석 참모처럼 받은편지함을 분류하고 회의를 조율하며 응답을 작성할 수 있다.
- 사용자 제어 및 감시 유지: 모든 시나리오에서 사용자는 Claude의 작업을 검토, 반복, 승인한 후 클라이언트에 제출하거나 기록에 등록하므로, 완전 자동화가 아닌 협력 형태로 운영된다. Claude Console에서 전체 도구 호출 및 의사결정에 대한 감사 로그를 제공한다.
- Claude Opus 4.7의 금융 성능: Vals AI의 Finance Agent 벤치마크에서 64.37%로 업계 선도 성능을 보이며, 금융 특화 작업에 최적화되었음을 입증한다.

> [!tip] 왜 중요한가
> 금융 서비스 업계의 개발팀과 운영팀은 복잡한 custom 에이전트를 몇 달에 걸쳐 구축하지 않고도 며칠 내에 프로덕션 에이전트를 배포할 수 있으며, Microsoft 365 통합으로 기존 워크플로우에 매끄럽게 통합되므로 도입 장벽이 크게 낮아진다.

## 금융 전문가를 위한 10가지 에이전트 템플릿 출시

금융 업계에서 가장 시간이 오래 걸리는 일들이 있죠. 피치북 작성, KYC 파일 심사, 월말 결산 같은 작업들 말입니다. Anthropic이 이런 반복 업무를 처리할 수 있는 10가지 에이전트 템플릿을 준비했습니다.

이 템플릿들은 Claude Cowork과 Claude Code에 플러그인으로 제공되고, Claude Managed Agents를 위한 쿡북 형태로도 사용할 수 있습니다. 결과적으로 팀이 몇 개월이 아닌 며칠 안에 Claude를 실제 금융 업무에 투입할 수 있다는 뜻입니다.

## Microsoft 365 환경에서 Claude 활용하기

또한 Claude는 이제 Microsoft Excel, PowerPoint, Word, Outlook(곧 출시)을 지원하는 애드인을 통해 직접 작동합니다. 애드인을 설치하면 애플리케이션 간에 컨텍스트가 자동으로 유지되므로, 한 도구에서 시작한 작업을 다른 도구에서 마무리할 때 같은 내용을 다시 설명할 필요가 없습니다.

동시에 파트너 생태계도 계속 확장하고 있습니다. 새로운 커넥터와 MCP 앱이 추가되면서 에이전트들이 금융 전문가들이 이미 사용 중인 데이터에 접근할 수 있게 됐습니다. 커넥터는 Claude에 데이터 제공자의 실시간 데이터에 대한 제어된 접근을 제공하고, MCP 앱은 더 나아가 제공자의 도구를 Claude에 직접 내장시킵니다.

이 모든 기능은 Claude Opus 4.7과 함께 제공됩니다. 이 모델은 금융 업무에서 업계 최고 수준의 성능을 보여주며, Vals AI의 Finance Agent 벤치마크에서 64.37%의 점수로 업계를 선도하고 있습니다.

## 금융 업무를 위한 새로운 에이전트 템플릿들

각 에이전트 템플릿은 세 가지를 패키징한 참조 아키텍처입니다.

**첫째, 스킬(Skill)** — 작업에 필요한 지시사항과 도메인 지식입니다. **둘째, 커넥터(Connector)** — 데이터에 대한 제어된 접근입니다. **셋째, 서브에이전트(Subagent)** — 비교 분석 선정이나 방법론 검증 같은 세부 작업을 처리하기 위해 호출되는 추가 Claude 모델들입니다.

각 템플릿은 기업의 모델링 관례, 위험 정책, 승인 절차에 맞게 커스터마이징할 수 있습니다.

이 에이전트 템플릿들은 Claude Cowork이나 Claude Code 내에서 플러그인으로, 또는 Claude Managed Agents를 위한 쿡북으로 활성화할 수 있습니다. 모든 플러그인과 쿡북은 금융 서비스 마켓플레이스에서 찾을 수 있습니다.

### 리서치 및 클라이언트 커버리지

- **Pitch Builder** — 대상 리스트를 작성하고, 비교 분석을 실행하며, 클라이언트 미팅용 피치북을 작성합니다.
- **Meeting Preparer** — 통화 전에 클라이언트와 상대방 정보를 정리합니다.
- **Earnings Reviewer** — 실적 발표 트랜스크립트와 공시 자료를 읽고, 모델을 업데이트하며, 투자 테제와 관련된 변화를 표시합니다.
- **Model Builder** — 공시 자료, 데이터 피드, 분석가 입력을 바탕으로 재무 모델을 생성하고 유지관리합니다.
- **Market Researcher** — 섹터와 발행자의 움직임을 추적하고, 뉴스와 공시 자료, 리서치를 종합하며, 신용 및 위험 검토가 필요한 사항을 표시합니다.

### 재무 및 운영

- **Valuation Reviewer** — 밸류에이션을 비교 분석, 방법론, 회사의 검토 기준과 대조합니다.
- **General Ledger Reconciler** — 총계정원장 계정을 조정하고 기록 대조 기준으로 순자산 가치를 계산합니다.
- **Month-end Closer** — 마감 체크리스트를 실행하고, 분개를 준비하며, 마감 보고서를 생성합니다.
- **Statement Auditor** — 재무제표의 일관성, 완결성, 감사 준비 상태를 검토합니다.
- **KYC Screener** — 법인 파일을 정리하고, 원본 문서를 검토하며, 규정 준수 검토를 위한 상향 보고 내용을 패키징합니다.

## 실제 운영 방식: 두 가지 시나리오

### 1. 플러그인으로 사용 (Claude Cowork/Claude Code)

템플릿이 분석가 옆에서 실시간으로 작동합니다. 이미 사용 중인 데스크톱 소프트웨어와 함께 동작한다는 뜻입니다.

예를 들어 Pitch 에이전트에 대상 리스트를 건네면, Excel 형식의 비교 분석 모델, PowerPoint로 작성된 피치북, Outlook에서 발송할 커버 레터까지 받을 수 있습니다.

### 2. Claude Managed Agent로 운영 (자동화 실행)

동일한 템플릿이 Claude 플랫폼에서 자율적으로 작동하며, 여러 건의 딜을 처리하거나 매일 밤 정해진 시간에 실행되는 업무에 적합합니다.

쿡북은 기업이 직접 구축해야 할 빌딩 블록들을 함께 제공합니다. 여기에는 여러 시간에 걸친 딜 마감을 처리할 수 있는 오래 지속되는 세션, 도구별 권한 설정, 관리형 자격증명 보관소, Claude Console의 완전한 감사 로그가 포함됩니다. 규정 준수 팀과 엔지니어링 팀은 여기서 모든 도구 호출과 의사결정을 추적할 수 있습니다.

**중요한 점은, 어떤 방식이든 사람이 항상 루프 안에 있다는 것입니다.** 분석가는 Claude의 작업을 검토하고, 필요하면 수정하고, 최종 승인한 후에야 클라이언트에게 제출하거나 보관하거나 실행합니다.

## Excel, PowerPoint, Word, Outlook에서 Claude 활용하기

Claude는 이제 애드인을 통해 Microsoft Excel, PowerPoint, Word, Outlook에서 직접 작동합니다.

**Outlook에서는** 수석 보좌관처럼 받은 메일을 분류하고, 회의를 일정에 넣고, 당신의 톤으로 응답을 작성합니다.

**Excel에서는** 공시 자료와 데이터 피드로부터 재무 모델을 만들고, 연결된 워크북 전체의 수식을 감사하며, 민감도 분석을 실행합니다.

**PowerPoint에서는** 기저 수치가 바뀌면 자동으로 업데이트되는 슬라이드를 작성합니다.

**Word에서는** 신용 메모를 회사의 템플릿에 맞춰 편집합니다.

Claude는 네 플랫폼 전체에서 자신의 지식과 컨텍스트를 유지합니다. Excel에서 모델 작성을 시작한 분석가가 그 작업을 PowerPoint로 옮길 때, 같은 내용을 다시 설명할 필요가 없다는 뜻입니다.

Claude Cowork에서는 **Dispatch**를 이용해 어디서나 텍스트나 음성으로 Claude에 작업을 지시할 수 있습니다. Claude는 분석가가 자리를 비운 동안에도 로컬 파일에서 계속 작업하고 있다가, 돌아왔을 때 검토할 준비가 완료된 결과물을 제공합니다.
