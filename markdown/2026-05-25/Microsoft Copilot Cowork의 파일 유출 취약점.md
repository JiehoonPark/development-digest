# Microsoft Copilot Cowork, 파일 유출 취약점 노출

Microsoft Copilot Cowork에서 간접 프롬프트 주입 공격을 통한 파일 유출 취약점이 발견됐습니다. 공격자들이 에이전트 프로세스를 악용하면, 사용자의 명시적 승인 없이도 Teams, 이메일, 공유 플랫폼에 접근할 수 있다는 게 문제입니다. 특히 사용자가 파일을 업로드하거나 손상된 콘텐츠와 상호작용할 때 개인정보와 금융 정보 탈취가 가능해집니다.

근본적인 원인은 시스템이 너무 광범위한 권한을 부여하도록 설계됐다는 점입니다. 여기에 지속적인 공격 벡터가 더해지면서 공격 표면이 더욱 넓어지는 셈이죠. 대응책으로는 다운로드 링크 접근을 제한하고 권한을 더 엄격하게 관리해야 합니다.

# Go로 eBPF 프로그램을 작성하세요 (C는 이제 그만)

Bora Tanrikulu가 만든 **gobee**라는 도구를 소개합니다. 이건 eBPF 프로그램을 C 대신 Go의 제한된 서브셋으로 작성할 수 있게 해주는 트랜스파일러입니다.

작성한 Go 코드를 BPF C로 변환하고, 유저스페이스 쪽을 위해 타입이 지정된 cilium/ebpf 바인딩을 자동으로 생성해줍니다. XDP, tracepoint, kprobe, LSM을 포함한 8가지 프로그램 타입과 19가지 맵 타입을 지원하며, libbpf v1.5.0의 약 200개 BPF 헬퍼를 자동 생성합니다.

특히 좋은 점들을 꼽으면:
- 검증기 오류를 원래 Go 소스 위치로 매핑
- bpfvet를 통한 커널 버전 게이팅
- clang을 활용한 CO-RE 자동 감지
- 소스맵 사이드카 생성
- Linux arm64, amd64 크로스 아키텍처 지원

gobee는 LLVM을 직접 타겟하지 않고, clang의 검증된 BPF 백엔드를 재사용합니다. 덕분에 BTF와 검증기 친화적인 코드 생성이 가능합니다. 결국 커널 영역과 유저스페이스 코드를 하나의 Go 모듈로 통합할 수 있으면서, cilium/ebpf, clang 같은 기존 도구들을 그대로 활용할 수 있다는 게 강점입니다.

# 해커 뉴스 프론트페이지에서 본 주목할 기술들

요즘 IT와 과학계에서 화제인 소식들을 모아봤습니다.

**Apple의 PICO (지각형 이미지 코덱)**는 기존 코덱 대비 최대 3배의 비트레이트 감소를 달성했고, iPhone 17 Pro Max에서도 동작합니다.

**SpaceX의 Starship V3** 준궤도 테스트가 성공했습니다. 엔진 하나가 고장났음에도 핫 스테이징에 성공했고, 프로토타입 Starlink 위성들을 탑재했습니다.

**DeepSeek은 V4 Pro 모델의 가격을 영구적으로 인하**했고, 토큰 기반 가격 책정을 도입하면서 모델 이름도 바꿀 예정입니다.

**Microsoft는 MS-DOS 초기 소스코드를 오픈소스로 공개**하면서 MS-DOS의 역사를 조명했습니다.

**이탈리아는 Airbus A330 MRTT 급유기 6대를 구매**하기로 결정했고, 기존의 Boeing KC-46에서 유럽 플랫폼으로 전환합니다.

**호주에서 1,000년 전 딩고 화석이 발견**됐는데, 부상 치료와 의식적 매장의 흔적이 남아있어 원주민의 돌봄이 있었음을 시사합니다.

이 외에도 LLM 에이전트의 "제약 붕괴(constraint decay)" 연구, 웹 기반 무료 오픈소스 멀티트랙 오디오 에디터 Audiomass, 불확실한 샘플링 좌표를 다루는 베이지안 가우시안 프로세스 모델 등이 주목받고 있습니다.

# 캐나다의 '조용한 두뇌 유출' 현상

캐나다가 인재 유출로 골머리를 앓고 있습니다. TD 이코노믹스 리포트에 따르면, 숙련된 근로자, 기업가, STEM 전문가들이 더 낮은 세금과 강한 경제 전망을 찾아 미국으로 떠나고 있는데요.

특히 온타리오와 브리티시컬럼비아의 높은 개인 및 기업세가 전문가들을 텍사스와 플로리다로 몰아내고 있습니다. 생산성 부진도 문제인데, 이는 약한 비즈니스 확장성과 부족한 벤처캐피탈 때문입니다.

보고서는 인재 유지를 위해선 세제 개혁과 국내 성장 촉진을 포함한 경제 경쟁력 개선이 필수라고 강조합니다.

# 일본의 극초음속 엔진, 미국까지 2시간 비행 현실화할까

일본의 JAXA가 와세다대, 도쿄대, 게이오대와 함께 카쿠다 우주센터에서 마하 5 램제트의 지상 연소 테스트를 성공적으로 완료했습니다.

테스트는 고도 25km, 음속의 5배라는 극한 조건을 시뮬레이션했으며, 내열 차폐, 제어면, 엔진 성능 모두 검증됐습니다. 이는 2040년대 상용 극초음속 여객기 운항을 향한 중요한 한 걸음입니다.

현재 도쿄-로스앤젤레스 항공편은 약 10시간 걸리는데, 이 기술이 상용화되면 약 2시간으로 단축될 수 있습니다.

# IBM, 미국 최초의 양자칩 파운드리 분사

미국 상무부와 IBM이 $20억 규모의 이니셔티브를 발표했습니다. 뉴욕주 올버니에 미국 최초의 순수 양자칩 파운드리인 **Anderon**을 설립할 계획입니다.

CHIPS법으로부터 $10억, IBM으로부터 $10억을 받아 300mm 초전도 실리콘 웨이퍼 제조 시설을 운영할 예정입니다. 이를 통해 미국이 양자 제조에서 세계 최고 수준 경쟁력을 확보하겠다는 전략입니다.

특히 초전도 실리콘 기술에 중점을 두고 있는데, 이는 이미 검증된 반도체 공정을 활용해 확장 가능하고 빠른 반복 개발이 가능하다는 게 장점입니다. 더 작은 200mm 대안보다 전략적 우위를 차지합니다.

이러한 자금 구조는 2단계 양자 생태계를 만들며, 인프라 자본을 제조 준비가 된 기술에 집중시킵니다.
