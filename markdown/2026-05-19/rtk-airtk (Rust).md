## 핵심 개념

> [!abstract]
> 일반적인 개발 명령어에서 LLM 토큰 소비를 60-90% 감소시키는 CLI 프록시로, 단일 Rust 바이너리로 의존성 없이 동작한다.

## 상세 내용

- 토큰 효율성: 일반적인 개발 명령어 실행 시 LLM 토큰 소비를 60-90% 감소시켜 AI 모델 이용 비용을 대폭 절감할 수 있다.
- 경량 구현: 단일 Rust 바이너리로 구현되어 의존성이 없으므로 배포와 설치가 간편하다.

> [!tip] 왜 중요한가
> LLM 기반 개발 도구의 운영 비용을 크게 감소시켜 AI 개발 도구의 경제성을 개선한다.
