## 핵심 개념

> [!abstract]
> Claude Code를 터미널, VSCode 확장, Discord에서 무료로 사용할 수 있게 하는 Python 기반 도구로, 음성 지원도 제공한다.

## 상세 내용

- OpenClaw와 유사한 멀티플랫폼 지원으로 Claude 모델에 쉽게 접근 가능
- 음성 입력 지원으로 핸즈프리 코딩 경험 제공

> [!tip] 왜 중요한가
> 유료 Claude API 없이도 고급 코드 생성 기능을 다양한 환경에서 활용할 수 있다.
