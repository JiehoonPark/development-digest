## 핵심 개념

> [!abstract]
> Google이 Gemini 3.5 모델 시리즈를 발표했으며, 먼저 Gemini 3.5 Flash를 공개했다. 3.5 Flash는 에이전트 작업과 코딩에 최적화되어 있으며, Terminal-Bench 2.1에서 76.2%, GDPval-AA에서 1656 Elo, MCP Atlas에서 83.6%의 성능을 기록하며 Gemini 3.1 Pro를 능가한다. 출력 토큰/초 기준으로 다른 최첨단 모델보다 4배 빠르면서도 최고 성능을 유지하며, 개발자와 엔터프라이즈 고객을 위해 이미 출시되었고, 3.5 Pro는 내달 출시될 예정이다. Google Antigravity 플랫폼을 통해 다중 에이전트 워크플로우를 지원하여 복잡한 장기 작업을 기존 대비 절반 이하의 비용으로 완료할 수 있다.

## 상세 내용

- 성능 벤치마크: Terminal-Bench 2.1 76.2%, GDPval-AA 1656 Elo, MCP Atlas 83.6%로 기존 Gemini 3.1 Pro를 능가하며, CharXiv Reasoning에서 84.2%의 멀티모달 이해도를 달성했다.
- 속도 최적화: 다른 최첨단 모델 대비 출력 토큰/초 기준 4배 빠른 속도로, Artificial Analysis 지수의 우상향 사분면에 위치하여 성능과 지연 시간의 최적 균형을 제공한다.
- 에이전트 능력: Antigravity 플랫폼과 함께 사용하면 다중 단계 워크플로우를 안정적으로 실행하고 복수의 서브에이전트를 병렬로 배포하여 장시간 작업을 감독 하에 수행할 수 있다.
- 실제 사용 사례: Shopify는 전역 규모의 판매자 성장 예측에 병렬 서브에이전트를 사용하고, Macquarie Bank는 100페이지 이상의 복잡한 문서 분석으로 고객 온보딩을 가속화하며, Salesforce는 Agentforce에 통합하여 복잡한 엔터프라이즈 작업을 자동화한다.
- 비용 효율성: 개발자가 며칠 걸리거나 감사자가 수주 소요하던 작업을 단편의 시간에 완료할 수 있으며, 다른 최첨단 모델의 절반 이하 비용으로 운영 가능하다.
- 멀티모달 및 UI 생성: Gemini 3 기반의 강력한 멀티모달 능력을 바탕으로 더 풍부하고 인터랙티브한 웹 UI와 그래픽을 생성하며, AI Studio에서 60초 내에 여러 UX 접근 방식을 생성할 수 있다.
- 배포 가능성: Gemini 앱, Google Search의 AI Mode, Gemini API, Google Android Studio, 엔터프라이즈 플랫폼 등 다양한 채널을 통해 수십억 명의 사용자에게 제공되고 있다.

> [!tip] 왜 중요한가
> 3.5 Flash는 기존 대비 4배 빠른 속도로 최첨단 성능을 유지하면서 에이전트 기반 워크플로우를 프로덕션 규모로 실행할 수 있게 해주므로, 개발자와 엔터프라이즈가 복잡한 자동화 작업을 비용 효율적으로 구현할 수 있는 실용적인 선택지가 된다.

## 전문 번역

# Gemini 3.5: 지능형 에이전트의 새로운 시대

2026년 5월 19일

Google DeepMind와 Google이 새로운 AI 모델 시리즈인 Gemini 3.5를 공개했습니다. 이번 모델은 뛰어난 지능과 실행 능력을 결합해 더욱 강력한 AI 에이전트를 만드는 데 초점을 맞췄는데요. 먼저 3.5 Flash 모델부터 출시하고 있으며, 향후 3.5 Pro도 선보일 예정입니다.

## Gemini 3.5 Flash: 에이전트와 코딩의 새로운 기준

Gemini 3.5 Flash는 정말 인상적인 성능을 자랑합니다. Flash 시리즈의 빠른 속도를 유지하면서도 대형 모델 수준의 지능을 제공하거든요.

코딩과 에이전트 능력에서는 특히 돋보이는데, Gemini 3.1 Pro를 앞서갑니다. Terminal-Bench 2.1에서 76.2%, MCP Atlas에서 83.6%, 멀티모달 이해도에서 84.2%의 성능을 기록했습니다. 더 놀라운 점은 속도인데요. 다른 최첨단 모델 대비 초당 4배 빠른 토큰 처리 속도를 자랑합니다.

**더 이상 성능과 속도 사이에서 선택할 필요가 없습니다.** Artificial Analysis 지표에서도 3.5 Flash는 최고의 성능 구간에 위치하고 있죠.

## 장기적인 작업을 해내는 능력

이러한 속도와 성능의 조합이 3.5 Flash를 복잡한 장기 작업에 최적화했습니다. 개발자가 며칠 걸리고 감시자가 몇 주 걸리던 작업을 이제는 훨씬 짧은 시간에, 그것도 다른 최첨단 모델의 절반 이하 비용으로 처리할 수 있습니다.

새롭게 업데이트된 Antigravity 플랫폼과 함께 사용하면 진가를 드러냅니다. 3.5 Flash는 여러 개의 협력 에이전트를 배포해 복잡한 문제를 규모 있게 해결할 수 있거든요. 감독하에 다단계 워크플로우와 코딩 작업을 안정적으로 실행하면서도 최고 수준의 성능을 유지합니다.

### 실제 활용 예시

**데이터 정리 및 분류**
Antigravity의 힘을 借하면 3.5 Flash는 다단계 워크플로우를 자동으로 실행해 정형화되지 않은 데이터를 동적 기준에 따라 이름 붙이고 분류합니다.

**학술 논문 분석 및 게임 개발**
두 개의 에이전트가 협력해 AlphaZero 논문을 분석하고 완전히 플레이 가능한 게임을 6시간 만에 개발했습니다.

**레거시 코드 현대화**
Antigravity의 에이전트 시스템이 복잡한 레거시 코드베이스를 Next.js로 변환합니다.

**창의적 작업**
도시 경관 생성부터 시작해 다양한 창의적 작업도 가능합니다.

**자체 개선 루프**
게임 개발자와 플레이어 에이전트가 협력해 빠른 자체 개선 루프 속에서 게임을 개발합니다.

## 멀티모달 능력으로 풍부한 UI 생성

Gemini 3의 강력한 멀티모달 기반을 바탕으로 3.5 Flash는 더욱 풍부하고 상호작용이 뛰어난 웹 UI와 그래픽을 만들어냅니다.

- **인터랙티브 애니메이션**: 연구 논문을 위한 동적 시각화
- **텍스트에서 하드웨어로**: 일반 텍스트 설명을 실제 작동하는 하드웨어로 변환
- **병렬 개념 구현**: 학교 펀드레이징을 위한 완전한 브랜딩 개념을 한 번에 여러 방향으로 구현
- **빠른 UX 디자인**: 결제 흐름의 다양한 UI 접근 방식을 단 60초 만에 생성

## 실제 비즈니스 임팩트

Gemini 3.5 시리즈 개발 과정에서 업계 파트너들과 긴밀히 협력했습니다. 그들의 워크플로우에서 어떤 반복적이고 복잡한 작업이 발생하는지 이해하기 위해서죠. 이미 눈에 띄는 성과들이 나타나고 있습니다.

**Shopify**: 여러 에이전트를 병렬로 실행해 복잡한 데이터를 장기간 분석함으로써 전 세계 규모의 더 정확한 상인 성장 예측이 가능해졌습니다.

**Macquarie Bank**: 100페이지 이상의 복잡한 문서를 분석하고 관련 정보를 추출해 신뢰할 수 있는 추천을 낮은 지연 시간으로 제공함으로써 고객 온보딩을 가속화하고 있습니다.

**Salesforce**: Agentforce에 3.5 Flash를 통합해 맥락을 유지하고 복잡한 멀티턴 도구 호출을 실행하는 다중 에이전트를 배포함으로써 복잡한 엔터프라이즈 작업을 안정적으로 자동화합니다.

**Ramp**: 멀티모달 이해 능력으로 복잡한 인보이스를 분석하고 역사적 패턴에 대한 추론을 결합해 더욱 스마트하고 신뢰할 수 있는 OCR을 제공합니다.

**Xero**: 에이전트 기술을 배포해 회계 작업 자동화를 진행 중입니다.
