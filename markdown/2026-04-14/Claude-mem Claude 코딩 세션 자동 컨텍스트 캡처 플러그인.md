## 핵심 개념

> [!abstract]
> Claude Code 플러그인으로 코딩 세션 중 Claude의 모든 작업을 자동으로 캡처하고 AI로 압축한 후 향후 세션에 관련 컨텍스트를 주입하는 도구이다.

## 상세 내용

- 코딩 세션의 전체 작업 내역을 자동 캡처하고 Claude의 agent-sdk로 압축 처리
- 이전 세션의 컨텍스트를 새로운 세션에 자동으로 주입하여 작업 연속성 향상

> [!tip] 왜 중요한가
> Claude와의 장기 코딩 프로젝트에서 작업 컨텍스트 손실을 방지하고 효율성을 높일 수 있다.
