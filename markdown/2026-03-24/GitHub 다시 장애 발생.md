## 핵심 개념

> [!abstract]
> GitHub 서비스에 장애가 발생했으며 상태 페이지에서 사건이 보고되었습니다.

## 상세 내용

- GitHub 웹훅, 인증 등 주요 서비스 영향 가능성
- 사용자들이 이메일 또는 SMS로 장애 알림 구독 가능

> [!tip] 왜 중요한가
> CI/CD 파이프라인과 개발 워크플로우가 GitHub에 의존하므로 장애 시 즉각적인 영향을 받을 수 있습니다.

## 전문 번역

# OTP를 받지 못했나요?

OTP 재전송

# 웹훅을 전송할 URL

엔드포인트가 실패하면 이메일로 알려드립니다.

# 팁, 기술 가이드, 모범 사례를 받아보세요

월 2회 주기로 메일함에 직접 전달됩니다.

GitHub 서비스와의 중단(Disruption)에 대한 업데이트를 이메일이나 문자 메시지로 받아보세요. 사건이 업데이트되면 이메일 알림을, GitHub가 사건을 생성하거나 해결하면 문자 메시지 알림을 받게 됩니다.
