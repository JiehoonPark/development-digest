## 핵심 개념

> [!abstract]
> Anthropic이 Claude Opus 4.8을 출시했으며, 이는 Opus 4.7의 개선 버전으로 코딩, 에이전트 작업, 추론, 실무 작업 벤치마크에서 향상된 성능을 보인다. 동일한 가격대에서 제공되며 Claude Code의 새로운 '동적 워크플로우' 기능, claude.ai에서의 작업 노력 제어 옵션, 그리고 Opus 4.8의 고속 모드(2.5배 속도)가 이전 모델 대비 3배 저렴해졌다. 초기 테스터들은 에이전트 작업에서 더 나은 판단력과 신뢰성을 보고했으며, 법률 문서, 데이터 분석, 컴퓨터 사용 같은 전문 업무에서 이전 모델과 경쟁 모델을 능가한다.

## 상세 내용

- Claude Opus 4.8은 Opus 4.7 대비 벤치마크 전반에서 성능이 향상되었으며 더 효과적인 협업 파트너가 되었다. 동일한 가격대에서 제공되고, 고속 모드는 이전 모델보다 3배 저렴해졌다.
- claude.ai 사용자는 이제 작업에 소비할 노력의 정도를 조절할 수 있고, Claude Code는 대규모 문제를 해결할 수 있는 '동적 워크플로우' 기능이 새로 추가되었다.
- Super-Agent 벤치마크에서 Claude Opus 4.8은 동일 비용 기준으로 모든 케이스를 완료한 유일한 모델이며, 이전 Opus 모델과 GPT-5.5를 능가한다. 번역, 심층 연구, 슬라이드 생성, 분석 같은 에이전트 제품에서 강력한 신뢰성을 제공한다.
- Legal Agent Benchmark에서 Claude Opus 4.8은 기록상 최고 점수를 기록했으며, 전체 통과 기준에서 10% 이상을 달성한 첫 번째 모델이다. 이는 법률 업무의 실제 자동화 가능성을 나타낸다.
- Online-Mind2Web 벤치마크에서 84% 점수를 기록해 컴퓨터 사용 및 브라우저 에이전트 분야에서 가장 강력하며, Opus 4.7과 GPT-5.5 대비 의미 있는 개선을 보였다.
- 도구 호출이 더 효율적이 되어 같은 지능 수준을 위해 더 적은 단계를 사용하며, 엔드투엔드 작업을 일관되게 수행할 수 있다. 자율 엔지니어링 워크로드가 무인으로 계속 실행될 수 있도록 필요한 신뢰성을 제공한다.
- 긴 세션에서도 문맥과 스타일 방향을 일관되게 유지하며, 성능(속도), 협업 용이성, 장기 문맥 유지 측면에서 Opus 4.7 대비 주목할 만한 삶의 질 향상을 제공한다.

> [!tip] 왜 중요한가
> Claude Opus 4.8의 향상된 판단력, 신뢰성, 그리고 비용 효율성(특히 고속 모드)은 법률, 금융, 데이터 분석 같은 전문 분야에서 AI 에이전트 자동화를 실질적으로 확대할 수 있게 한다.

## 전문 번역

# Claude Opus 4.8 출시: 더 똑똑하고 빠른 AI 협력자

5월 28일, Anthropic이 Claude Opus의 새로운 버전인 Opus 4.8을 공개했습니다. Opus 4.7을 기반으로 전반적인 성능을 향상시켰으면서도 같은 가격대를 유지하고 있어요.

## 동시 공개되는 새로운 기능들

Opus 4.8과 함께 여러 신기능이 함께 론칭되었습니다. claude.ai 사용자들은 이제 Claude가 작업에 투입하는 노력 수준을 직접 조절할 수 있게 되었어요. Claude Code에는 대규모 문제를 해결할 수 있는 "동적 워크플로우"라는 새 기능도 추가되었습니다. 그리고 Opus 4.8의 빠른 모드는 2.5배 속도로 작동하면서 이전 모델 대비 가격이 3분의 1 수준으로 떨어졌습니다.

## Opus 4.8의 성능 개선

코딩, 에이전트 작업, 추론, 실무 기반 작업 등 여러 벤치마크에서 Opus 4.8이 이전 버전과 다른 모델들을 어떻게 상회하는지 보여주는 테이블이 있습니다. 더 자세한 내용과 광범위한 성능 평가는 Claude Opus 4.8 System Card에서 확인할 수 있어요.
