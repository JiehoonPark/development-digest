## 핵심 개념

> [!abstract]
> GitHub은 Windows 0-day 익스플로잇을 공개한 보안 연구자의 계정을 정지했으며, 전문가들은 이 조치가 과도하다고 비판하고 있습니다.

## 상세 내용

- 보안 연구자의 정당한 취약점 공개 활동이 규제됨
- 공개 책임감(responsible disclosure)과 플랫폼 정책의 충돌

> [!tip] 왜 중요한가
> 보안 연구 커뮤니티의 자유도와 기업 정책 간의 긴장 관계가 개발자의 활동 환경에 직접 영향을 미칩니다.
