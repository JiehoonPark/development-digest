## 핵심 개념

> [!abstract]
> LLM으로 작성된 텍스트와 웹사이트에서 나타나는 일관된 패턴(em-dash 과다 사용, 반복적인 짧은 문장, 특정 UI 컴포넌트 등)을 식별하는 분석 기사입니다.

## 상세 내용

- LLM 생성 텍스트의 구조적 특징: 과도한 말줄임표, 'X는 Y의 Z' 패턴, 연속된 짧은 문장
- AI 생성 웹사이트의 시각적 특징: JetBrains Mono 폰트, 동일한 버튼/카드 디자인, 특정 배지 애니메이션

> [!tip] 왜 중요한가
> AI 생성 콘텐츠의 특징을 인식함으로써 콘텐츠 품질 검토와 원본성 평가 능력을 향상시킵니다.
