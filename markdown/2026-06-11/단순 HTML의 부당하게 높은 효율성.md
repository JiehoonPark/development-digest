## 핵심 개념

> [!abstract]
> 2021년 작성된 글로, 복잡한 프레임워크 없이 단순한 HTML만으로도 효과적인 웹 구축이 가능함을 주장합니다.

## 상세 내용

- HTML의 단순함이 유지보수성과 접근성을 높인다
- 과도한 JavaScript 의존성 없이도 기능적 웹사이트 구현 가능

> [!tip] 왜 중요한가
> 웹 개발의 기본으로 돌아가 불필요한 복잡성을 제거하는 것의 가치를 일깨웁니다.
