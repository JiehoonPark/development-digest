## 핵심 개념

> [!abstract]
> Context Mode는 AI 코딩 에이전트의 컨텍스트 윈도우를 최적화하는 TypeScript 도구로, 도구 출력을 샌드박싱하여 토큰 사용량을 98% 감소시킨다. 12개 플랫폼을 지원하며 LLM 기반 코딩 에이전트의 비용 효율성과 성능을 크게 개선한다.

## 상세 내용

- 도구 출력 샌드박싱으로 98% 토큰 사용량 감소 달성. 불필요한 출력을 필터링하여 컨텍스트 윈도우 효율을 극대화함으로써 API 비용을 대폭 절감
- 12개 플랫폼 지원으로 광범위한 호환성 제공. 다양한 AI 코딩 에이전트 환경에서 일관되게 작동 가능
- AI 에이전트의 응답 속도와 정확도 향상. 불필요한 토큰을 제거하여 모델이 더 빠르고 정확한 코드 생성에 집중

> [!tip] 왜 중요한가
> AI 코딩 에이전트 비용이 지속적으로 증가하는 상황에서 이 도구는 토큰 사용을 대폭 줄여 LLM 기반 개발 도구의 경제성을 획기적으로 개선한다.
