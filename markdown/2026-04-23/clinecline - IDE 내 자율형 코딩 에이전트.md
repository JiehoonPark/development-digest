## 핵심 개념

> [!abstract]
> Cline은 IDE 내에서 직접 작동하는 자율형 코딩 에이전트로, 사용자의 권한 하에 파일 생성/편집, 명령 실행, 브라우저 사용 등의 작업을 수행합니다.

## 상세 내용

- IDE에 직접 통합된 자율형 코딩 에이전트로, 단계별 사용자 승인 메커니즘을 통해 파일 생성, 수정, 명령 실행 등을 처리합니다.
- 브라우저 사용 기능까지 포함하여 개발 작업의 자동화 범위를 확대합니다.

> [!tip] 왜 중요한가
> IDE 내 통합형 AI 에이전트로 개발자의 생산성을 향상시키고 반복적인 코딩 작업을 자동화할 수 있습니다.
