## 핵심 개념

> [!abstract]
> Mark Erikson의 블로그 포스트로, React Compiler 출시로 인한 React 렌더링의 기본 개념과 고급 기능 변화를 설명하는 프레젠테이션 가이드이다. 저자는 React 렌더링 메커니즘을 수천 명의 개발자에게 설명해온 경험을 바탕으로 React Compiler가 기존 렌더링 방식을 어떻게 변화시키는지 해석한다.

## 상세 내용

- React Compiler의 등장으로 React 렌더링 최적화의 패러다임 전환 예상. useMemo, useCallback 같은 수동 최적화가 자동으로 처리될 수 있으며, 이는 개발자의 성능 튜닝 부담을 줄임
- 저자가 과거 발표한 React 렌더링 기초 자료가 가장 많이 읽힌 콘텐츠. 이는 개발자들이 렌더링 메커니즘 이해에 얼마나 큰 어려움을 겪어왔는지 증명

> [!tip] 왜 중요한가
> React Compiler의 도입으로 React 렌더링 방식이 근본적으로 변화하고 있으므로, 이 가이드는 기존 이해를 업데이트하고 새로운 최적화 방식을 학습할 필수 자료다.
