## 핵심 개념

> [!abstract]
> Microsoft의 ONNX Runtime은 다양한 플랫폼에서 머신러닝 모델의 고속 추론과 학습을 지원하는 C++ 기반 프레임워크이다.

## 상세 내용

- 크로스 플랫폼 호환성으로 일관된 ML 워크플로우 구현. CPU, GPU, TPU 등 다양한 하드웨어에서 동일한 모델 실행 가능
- 고성능 추론 가속으로 프로덕션 환경 최적화. 모델 최적화와 연산 그래프 변환을 통해 추론 속도 극대화

> [!tip] 왜 중요한가
> 프로덕션 환경에서 ML 모델의 신뢰성 있는 배포와 성능 최적화가 필요한 개발자에게 산업 표준 추론 엔진을 제공한다.
