## 핵심 개념

> [!abstract]
> 천문학을 좋아하는 가족을 위해 포치에 설치한 태양계 행성 LED 조명을, React Native 기반 TV 개발 플랫폼인 Amazon Vega로 거실 프로젝션 스크린과 연동한 취미 프로젝트를 소개합니다. VS Code 확장과 로컬 시뮬레이터, CLI 도구 덕분에 개발이 예상보다 쉬웠고, WLED 컨트롤러 제어는 fetch API로, 음성 명령은 aria-label만 추가해 간단히 구현했습니다. React/React Native 지식만 있으면 큰 어려움 없이 Fire TV용 앱을 만들 수 있다는 경험담입니다.

## 아티클

## 취미 프로젝트로 알아보는 Amazon Vega: React Native로 만드는 TV 앱

집 안 포치(porch)를 태양계 행성 모형으로 꾸미고 LED로 조명을 밝히는 개인 프로젝트를 진행하다가, 이를 거실 프로젝션 스크린과 연결하고 싶어진 사례를 소개합니다. 이 과정에서 Amazon이 내놓은 Fire TV용 개발 플랫폼 **Vega**를 사용해봤는데, React와 React Native 경험이 있다면 생각보다 훨씬 쉽게 TV 화면까지 애플리케이션을 올릴 수 있었다는 경험을 공유합니다.

## 프로젝트 배경: 포치의 태양계와 LED 조명

이 프로젝트를 진행한 사람은 천문학에 관심이 많은 가족을 위해 3D 프린터로 출력한 행성 모형들을 포치 뒤편에 나란히 배치하고, 각 행성에 제어 가능한 LED 조명을 달아 꾸며두었습니다. 영화를 볼 때 분위기를 살리기에 좋았지만, 여기서 한 걸음 더 나아가 포치의 행성 조명과 거실의 대형 프로젝션 스크린을 연결하고 싶다는 아이디어가 떠올랐습니다.

## 왜 Amazon Vega를 선택했나

여러 옵션을 살펴본 끝에 Fire TV Stick과 Amazon Vega 조합을 선택했는데, 그 이유는 단순합니다. Vega가 UI 구현에 **React Native**를 사용하기 때문입니다. 평소 React와 React Native에 익숙했던 만큼, 새로운 플랫폼을 처음부터 익혀야 하는 부담 없이 곧바로 개발에 뛰어들 수 있었습니다. 실제로 작업해본 결과, 예상보다 훨씬 수월했다고 합니다.

## 개발 경험: VS Code 확장, 시뮬레이터, AI 코딩 에이전트

Vega 개발 과정에서 특히 도움이 됐던 요소들은 다음과 같습니다.

- **VS Code용 Vega 확장**: 로컬 시뮬레이터가 내장되어 있어 실제 기기 없이도 코드를 손쉽게 디버깅할 수 있었습니다.
- **AI 코딩 에이전트와의 궁합**: React Native 기반이다 보니 AI 코딩 에이전트들이 이미 관련 코드를 잘 다뤘고, 여기에 Vega 공식 문서를 참고 자료로 던져주는 것만으로 세부 구현을 무난하게 진행할 수 있었습니다.
- **CLI 도구**: 프로젝트를 처음 시작할 때 필요한 스캐폴딩을 CLI 앱이 제공해서 초기 세팅 부담이 크지 않았습니다.

## 결과물: 태양계 지도와 조명 제어

완성된 앱은 손님이 놀러 왔을 때 화면에 태양계 지도를 띄우고 각 행성에 대한 정보를 보여줄 수 있습니다. 리모컨의 D-pad를 이용해 포치에 설치된 조명들과 화면 속 행성 지도를 함께 탐색할 수 있는데요. LED 제어는 **WLED 컨트롤러**를 통해 이뤄지며, Vega 앱에서 일반적인 `fetch`를 사용해 HTTP로 직접 명령을 전송하는 방식으로 구현했습니다.

## 마무리 포인트: 음성 인식으로 완성도 높이기

프로젝트의 완성도를 한층 끌어올리기 위해 행성 이름을 말하면 화면과 조명에 해당 행성이 표시되도록 음성 인식 기능을 추가했습니다. 이 부분이 특히 인상적이었는데, 별도의 복잡한 음성 처리 로직을 구현할 필요 없이 컴포넌트에 `aria-label`만 추가하면 됐습니다. Vega가 음성 인식 관련 작업을 전부 알아서 처리하고, 사용자가 이름을 말하면 해당 요소의 press 이벤트를 자동으로 발생시켜주기 때문입니다. 예를 들어 "Saturn"이라고 말하면 토성 관련 화면과 조명이 바로 반응하는 식인데, 이 매끄러운 경험이 프로젝트 전체를 한층 생동감 있게 만들어줬습니다.

## 정리

- Amazon Vega는 React Native 기반의 Fire TV용 개발 플랫폼으로, 기존 React/React Native 지식을 그대로 활용해 TV 앱을 만들 수 있습니다.
- VS Code 확장과 로컬 시뮬레이터 덕분에 실기기 없이도 빠른 디버깅이 가능하고, CLI로 프로젝트를 손쉽게 시작할 수 있습니다.
- React Native 생태계 위에 있다 보니 AI 코딩 에이전트의 지원을 받기도 수월하며, 공식 문서를 참고 자료로 활용하는 것만으로 충분했습니다.
- 외부 기기 제어(WLED LED 컨트롤러)도 표준 `fetch` API로 간단히 HTTP 요청을 보내는 방식으로 구현 가능합니다.
- 음성 명령 지원은 `aria-label` 같은 접근성 속성만 추가하면 Vega가 음성 인식과 이벤트 처리를 알아서 담당해주므로, 별도 음성 인식 로직 없이도 자연스러운 음성 UX를 붙일 수 있습니다.
