## 핵심 개념

> [!abstract]
> 《Hello 算法》는 애니메이션과 그래프를 통해 설명하는 데이터구조와 알고리즘 교과서로, 원버튼 실행 기능을 지원합니다. Python, Java, C++, C, C#, JavaScript, Go, Swift, Rust, Ruby, Kotlin, TypeScript, Dart 등 13가지 프로그래밍 언어로 코드 구현을 제공합니다.

## 상세 내용

- 동적 애니메이션과 도식을 활용하여 알고리즘과 데이터구조의 작동 원리를 직관적으로 설명하며, 추상적인 개념을 시각화하여 이해도를 높입니다.
- 13가지 프로그래밍 언어(Python, Java, C++, C, C#, JS, Go, Swift, Rust, Ruby, Kotlin, TS, Dart)로 동일한 알고리즘 구현을 제공하여 언어 간 비교 학습이 가능합니다.
- 간체중문, 번체중문, 영어, 일본어 등 다국어 지원으로 세계적인 학습자들이 접근할 수 있으며, 각 언어 커뮤니티의 참여를 통해 지속적으로 개선됩니다.
- 원버튼 실행 기능으로 학습자가 바로 코드를 테스트할 수 있어, 이론 학습과 실습을 즉시 연결할 수 있는 구조를 제공합니다.

> [!tip] 왜 중요한가
> 알고리즘과 자료구조를 시각적으로 이해하고자 하는 개발자들에게 직관적인 학습 자료를 제공하며, 여러 언어 구현을 한 곳에서 비교할 수 있어 언어 선택 시 참고 자료로 활용할 수 있습니다.
