## 핵심 개념

> [!abstract]
> AI 에이전트가 Twitter, Reddit, YouTube, GitHub, Bilibili, XiaoHongShu 등 다양한 플랫폼에 접근해 정보를 검색하고 수집할 수 있게 해주는 Python 기반 CLI 도구이다.

## 상세 내용

- 하나의 CLI로 여러 플랫폼에 API 비용 없이 접근 가능
- AI 에이전트의 정보 수집 능력 확장

> [!tip] 왜 중요한가
> AI 에이전트 애플리케이션 개발 시 다양한 소스에서 실시간 정보를 효율적으로 수집할 수 있다.
