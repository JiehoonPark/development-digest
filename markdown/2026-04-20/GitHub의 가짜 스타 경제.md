## 핵심 개념

> [!abstract]
> GitHub 저장소의 가짜 스타 증가 현상과 그것이 만드는 왜곡된 생태계를 조사합니다.

## 상세 내용

- GitHub 스타 수를 조작하는 관행이 프로젝트의 실제 품질 평가를 방해
- 개발자 커뮤니티의 신뢰도와 평가 체계의 신뢰성 문제 제기

> [!tip] 왜 중요한가
> 프로젝트 선택 시 스타 수만으로 판단하는 것의 위험성을 인식하고 더 신중한 평가가 필요합니다.

## 전문 번역

저희는 Google Analytics를 통해 방문자가 웹사이트와 어떻게 상호작용하는지 파악하고 있습니다. 어떤 페이지가 인기 있는지, 방문자가 어디서 유입되는지, 사이트를 어떻게 탐색하는지 등을 분석하죠. 이 정보는 콘텐츠와 사용자 경험을 개선하는 데 활용되며, 수집된 데이터는 Google으로 전송됩니다.
