## 핵심 개념

> [!abstract]
> React Server Components(RSC)가 번들러와 깊게 통합되는 이유와 필요성을 설명하는 글입니다. 모듈 직렬화의 복잡성과 RSC의 동작 원리를 다룹니다.

## 상세 내용

- RSC는 단순한 직렬화 이상의 복잡한 처리가 필요해 번들러 통합이 필수적
- 번들러 레벨의 통합을 통해 서버와 클라이언트 코드의 경계를 자동으로 처리

> [!tip] 왜 중요한가
> RSC 기반 프로젝트를 구축하는 개발자가 번들러 통합의 필요성을 이해하고 올바르게 구성하는 데 도움이 됩니다.
