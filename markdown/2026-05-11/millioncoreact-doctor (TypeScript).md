## 핵심 개념

> [!abstract]
> React 코드의 성능 문제와 안티패턴을 자동으로 감지하는 에이전트 도구이다. TypeScript로 구현되었으며, 개발자가 작성한 미흡한 React 코드를 분석하여 문제를 지적한다.

## 상세 내용

- AI 에이전트 기반의 React 코드 리뷰 도구로, 성능 문제와 잘못된 패턴을 자동으로 검출한다.
- 개발 중 React 컴포넌트의 렌더링 최적화, 훅 사용법, 상태 관리 등 일반적인 실수를 미리 파악할 수 있다.
- TypeScript 타입 안정성과 함께 런타임 패턴 검사를 수행한다.

> [!tip] 왜 중요한가
> React 개발 시 자주 발생하는 성능 문제와 베스트 프랙티스 위반을 조기에 감지하여 코드 품질 향상을 도와준다.
