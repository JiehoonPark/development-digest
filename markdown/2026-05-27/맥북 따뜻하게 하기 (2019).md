## 핵심 개념

> [!abstract]
> 추운 환경에서 맥북 키보드가 차가워지는 문제를 해결하기 위해 CPU를 100% 사용하도록 부하를 주는 방법을 소개한다. 터미널 명령어나 stress 유틸리티를 사용하여 프로세서를 집중적으로 작동시켜 열을 발생시킬 수 있다.

## 상세 내용

- yes 명령어로 CPU를 100% 사용률까지 높이는 단순한 방법 제공
- stress 유틸리티를 사용해 6개 스레드로 CPU를 부하시키고 300초 타임아웃 설정 가능
- bash-profile에 alias 추가로 자동화할 수 있음

> [!tip] 왜 중요한가
> 추운 날씨에서 개발자가 노트북을 빠르게 사용 가능한 상태로 만들 수 있는 실용적인 팁이다.

## 전문 번역

# 얼어붙은 노트북을 빠르게 데우는 방법

겨울이 되면 정말 그런 경험이 있을 거예요. 추운 차에서 배낭을 꺼내고, 위스콘신의 매서운 바람을 맞으며 걸어서 출근하거나, 얼어붙은 호수를 건너 자전거를 타고 직장에 도착합니다. 그리고 드디어 앉아서 키보드에 손을 올리는 순간, 금속에서 흡수되는 차가운 열기에 깜짝 놀라 손을 뺍니다.

얼어붙은 노트북을 빠르게 데우려면 어떻게 할까요? 간단합니다. 컴퓨터에 일을 많이 시키면 됩니다.

## 기본 방법: yes 명령어

터미널에서 실행할 수 있는 한 줄짜리 내장 명령어가 있습니다. 이것은 CPU를 100%까지 끌어올립니다:

```bash
yes > /dev/null &
```

터미널에서 이 명령어를 실행하면 되는데, 실행 후에는 잊지 마세요(웃음). 이 명령어는 'yes'라는 단어를 반복해서 null 디바이스로 보내면서 CPU를 전부 사용하게 되는 거죠.

## 더 빠르게 데우고 싶다면: stress 유틸리티

Mac에서 좀 더 빠르고 강력하게 CPU를 가열하고 싶다면 stress 유틸리티를 사용해보세요:

```bash
stress --cpu 6 --vm 2 --timeout 300s
```

이 명령어는 CPU를 100%로 사용하는 스레드 6개와 메모리 할당/해제 작업을 하는 스레드 2개를 실행합니다. 그리고 혹시 컴퓨터를 잊어버리고 과열되지 않도록 300초(5분) 뒤에 자동으로 종료됩니다.

## 한 단계 더: 별칭 설정

보너스 팁으로, ~/.bash_profile에 별칭을 추가해서 이런 옵션들을 자동화할 수도 있어요:

```bash
alias warmup='stress --cpu 6 --vm 2 --timeout 300s'
```

이렇게 설정해두면 앞으로는 `warmup` 명령어만 입력하면 됩니다.
