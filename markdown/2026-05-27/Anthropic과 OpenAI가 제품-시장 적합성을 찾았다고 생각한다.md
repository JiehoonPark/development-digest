## 핵심 개념

> [!abstract]
> OpenAI와 Anthropic이 2026년 4월 엔터프라이즈 가격을 API 토큰 기반으로 전환하며, 코딩 에이전트 같은 고사용량 제품이 진정한 수익성 있는 사업 모델을 만들어내고 있다. 개인 파워 유저 기준 월 $200 구독으로 $2,000+ 상당의 API 사용량을 얻을 수 있으며, 기업은 이러한 가격 책정으로 실제 고비용을 지불하기 시작했다.

## 상세 내용

- OpenAI와 Anthropic이 엔터프라이즈 요금제를 API 토큰 기반 가격으로 통일, 기존 할인 제거로 기업의 실제 비용 증가
- 코딩 에이전트(Claude Code/Cowork, Codex)가 대량의 토큰을 소비하면서도 개발자들의 필수 도구가 되어 고부가가치 고객 확보
- OpenAI와 Anthropic이 채용 공고 대비 26-32%를 엔터프라이즈 판매/지원 직군으로 채우며 B2B 사업 집중

> [!tip] 왜 중요한가
> AI 모델 가격 책정 구조가 근본적으로 변화하고 있으며, 개발자는 AI 도구 구독 전략을 다시 검토해야 한다.

## 전문 번역

# AI 모델 기업들이 드디어 돈을 벌기 시작했다

Anthropic이 첫 흑자 분기를 앞두고 있다는 소문이 돌고 있습니다. 여러 회사들이 직원들의 LLM 사용으로 인한 청구액의 규모에 깜짝 놀라는 일이 빈번해졌거든요. 제 생각엔 OpenAI와 Anthropic이 드디어 제대로 된 시장 기회를 찾아낸 것 같습니다.

저는 현재 Anthropic의 월 $100짜리 Max 요금제와 OpenAI의 월 $100짜리 Pro 요금제를 구독 중입니다. 코딩 에이전트를 자주 사용하는 사람이라면 이 정도 요금은 정말 훌륭한 거래예요. 최근 제 노트북에서 ccusage 도구를 실행해서 지난 30일간의 API 토큰 사용료를 계산해봤습니다.

**월 $200의 요금으로 $2,180.16어치의 토큰을 사용했네요**—꽤 괜찮은 교환 비율입니다. 저는 이 도구들을 꽤 자주 사용하는 편이지만, 하루 종일 에이전트를 돌리거나 그러지는 않습니다.

## 기업들이 예상 밖으로 비싼 청구를 받고 있다

저는 에이전트를 광범위하게 사용하는 기업들도 비슷한 수준의 할인을 받을 거라고 생각했는데, 정말 틀렸어요.

정확한 시점을 파악하기는 어렵지만, 지난 6개월 사이에 Anthropic이 Enterprise 요금제를 개편했습니다. 원래는 "Claude 시트에 전형적인 업무일 정도의 사용량이 포함된다"는 식이었는데(2025년 8월 기준), 이제는 **월 $20/시트 + 초과 사용량에 대한 API 가격**으로 변경된 것입니다. The Information의 기사(2026년 4월 14일자)에 따르면 Anthropic 대변인은 이 변경이 2025년 11월에 시행되었다고 주장했는데요, 기존 고객들은 계약 갱신할 때 이 변화를 알게 되는 상황입니다.

OpenAI도 4월에 비슷한 조치를 취했습니다. 현재의 Codex 요금표를 보면 이런 공지가 있습니다:

> **2026년 4월 2일**, Codex 요금을 메시지당 가격에서 API 토큰 사용량 기반으로 변경했습니다. 이는 신규 및 기존의 Plus, Pro, ChatGPT Business, 신규 ChatGPT Enterprise 요금제에 적용됩니다.
>
> **2026년 4월 23일**, 기존의 모든 ChatGPT Enterprise 요금제(Edu, Health, Gov, ChatGPT for Teachers 포함)에도 동일하게 적용했습니다.

OpenAI는 가격을 "크레딧" 단위로 표시하는 바람에 조금 복잡하지만, 조사해본 바로는 이 크레딧 가격이 API 토큰 가격과 정확히 일치합니다.

## 엔터프라이즈 가격 = API 가격

결국 **2026년 4월부터 OpenAI Codex와 Anthropic Claude Code/Cowork의 Enterprise 요금은 공시된 API 가격과 동일**해졌습니다.

새로 출시된 모델들도 가격이 올랐어요. 4월 23일 출시된 GPT-5.5는 GPT-5.4의 2배 가격이고, 4월 16일 출시된 Opus 4.7은 새로운 토크나이저를 고려하면 Opus 4.6 대비 약 1.4배 정도입니다.

4월은 두 회사 모두 새로운 최신형 모델을 더 비싼 API 가격으로 출시했고, 동시에 장기 계약(보통 1년)을 체결하는 엔터프라이즈 고객들을 그 새로운 API 가격대에 잠그는 조치를 취한 달이었습니다.

## 왜 갑자기 가격을 올렸을까?

두 회사 모두 IPO를 준비 중이지만, 제 생각엔 더 근본적인 이유가 있습니다. **코딩 에이전트와 범용 에이전트 제품(Claude Code/Cowork, Codex)이 진정한 시장 기회를 찾아냈다는 것**입니다.

ChatGPT 같은 도구는 엄청나게 인기가 많습니다. 하지만 이 인기를 실제 수익으로 바꾸기는 어려웠거든요. OpenAI가 2월에 공개한 통계를 보면 ChatGPT의 주간 활성 사용자는 9억 명이 넘지만, 유료 구독자는 **5천만 명—단 5.6%**에 불과합니다.

월 $10~$20을 내는 개별 사용자들로는 그다지 매력적인 사업이 아닙니다. $1조 규모의 인프라를 4년 동안 커버하려면 10~20억 명의 구독자가 필요하거든요.

하지만 **사용자 1명당 월 $200 이상을 쓰는 기업 고객들은 완전히 다른 이야기**입니다. 한 명당 벌어들이는 수익이 훨씬 크니까요. 저만 해도 이미 공급사당 월 ~$1,000 정도의 API 비용을 소비하고 있습니다.

## 코딩 에이전트가 게임을 바꿨다

코딩 에이전트는 정말 다릅니다. 훨씬 많은 토큰을 소비하지만, 동시에 고임금 전문가들의 **일상 업무 도구**가 되어가고 있거든요. 지금은 주로 소프트웨어 엔지니어들이 사용하지만, 에이전트란 결국 "컴퓨터에 타이핑으로 할 수 있는 모든 작업"을 자동화할 수 있는 도구입니다. 따라서 훨씬 더 광범위한 지식 근로자들에게 적용될 잠재력이 있습니다.

제가 이 블로그에서 자세히 다뤘듯이, **2025년 11월에 출시된 모델들이 에이전트를 진짜 유용한 수준으로 끌어올렸습니다**. 이제 6개월째 우리가 이 현실에 익숙해지고 있는데, 기업들이 본격적으로 돈을 쓰기 시작한 건 당연한 일이죠.

ChatGPT가 2023년 2월에 역대 가장 빠르게 성장한 소비자 앱이 된 것을 "제품-시장 적합성 달성"이라고 할 수는 있습니다. 하지만 당시에는 실제로 돈을 벌지 못했으니까요. **코딩 에이전트 + 엔터프라이즈 가격 책정이 바로 이 회사들이 진정한 수익을 창출하기 시작하는 순간**입니다. 어쩌면 드디어 비용을 충당할 만큼 말이에요!

## 채용 공고가 말해주는 것

엔터프라이즈 에이전트가 실제 시장 기회라는 또 다른 증거는 이들의 채용 공고를 보면 알 수 있습니다.

OpenAI는 현재 703개의 공개 채용 공고를 올려놓았는데, 이 중 **229개(32.6%)가 엔터프라이즈 영업 및 지원 관련**입니다. Account Executive, Go To Market, Forward Deployed Engineer 같은 직책들이죠.

Anthropic은 390개의 공개 채용 공고 중 **105개(26.9%)가 엔터프라이즈 관련**입니다.

회사가 엔터프라이즈 영업에 이토록 공격적으로 자원을 집중하는 것만 봐도, 그들이 어디에 미래를 걸고 있는지 명확합니다.
