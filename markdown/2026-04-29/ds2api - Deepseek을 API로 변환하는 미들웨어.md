## 핵심 개념

> [!abstract]
> Go로 작성된 경량 고성능 미들웨어로, Deepseek을 다양한 API 형식으로 변환하며 다중 계정 로테이션, Vercel 서버리스, Docker를 지원합니다.

## 상세 내용

- Google, Claude, OpenAI API 형식과 호환되는 프로토콜 변환
- 다중 계정 로테이션 및 컴파일된 바이너리 제공
- Vercel Serverless와 Docker 배포 지원

> [!tip] 왜 중요한가
> LLM API 통합이 필요한 개발자들이 다양한 AI 모델을 통일된 인터페이스로 사용할 수 있게 합니다.
