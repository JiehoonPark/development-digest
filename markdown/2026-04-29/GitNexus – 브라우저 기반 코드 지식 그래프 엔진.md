## 핵심 개념

> [!abstract]
> GitHub 저장소나 ZIP 파일을 브라우저에서 직접 분석하여 인터랙티브한 지식 그래프를 생성하는 클라이언트 사이드 도구이며, 내장된 Graph RAG 에이전트로 코드 탐색이 가능하다.

## 상세 내용

- 클라이언트 사이드에서 실행되어 서버 없이 코드베이스 분석 가능
- 지식 그래프 시각화와 Graph RAG 에이전트를 통한 코드 탐색 지원

> [!tip] 왜 중요한가
> 개발자가 대규모 프로젝트의 코드 구조를 빠르게 이해하고 의존성을 시각적으로 파악할 수 있는 도구를 로컬에서 사용 가능하다.
