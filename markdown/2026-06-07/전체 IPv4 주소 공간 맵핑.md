## 이 서비스에서 할 수 있는 것들

**IP 주소 조회** — 8.8.8.8 같은 IPv4 주소를 입력하면 소유 기관, ASN, 도시, 국가, 역방향 DNS를 확인할 수 있습니다.

**CIDR 대역 조회** — 8.8.8.0/24처럼 Google의 DNS 블록에 포함된 256개 IP를 모두 살펴볼 수 있습니다.

**ASN(자율 시스템) 조회** — BGP로 라우팅되는 주소 블록을 추적해서 운영 기관을 파악합니다. 예를 들어 AS15169는 Google, AS13335는 Cloudflare죠.

**기관별 IP 풋프린트** — 특정 기관이 보유한 모든 CIDR 블록과 운영 국가를 확인할 수 있습니다.

**국가, 시/도, 도시별 분석** — 250개 국가에서 3,000개 이상의 시/도를 거쳐 120,000개 이상의 도시 단계까지 내려가며 각 레벨에서 상위 ISP와 IP 할당 현황을 볼 수 있습니다.

**IPv4 고갈 현황 추적** — 43억 개의 IPv4 주소 공간 중 얼마나 할당되었는지, RIR별 분석 결과는 어떤지 확인할 수 있습니다.

## WorldIP.io는 어떻게 작동하나요?

WorldIP.io는 세 가지 신뢰할 수 있는 IPv4 데이터 출처를 통합합니다.

- **MaxMind GeoLite2** — 도시 단위 지리적 위치 정보
- **RIR 위임 파일** (ARIN, RIPE, APNIC, LACNIC, AfriNIC) — 할당 소유권
- **Team Cymru** — ASN과 기관명 매핑

모든 IPv4 주소는 고유한 URL을 가지고 있습니다. `/1.1.1.1`에 접속하면 해당 IP가 속한 대역의 소유 기관(Cloudflare), ASN(AS13335), BGP 라우팅 경로, MaxMind 지리 정보, 역방향 DNS 레코드, 같은 /24 대역의 인접한 16개 IP 정보를 한눈에 볼 수 있고, 모두 상호 연결되어 있습니다.

데이터는 MaxMind에서 매주 갱신됩니다. 자세한 내용은 데이터 출처 페이지나 FAQ를 참고하세요.

## 자주 묻는 질문

**IPv4 주소란 무엇인가요?**
IPv4 주소는 인터넷에 연결된 모든 기기에 할당되는 32비트 숫자 식별자입니다. 8.8.8.8처럼 점으로 구분된 네 개의 숫자로 표기하며, 약 43억 개의 IPv4 주소가 존재합니다. 각 주소는 기관, ISP, 또는 정부 기관에 할당됩니다.

**IP 주소 소유자를 어떻게 찾나요?**
위의 검색창에 IPv4 주소를 입력하면 됩니다. WorldIP.io는 모든 할당된 IPv4 주소에 대해 소유 기관, ASN(자율 시스템 번호), 지리적 위치, CIDR 대역, 역방향 DNS 레코드를 보여줍니다.

**ASN이란 무엇인가요?**
자율 시스템 번호(ASN)는 자체 라우팅 정책을 독립적으로 관리하는 네트워크를 식별합니다. ISP, 클라우드 공급자, 대형 엔터프라이즈는 각각 하나 이상의 ASN을 보유하고 있습니다. WorldIP.io는 79,232개 이상의 활성 ASN을 추적합니다.

**CIDR 대역이란 무엇인가요?**
8.8.8.0/24 같은 CIDR(무분류 도메인 간 라우팅) 표기법은 IP 주소 블록을 나타냅니다. 슬래시 뒤의 숫자가 대역에 포함된 주소 개수를 결정합니다. /24는 256개, /16은 65,536개, /8은 1,670만 개의 주소를 포함합니다.

**역방향 DNS(PTR) 레코드란 무엇인가요?**
역방향 DNS 레코드는 IP 주소를 도메인명으로 변환합니다. 일반적인 DNS는 도메인을 IP로 변환하지만, PTR 레코드는 반대 방향입니다.
