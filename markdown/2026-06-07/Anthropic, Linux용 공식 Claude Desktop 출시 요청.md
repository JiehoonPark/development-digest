## 전문 번역

# Linux(Ubuntu LTS/Debian)용 Claude Desktop 공식 빌드 추가 요청

## 개요

현재 Claude Desktop은 Windows와 macOS에서만 공식 지원되고 있습니다. Linux 사용자들, 특히 Ubuntu LTS와 Debian을 쓰는 개발자들도 Claude Desktop의 네이티브 애플리케이션 경험을 원하고 있는 상황입니다.

이 기능 요청은 Ubuntu LTS와 Debian 배포판을 대상으로 한 공식 Claude Desktop 빌드를 제공해달라는 것입니다.

## 필요한 이유

Linux는 개발자 커뮤니티에서 점점 더 많이 사용되는 플랫폼입니다. 특히 서버 개발이나 DevOps 업무를 하는 개발자들에게는 필수적인 운영체제죠. 하지만 현재 Claude Desktop이 Linux에서 공식 지원되지 않다 보니, 이런 개발자들은 웹 브라우저를 통해서만 Claude에 접근해야 하는 불편함을 겪고 있습니다.

공식 Linux 빌드가 제공된다면:
- Linux 개발 환경과의 더 나은 통합이 가능합니다
- 네이티브 애플리케이션의 성능과 안정성을 경험할 수 있습니다
- Ubuntu LTS와 Debian이라는 안정적인 배포판을 지원함으로써 기업 환경에서도 신뢰할 수 있습니다

## 제안사항

Ubuntu LTS(장기 지원 버전)와 Debian 안정판을 우선적으로 지원하는 것을 권장합니다. 이 두 배포판은 충분한 사용자 기반을 가지고 있으면서도 상대적으로 버전 호환성이 좋기 때문입니다.
