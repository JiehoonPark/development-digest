## 핵심 개념

> [!abstract]
> DFlash는 블록 확산 기법을 활용한 플래시 추측 디코딩 방식을 제안하는 Python 라이브러리입니다. 대규모 언어 모델의 추론 속도를 향상시키는 새로운 접근법을 제공합니다.

## 상세 내용

- 블록 확산 기반의 추측 디코딩 최적화 기법 제시
- LLM 추론 성능 개선을 목표로 하는 오픈소스 프로젝트

> [!tip] 왜 중요한가
> LLM 추론 성능 최적화는 실제 배포 환경에서의 비용 절감과 응답 시간 개선에 직결됩니다.
