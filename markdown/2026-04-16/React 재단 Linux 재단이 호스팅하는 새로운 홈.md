## 핵심 개념

> [!abstract]
> React 재단이 공식적으로 출범하여 Linux 재단의 지원 하에 운영된다.

## 상세 내용

- 공식 거버넌스 확립: React가 Linux 재단의 후원을 받으며 공식적인 오픈소스 재단 구조로 전환되어 투명한 거버넌스와 커뮤니티 운영이 보장된다.
- 프로젝트 지속성: 중립적인 재단 체계를 통해 React의 장기적 지속성과 신뢰성이 강화된다.

> [!tip] 왜 중요한가
> React의 공식 재단화는 프로젝트의 거버넌스 투명성을 높이고 생태계 안정성을 보장한다.
