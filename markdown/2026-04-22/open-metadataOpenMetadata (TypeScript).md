## 핵심 개념

> [!abstract]
> 중앙 메타데이터 저장소, 컬럼 레벨 계보 추적, 팀 협업을 통해 데이터 디스커버리, 데이터 옵저버빌리티, 데이터 거버넌스를 제공하는 통합 메타데이터 플랫폼입니다.

## 상세 내용

- 중앙화된 메타데이터 저장소로 조직 전체의 데이터 자산을 관리합니다.
- 컬럼 레벨의 상세한 lineage 추적으로 데이터 출처와 흐름을 명확히 파악할 수 있습니다.

> [!tip] 왜 중요한가
> 데이터 플랫폼을 운영하는 팀이 데이터 거버넌스와 컴플라이언스를 효율적으로 관리할 수 있게 도와줍니다.
