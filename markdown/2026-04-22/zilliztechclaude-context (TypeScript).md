## 핵심 개념

> [!abstract]
> Claude Code를 위한 코드 검색 MCP(Model Context Protocol)로, 전체 코드베이스를 모든 코딩 에이전트의 컨텍스트로 활용 가능합니다.

## 상세 내용

- Claude Code 사용자가 대규모 프로젝트에서 효율적으로 작업할 수 있도록 코드베이스 전체를 컨텍스트로 제공합니다.
- TypeScript 기반의 MCP 구현으로 AI 코딩 에이전트의 코드 이해도를 향상시킵니다.

> [!tip] 왜 중요한가
> AI 코딩 에이전트를 사용하는 개발자에게 더 정확한 코드 컨텍스트를 제공하여 생성 코드의 품질을 높일 수 있습니다.
