## 핵심 개념

> [!abstract]
> 이 영상은 React Server Components(RSC)를 활용하여 페이지의 특정 부분만 효율적으로 캐싱하는 기법을 다룹니다.

## 상세 내용

- RSC 캐싱 기본 개념: React Server Components를 사용하면 서버에서 렌더링된 컴포넌트를 캐싱하여 불필요한 재렌더링 방지 가능
- 선택적 캐싱: 동적 부분은 캐싱 제외하고 정적 부분만 캐싱하여 신선도와 성능의 균형 유지
- 성능 최적화: 부분 캐싱으로 초기 로드 시간 단축 및 서버 부하 감소

> [!tip] 왜 중요한가
> Next.js와 React 18+ 환경에서 RSC의 강력한 캐싱 기능을 올바르게 활용하면 애플리케이션의 성능을 크게 향상시킬 수 있습니다.
