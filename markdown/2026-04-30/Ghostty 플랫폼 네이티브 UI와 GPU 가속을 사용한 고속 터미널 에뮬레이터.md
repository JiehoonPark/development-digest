## 핵심 개념

> [!abstract]
> Ghostty는 Zig로 구축된 빠르고 기능이 풍부한 크로스플랫폼 터미널 에뮬레이터로, 플랫폼 네이티브 UI와 GPU 가속을 활용합니다.

## 상세 내용

- 크로스플랫폼 지원: 여러 운영체제에서 일관된 환경 제공
- 성능 최적화: GPU 가속으로 빠른 렌더링 성능 달성
- 네이티브 UI: 각 플랫폼의 기본 UI 요소를 사용하여 자연스러운 사용자 경험 제공

> [!tip] 왜 중요한가
> 개발자들이 고성능의 현대적 터미널 환경에서 작업할 수 있으며, GPU 활용은 대량의 텍스트 출력 시나리오에서 두드러진 성능 향상을 제공합니다.
