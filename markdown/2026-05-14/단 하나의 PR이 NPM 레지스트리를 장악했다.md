## 핵심 개념

> [!abstract]
> NPM 레지스트리에 대한 보안 위협으로 단일 PR이 전체 시스템에 영향을 미칠 수 있음을 보여주는 사건이다. 패키지 관리자의 보안 취약점에 대한 경각심을 환기한다.

## 상세 내용

- NPM 레지스트리의 구조적 보안 문제 노출
- 오픈소스 패키지 관리 시스템의 잠재적 위험성 증명

> [!tip] 왜 중요한가
> 의존성 관리와 패키지 검증의 중요성을 강조하며 개발자들의 주의가 필요하다.
