## 핵심 개념

> [!abstract]
> CodeRabbit 도구를 사용하여 로컬 환경에서 PR 리뷰를 개선하고 코드 품질을 향상시키는 방법을 소개합니다.

## 상세 내용

- CodeRabbit을 이용한 자동화된 코드 리뷰 프로세스를 제시합니다.
- 로컬 개발 환경에서 즉시 피드백을 받을 수 있습니다.

> [!tip] 왜 중요한가
> 개발자는 PR 리뷰 과정에서 코드 품질을 사전에 검증하여 리뷰 시간을 단축할 수 있습니다.
