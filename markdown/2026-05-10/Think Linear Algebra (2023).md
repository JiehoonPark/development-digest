## 전문 번역

# Think Linear Algebra#

**Think Linear Algebra**는 선형대수의 핵심 개념을 실제로 이해하고 활용하고 싶은 사람들을 위한 실습 중심 입문서입니다. 추상적인 이론만 배우는 게 아니라 실제 문제를 풀면서 배우는 방식이죠.

각 장은 웹 트래픽 모델링, 새 떼 시뮬레이션, 전기회로 분석 같은 현실의 구체적인 문제에서 출발합니다. Python과 NumPy, SciPy, SymPy, NetworkX 같은 강력한 라이브러리를 활용해서 직접 작동하는 솔루션을 만들어보면서 선형대수가 얼마나 우아하고 범용적인 도구인지 자연스럽게 깨닫게 됩니다.

이 책은 기존의 수학 교육에 어려움을 겪었거나 더 직관적이고 손으로 만지며 배우고 싶은 사람들을 위해 만들어졌습니다. Jupyter 노트북에서 코드를 작성하고 시뮬레이션을 실행하고 결과를 시각화하면서 즉각적인 피드백을 얻을 수 있거든요. 수학적 형식주의부터 시작하는 대신, 의미 있는 응용 사례부터 시작해서 필요할 때마다 이론을 쌓아올리는 구조입니다. 그 결과는 선형대수를 실제 문제를 푸는 언어로서 배우는 참신하고 실용적인 경험이 됩니다.

선형대수는 머신러닝, 과학 계산, 컴퓨터 그래픽스의 기초인데, 이 모든 분야는 현재 빠르게 성장하고 있습니다. 검색 엔진에서 GPS 추적, 신호 처리, 구조 공학까지 우리 세상을 만드는 많은 기술들이 선형대수라는 언어로 작동하고 있습니다. 이 책은 당신이 자신의 업무에서 이런 도구들을 어떻게 효과적으로 사용할 수 있을지 보여줍니다.

이 책은 Creative Commons 라이선스로 공개되어 있어서 출처를 명시하고 상업적으로 이용하지 않는 한 자유롭게 복사, 배포, 수정할 수 있습니다.

## 이 책에서 배우는 것#

이 책을 다 읽고 나면 다음을 할 수 있게 됩니다:

- 실제 문제를 벡터와 행렬로 표현하고 표준 선형대수 알고리즘으로 풀기
- Python을 활용해서 시스템을 시뮬레이션하고, 정사영을 계산하고, 방정식을 풀고, 행렬 분해하기
- 벡터 공간, 변환, 시스템 거동 같은 수학 개념을 인터랙티브한 코드 예제로 시각화하기
- 공학, 데이터 과학, 그래픽스, 로봇공학 등 다양한 계산 분야에 선형대수 도구 적용하기
- 수학 문제를 계산 관점에서 생각하고 수식과 작동하는 코드를 오갈 수 있기

## 노트북#

현재 이용할 수 있는 장들입니다. 곧 더 추가될 예정입니다!

**1장: 선형대수의 힘**

Colab에서 1장 실행하기:

박물관 방문자 흐름을 네트워크 모델로 표현해서 행렬 곱셈과 고유벡터를 소개합니다. 웹 페이지 품질을 정량화하는 PageRank 알고리즘도 직접 구현해봅니다.

**2장: 경로 추적 하루**

Colab에서 2장 실행하기:

GPS 추적 데이터를 활용해서 벡터의 덧셈과 뺄셈을 배웁니다. 수치 미분으로 속도와 가속도를 추정하고, 노이즈가 있는 가속도 데이터에서 위치를 복원할 때의 어려움도 살펴봅니다.

**4장: 정사영**

Colab에서 4장 실행하기:

벡터 정사영, 벡터 배제, 직교성, 내적을 당구공의 탄성 충돌 사례로 설명합니다.

**5장: 우주로 나아가며**

Colab에서 5장 실행하기:

행렬을 사용해서 벡터를 확대, 회전, 전단, 평행이동시킵니다. 이를 2D 컴퓨터 그래픽스에 적용하고 고전 게임 Asteroids를 다시 만들어봅니다.

**7장: 방정식 체계**

Colab에서 7장 실행하기:

LU 분해와 행렬 방정식을 전기회로 분석에 적용합니다. 선형대수가 실제 공학 문제를 어떻게 해결하는지 보여줍니다.

**8장: 영공간**

Colab에서 8장 실행하기:

화학 양론을 여러 개의 올바른 해를 가진 시스템으로 봅니다. 랭크와 영공간 개념을 사용해서 해 공간을 설명합니다.

**9장: 구조계를 믿다**

Colab에서 9장 실행하기:

미지수가 벡터 힘인 구조 시스템을 모델링합니다. 블록 행렬과 랭크 분석을 사용해서 트러스 내부의 응력을 계산합니다.

**10장: 회귀분석**

Colab에서 10장 실행하기:

QR 분해와 직교성 방정식을 사용해서 최소제곱 회귀를 계산합니다. 일반사회조사(GSS) 데이터로 다중 회귀를 수행하고 정치 이념과 시간, 나이, 출생연도의 관계를 탐색합니다.
