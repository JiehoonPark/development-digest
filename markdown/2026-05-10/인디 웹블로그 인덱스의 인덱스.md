## 핵심 개념

> [!abstract]
> 개인 블로그와 소규모 웹 커뮤니티를 찾기 위한 38개의 인덱스 사이트를 분류하고 관리하는 메타 인덱스입니다. 큐레이션 디렉토리, RSS 애그리게이터, 검색 엔진, 랜덤 발견 도구, 제약 기반 클럽 등으로 구성됩니다.

## 상세 내용

- 6개 카테고리에 걸쳐 1,000개 이상의 개인 블로그와 웹 사이트를 인덱싱하는 도구들을 제공
- Marginalia Search, Kagi Small Web 등 비상업적 콘텐츠에 특화된 검색 엔진과 인디 블로그 추천 플랫폼 소개
- 페이지 무게 제한(1MB Club, 250KB Club) 등 기술적 제약을 기반으로 한 커뮤니티로 웹 최적화 문화 장려

> [!tip] 왜 중요한가
> 대형 소셜 미디어 플랫폼 중심의 인터넷 구조에서 벗어나 개인 중심의 웹 커뮤니티를 발견할 수 있습니다.

## 전문 번역

# 인디 웹 인덱스들의 인덱스

소규모 웹 커뮤니티를 위한 정보 허브들을 한곳에 모았습니다. 6개 카테고리에 걸쳐 38개의 인덱스 사이트를 소개하고 있으며, 특별히 명시되지 않은 이상 모두 사용자의 제출을 받고 있습니다.

## 큐레이션 디렉토리

손으로 직접 선별한 블로그와 개인 사이트 목록들입니다. 자신의 사이트를 등록하거나 주제별로 둘러보기 좋습니다.

**Blogosphere**  
1,000개 이상의 독립적인 개인 블로그를 모아놓은 곳

**Blogroll Club**  
모두의 블로그롤과 포스트롤이 모이는 공간

**Blogs Are Back**  
인디 블로그를 손으로 직접 엄선한 디렉토리

**Blogscroll**  
GitHub에서 관리하는 개방형 개인 사이트 디렉토리

**HN Personal Websites**  
해커 뉴스 사용자들의 개인 사이트 모음

**Indieseek**  
검색 가능한 인디 웹 디렉토리

**Mataroa Collection**  
Mataroa 플랫폼에서 호스팅되는 블로그 디렉토리

**Minifeed**  
소규모 개인 웹 피드 애그리게이터

**PersonalSit.es**  
소유자가 직접 제출한 개인 사이트 디렉토리

**The Big List of Personal Websites**  
개인 사이트들을 쭉 나열한 리스트

**Web Jamboree**  
웹을 탐색하는 데 도움이 되는 링크 모음

**Ye Olde Blogroll**  
1,000개 이상의 개인 및 독립 블로그를 손으로 큐레이션한 목록

**blogblog.es**  
스페인어 블로거들을 위한 블로그 디렉토리

**blogs.hn**  
작지만 실용적인 기술 블로그 디렉토리

**ooh.directory**  
2,300개 이상의 블로그를 주제별로 분류한 야후 스타일 디렉토리

## RSS & 피드 애그리게이터

실시간으로 인디 블로그의 포스트를 노출시켜주는 피드 기반 인덱스들입니다. 소규모 웹을 흐르는 물줄기처럼 따라갈 수 있어요.

**RSS.Social**  
RSS 기반 소셜 디스커버리

**Scour**  
자신의 관심사를 기반으로 피드에서 포스트를 찾아주는 서비스

**feedle**  
블로그와 팟캐스트의 RSS 피드 검색 엔진

**indieblog.page**  
4,900개 이상의 사이트가 등록된 랜덤 인디 블로그 버튼

**powRSS**  
셔플과 랜덤 기능이 있는 공개 RSS 애그리게이터

## 검색 엔진

실제로 소규모 웹을 크롤링하고 인덱싱하는 검색 엔진들입니다. 특정 콘텐츠나 작성자를 찾을 때 유용합니다.

**Kagi Small Web**  
개인 블로그, 독립 유튜브 채널, 웹코믹 등을 검색

**Marginalia Search**  
콘텐츠가 풍부하고 비상업적이며 가벼운 사이트들을 검색하는 엔진

**Search My Site**  
3,400개 이상의 독립 사이트 포스트 검색

**Wiby**  
초기 웹의 스타일을 연상시키는 구식 페이지 검색 엔진

**blogsearch.io**  
블로그 중심의 검색 엔진

## 랜덤 디스커버리

예상 밖의 곳으로 당신을 데려갑니다. 뭘 찾을지 모를 때, 그냥 새로운 걸 발견하고 싶을 때 좋아요.

**Blog of the Day**  
매일 다른 인디웹 블로그를 소개하는 서비스 (RSS 피드 제공)

**Read Something Interesting**  
웹 곳곳의 훌륭한 글들을 노출시켜줍니다

**The Forest**  
웹에서 길을 잃고 싶을 때 - 랜덤 사이트 디스커버리

## 조건 기반 클럽

페이지 용량, JavaScript 미사용, 다크테마 등 특정 기술적 혹은 창의적 조건으로 정의된 인덱스들입니다. 의도적으로 미니멀하게 만들어진 사이트들을 찾기 좋습니다.

**1MB Club**  
1메가바이트 미만의 사이트

**250KB Club**  
250킬로바이트 미만의 사이트

**512KB Club**  
512킬로바이트 미만의 사이트

**Dark Theme Club**  
기본값으로 다크테마를 사용하는 사이트

**no-js.club**  
JavaScript가 없는 사이트

## 인디웹 인프라

웹링, /now 페이지, 웹멘션 네트워크 등 인디웹 커뮤니티를 연결하는 연결고리들입니다. 더 큰 커뮤니티에 참여하고 싶을 때 활용하세요.

**IndieWeb Directory**  
IndieWeb 위키의 개인 사이트 인덱스

**IndieWeb Webring**  
IndieWeb 커뮤니티 멤버들의 웹링

**Now Now Now**  
Derek Sivers가 운영하는 /now 페이지 디렉토리

**omg.lol directory**  
omg.lol을 통한 개인/인디웹 사이트 디렉토리

**uses.tech**  
/uses 페이지 디렉토리
