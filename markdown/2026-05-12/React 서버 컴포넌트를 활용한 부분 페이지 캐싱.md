## 핵심 개념

> [!abstract]
> React 서버 컴포넌트(React Server Components, RSC)를 활용하여 페이지의 특정 부분만 선택적으로 캐싱하는 기법을 다룬 동영상입니다.

## 상세 내용

- React 서버 컴포넌트의 캐싱 메커니즘과 부분 페이지 캐싱 구현 방법
- 페이지 전체 캐싱이 아닌 세밀한 수준의 캐싱 전략으로 성능 최적화 달성

> [!tip] 왜 중요한가
> Next.js와 React 최신 기능을 활용한 성능 최적화 기법으로, 대규모 트래픽을 처리하는 프로덕션 환경에서 렌더링 효율을 크게 향상시킬 수 있습니다.
