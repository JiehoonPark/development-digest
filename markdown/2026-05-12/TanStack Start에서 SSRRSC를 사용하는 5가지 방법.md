## 핵심 개념

> [!abstract]
> TanStack Start 프레임워크에서 SSR(Server-Side Rendering)과 RSC(React Server Components)를 구현하는 5가지 주요 방법과 패턴을 다룬 영상입니다. 각 방법의 구체적인 사용 사례와 구현 방식을 설명하여 개발자들이 프로젝트에 맞는 최적의 렌더링 전략을 선택할 수 있도록 합니다.

## 상세 내용

- SSR 기본 구현: TanStack Start에서 기본적인 Server-Side Rendering을 설정하는 방법과 초기 페이지 로드 성능 개선
- RSC 활용: React Server Components를 사용하여 서버에서 직접 데이터를 페칭하고 클라이언트 번들 크기를 줄이는 방식
- 하이브리드 렌더링: SSR과 RSC를 함께 사용하여 성능과 개발자 경험의 균형을 맞추는 전략
- 렌더링 전략 선택: 프로젝트 요구사항에 따라 5가지 방법 중 적절한 것을 선택하기 위한 판단 기준

> [!tip] 왜 중요한가
> TanStack Start는 현대적인 풀스택 프레임워크로서 SSR과 RSC 지원이 핵심 기능이며, 이 영상은 각 렌더링 방식의 장단점을 명확히 이해하여 성능 최적화된 웹 애플리케이션을 구축할 수 있게 합니다.
