## 핵심 개념

> [!abstract]
> Voker는 AI 에이전트의 상호작용을 구조화된 분석으로 변환하여 팀 전체가 에이전트 성능을 추적하고 최적화할 수 있는 분석 플랫폼입니다. Intent 분류, 문제 감지, 해결율 측정 등을 자동으로 수행합니다.

## 상세 내용

- 사용자의 의도를 자동으로 분류하고, 사용자가 원하는 답변을 받지 못할 때를 감지하며, 에이전트의 해결 성공률을 측정합니다.
- Python & TypeScript SDK로 간단하게 통합 가능하며 OpenAI, Anthropic, Gemini 등 주요 LLM을 지원합니다.
- PM, 분석가, 비즈니스팀이 에이전트 메트릭과 사용자 데이터를 연관시켜 ROI를 정량화할 수 있습니다.

> [!tip] 왜 중요한가
> 에이전트 기반 애플리케이션의 성능을 비즈니스 결과와 연결하여 측정할 수 있어 제품팀의 의사결정을 데이터 기반으로 만들 수 있습니다.

## 전문 번역

# AI 에이전트의 진정한 성과를 측정하고 있나요?

당신의 AI 에이전트가 사용자와 나누는 대화가 정말 도움이 되는지 알고 계신가요? 아마 잘 모르실 거예요. 실시간 추적 데이터만으로는 에이전트가 실제로 도움이 되는지, 정확한지, 아니면 막혀있는지 판단하기 어렵거든요. 문제를 발견하는 순간은 고객이 불평할 때거나 이탈이 급증했을 때뿐입니다.

## 현재의 어려움들

**데이터 수집이 자원을 많이 먹습니다**

모든 팀이 데이터를 필요로 합니다. 그런데 정보 요청이 들어올 때마다 엔지니어들이 개발 업무에서 벗어나야 하죠. 결국 누구도 스스로 인사이트를 얻을 수 없는 상황이 벌어집니다.

**AI 투자의 성과를 증명하기 어렵습니다**

프로덕트팀과 비즈니스팀은 AI 투자의 효과를 수치화해야 합니다. 사용 통계는 있지만, 에이전트 성능이 실제 전환율, 고객 유지, 매출로 이어지는지 파악하기는 어렵거든요.

## Voker가 제공하는 솔루션

Voker는 AI 에이전트와 사용자의 상호작용을 구조화된 데이터로 변환해줍니다. 이를 통해 팀 전체가 필요한 인사이트를 얻을 수 있습니다.

### 셀프서비스 애널리틱스

PM, 데이터 분석가, 비즈니스팀이 별도의 요청 없이 필요한 정보를 스스로 찾을 수 있습니다. 엔지니어링 팀의 병목 현상도 사라집니다.

### 에이전트 성능 추적

에이전트가 사용자에게 정확히 무엇을 말하고 있는지 파악하세요. 지식 격차를 찾아내고, 이상 신호를 감지하며, 시간에 따른 개선을 측정할 수 있습니다.

### 비즈니스 성과 연계

에이전트 메트릭을 실제 중요한 결과와 연결하세요. 대화 데이터와 이미 수집 중인 사용자 데이터를 함께 분석하면 됩니다.

## Voker가 추적하는 세 가지 핵심 요소

**의도(Intent)**

사용자가 원하는 것이 무엇인지 파악합니다. Voker는 자연스러운 대화에서 사용자의 목표를 자동으로 분류합니다.

예: "다음 휴가 예약 도와줄래?"

**수정 요청(Corrections)**

사용자가 원하는 결과를 얻지 못했을 때를 감지합니다. 고객 이탈로 이어지기 전에 문제를 파악할 수 있습니다.

예: "아니야, 날짜를 잘못 봤어. 4월 5일부터 4월 18일까지인데..."

**해결(Resolutions)**

에이전트가 사용자의 요청을 실제로 해결했는지 인식합니다. 모든 상호작용에서 성공률을 측정할 수 있습니다.

예: "좋아, 4월 5일부터 4월 18일까지 항공권을 예약했어."

## 어떤 팀에게 필요한가?

- 매달 1,000건 이상의 대화 세션을 처리하는 팀
- 다중 턴 복잡한 대화를 다루는 팀 (도구, RAG, MCP 포함)
- 에이전트 인사이트가 필요한 크로스펑셔널 팀

## 기술적 특징

**스택에 맞춰 통합됩니다**

OpenAI, Anthropic, Gemini는 물론 Langchain, CrewAI, Vercel AI SDK 등 모든 LLM 프레임워크를 지원합니다. 특정 제공사에 종속되지 않습니다.

**가볍고 빠릅니다**

Python과 TypeScript 모두 지원하며, 설치는 두 줄이면 끝입니다.

```
pip install voker
```

**데이터는 당신의 것입니다**

데이터 소유권을 완전히 유지할 수 있습니다. 자체 호스팅도 가능하며, 엔터프라이즈 요구사항을 고려해 설계되었습니다.

## 시작하기

단 몇 줄의 코드만으로 에이전트 분석을 시작할 수 있습니다. 인프라 변경이 불필요하며, 무료로 시작해서 성장에 맞춰 확장하면 됩니다.

### 1단계: SDK 설치

어떤 프레임워크든 한 코드 스니펫으로 설치합니다.

### 2단계: 에이전트 연결

자동으로 데이터 수집이 시작됩니다.

```python
from voker.ai.provider_openai import OpenAI
```

### 3단계: 분석 시작

대시보드에 인사이트가 자동으로 채워집니다.

## Voker의 주요 기능들

**대화 타임라인 조회**

사용자의 관점에서 에이전트 대화를 볼 수 있습니다. 주제, 의도, 문제점을 모든 대화에서 검색할 수 있습니다.

**성능 추적**

변화를 가했을 때 실제 개선을 수치화할 수 있습니다. 업데이트가 시스템을 망쳤다면 언제 롤백해야 할지 알 수 있습니다.

**사용자 행동 분석**

사용자들이 무엇을 묻고 있는지, 에이전트가 실제로 대답하고 있는지 추적합니다. 사용자가 답답해하거나 포기할 때 알림도 받을 수 있습니다.

**생태계 친화적**

Langfuse, Langsmith, PostHog, Mixpanel, Amplitude 등과 함께 사용할 수 있습니다.
