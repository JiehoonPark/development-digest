## 핵심 개념

> [!abstract]
> Claude Code, Cursor, Codex 같은 AI 코드 생성 도구에 대응하기 위한 디자인 기술 모음입니다. AI가 생성한 저품질 콘텐츠(AI-slop)를 피하는 방법을 CSS 기반으로 제시합니다.

## 상세 내용

- AI 코드 생성 도구의 보편화에 따른 디자인 품질 관리 필요성
- 실무 개발자를 위한 실질적인 디자인 기술 습득 자료

> [!tip] 왜 중요한가
> AI 생성 코드의 품질 문제가 커지면서 개발자가 전문적인 디자인 감각을 유지해야 할 필요성이 증가하고 있습니다.
