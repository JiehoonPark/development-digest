## 핵심 개념

> [!abstract]
> React Doctor는 AI 에이전트가 작성한 React 코드의 안티패턴과 문제를 자동으로 감지하고 지적하는 TypeScript 기반 도구입니다. React의 일반적인 나쁜 코드 패턴을 식별하여 코드 품질을 향상시키고, 특히 AI 코드 생성 모델이 만드는 실수를 포착하는 데 집중합니다.

## 상세 내용

- AI 에이전트가 생성한 React 코드의 품질 문제를 자동 감지하는 검사 도구로, 개발자가 수동으로 검토해야 할 부담을 줄입니다.
- React의 흔한 안티패턴(잘못된 훅 사용, 불필요한 리렌더링, 잘못된 의존성 배열 등)을 식별하여 런타임 버그를 사전에 방지합니다.
- TypeScript로 구현되어 타입 안정성을 제공하며, 코드 분석 파이프라인에 통합하여 CI/CD 워크플로우에 적용할 수 있습니다.

> [!tip] 왜 중요한가
> AI 코드 생성 도구의 활용이 증가하면서 생성된 코드의 품질 검증이 중요해지고 있으며, 이 도구는 React 생태계에서 자동화된 품질 관리를 가능하게 합니다.
