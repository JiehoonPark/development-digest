## 핵심 개념

> [!abstract]
> Anthropic이 출시한 Claude Code는 터미널에서 작동하는 에이전트형 코딩 도구로, 사용자의 코드베이스를 이해하고 자연어 명령을 통해 반복적 작업 자동화, 복잡한 코드 설명, Git 워크플로우 관리 등을 수행한다. 사용자가 터미널에서 직접 상호작용하며 일상적인 개발 업무를 효율화할 수 있도록 설계되었다.

## 상세 내용

- 터미널 기반 인터페이스로 IDE 전환 없이 자연어 명령으로 코딩 작업 수행 가능하며, 개발자의 기존 워크플로우에 자연스럽게 통합된다.
- 코드베이스 이해 능력을 갖춰 프로젝트 전체 구조를 파악한 후 관련 파일을 자동으로 찾아 작업을 수행한다.
- Git 워크플로우 자동화 포함으로 커밋, 브랜치 관리, PR 생성 등을 자연어로 요청할 수 있다.
- 복잡한 코드 설명 기능으로 레거시 코드나 어려운 로직을 이해하는 데 도움을 제공한다.
- 에이전트형 설계로 여러 단계의 작업을 자동으로 체인화하여 생산성을 향상시킨다.

> [!tip] 왜 중요한가
> 개발자들의 반복적인 작업 부담을 줄이고 터미널 환경에서 AI의 지원을 받을 수 있어, IDE 기반 도구들과 다른 차별화된 개발 경험을 제공한다.
