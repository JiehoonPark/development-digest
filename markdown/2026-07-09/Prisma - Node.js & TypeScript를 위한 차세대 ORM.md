## 핵심 개념

> [!abstract]
> Prisma는 Node.js와 TypeScript 환경에서 사용할 수 있는 현대적 ORM으로, PostgreSQL, MySQL, MariaDB, SQL Server, SQLite, MongoDB, CockroachDB 등 다양한 데이터베이스를 지원합니다.

## 상세 내용

- 다양한 데이터베이스 지원 - PostgreSQL, MySQL, MariaDB, SQL Server, SQLite, MongoDB, CockroachDB 등 주요 데이터베이스 플랫폼과의 호환성 제공
- TypeScript 중심 설계 - Node.js와 TypeScript 개발 환경에 최적화되어 타입 안정성 강화

> [!tip] 왜 중요한가
> TypeScript 프로젝트에서 데이터베이스 작업을 안전하고 효율적으로 수행할 수 있게 해줍니다.
