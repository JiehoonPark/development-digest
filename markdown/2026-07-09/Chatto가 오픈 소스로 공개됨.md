## 핵심 개념

> [!abstract]
> 팀 채팅 애플리케이션 Chatto가 오픈 소스로 공개되었으며, 자체 호스팅과 엔드-투-엔드 암호화를 지원합니다. 버전 0.4에서 시작하며 1.0.0 출시는 약 6-12개월 예상되고, Chatto Cloud를 통한 유료 호스팅도 곧 베타에 진입할 예정입니다.

## 상세 내용

- Homebrew로 간단히 설치 가능한 경량 자체 호스팅 솔루션
- 사용자별 키로 암호화된 모든 데이터와 엔드-투-엔드 암호화된 음성/영상 통화 지원
- Chatto Cloud는 유럽 인프라 기반 자동 스케일링과 무중단 업그레이드 제공

> [!tip] 왜 중요한가
> 개발자와 팀이 데이터 프라이버시를 유지하면서 자체 인프라에 채팅 시스템을 배포할 수 있습니다.

## 전문 번역

# Chatto가 오픈소스로 공개되었습니다

혹시 모르니 미리 말씀드리자면, 지난 1년간 개발해온 그룹 채팅 애플리케이션 Chatto가 드디어 오픈소스로 공개되었습니다. 누구나 자신의 서버에 설치해서 사용할 수 있게 된 거죠.

가장 빠르게 시작하는 방법은 Homebrew를 이용하는 것입니다:

```
brew install chattocorp/tap/chatto
chatto init
chatto run
```

자세한 내용은 [Chatto 시작 가이드](link)에서 확인할 수 있습니다.

## 채팅, 이제 진짜입니다

Chatto는 실제로 사용하고 싶은 그룹 채팅 앱이 되려고 합니다. 아마 여러분도 어떤 대중적인 채팅 서비스들을 알고 계실 거예요. Chatto는 기본적으로 그런 앱들과 비슷한데, 훨씬 더 가볍고 빠릅니다. 게다가 오픈소스고 직접 호스팅할 수 있죠. 비용도 전혀 들지 않습니다. (솔직히 좀 이상한 말이지만, 요즘 오픈소스 채팅 앱 생태계가 정말 독특해졌거든요!)

Chatto HQ 커뮤니티에 들어가면 실제로 어떻게 작동하는지 볼 수 있습니다.

## 자체 호스팅이 정말 간단합니다

실행 파일 하나만 띄우면 됩니다. 프론트엔드도 자동으로 제공되죠. 리소스도 거의 먹지 않고, 이 종류의 앱 중에서 가장 빠른 프론트엔드를 자랑합니다.

데이터 보호와 개인정보 보호를 최우선으로 생각했습니다. 모든 개인정보와 채팅 데이터는 사용자별 키로 암호화되어 있고, 사용자가 계정을 삭제하면 키도 함께 제거됩니다.

각 Chatto 서버는 하나의 커뮤니티를 운영하며, 서버 간 데이터 연동이나 서드파티 추적은 없습니다. 여러 서버에 동시에 접속하고 싶으면 클라이언트가 모두에 직접 연결하면 됩니다. 여러 커뮤니티를 호스팅하려면 그냥 Chatto 프로세스를 여러 개 실행하면 되고요. 정말 간단합니다!

음성 및 화상 통화 기능도 완벽하게 지원합니다. 화면 공유도 포함되어 있죠. 모든 통화는 엔드투엔드 암호화되며, 인프라가 감당할 수 있는 만큼의 참가자를 지원합니다.

Linux(x86_64, ARM64), macOS, Windows용 바이너리가 준비되어 있습니다. [Chatto 자체 호스팅 문서](link)에서 시작하세요.

## Chatto Cloud

스스로 서버를 관리하기 싫다면 Chatto Cloud도 있습니다. 곧 공개 베타가 시작될 예정입니다.

Chatto Cloud의 구성은 매우 단순합니다. Chatto 서버의 호스팅만 제공합니다. 프리미엄 구독도 없고, 광고도 없고, 이상한 것도 없습니다. 그냥 좋은 호스팅만 제공하죠.

유럽 기반의 완전히 유럽식 인프라로 시작하며, 2027년 초에 더 많은 지역이 추가될 예정입니다. Chatto Cloud에 호스팅된 서버는 자동 스케일링, 매일 밤 자동 백업, 무중단 버전 업그레이드를 모두 지원합니다.

락인도 없습니다. Chatto Cloud에서 호스팅되는 서버는 자체 호스팅되는 서버와 100% 호환되므로, 언제든지 데이터를 옮겨 나올 수 있습니다.

베타 시작 알림을 받고 싶으면 이 글 끝의 저용량 뉴스레터를 구독하세요.

## Chatto의 다음 계획

Chatto는 현재 0.4 버전입니다. 프로덕션 환경에서 충분히 안정적이지만, 아직 몇 가지 중요한 기능이 빠져 있습니다. [Chatto 로드맵](link)에서 자세히 확인할 수 있습니다.

0.5 버전의 초점은 추가 안전 기능(콘텐츠 신고 및 모더레이션)과 클라이언트 개선(특히 다중 서버 기능)에 맞춰질 것입니다. 정말 흥미로운 기능들을 준비 중이고, 여러분께 보여드릴 날이 기대됩니다.

1.0.0은 약 6~12개월 후에 나올 예정입니다. 그때까지는 파괴적인 변경이 발생할 수 있으니, 자체 호스팅한다면 새 버전이 나올 때마다 업데이트할 준비를 해두세요.
