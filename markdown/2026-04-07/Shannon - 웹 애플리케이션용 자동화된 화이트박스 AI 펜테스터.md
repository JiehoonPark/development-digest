## 핵심 개념

> [!abstract]
> Shannon Lite는 웹 애플리케이션과 API의 소스 코드를 분석하여 공격 벡터를 식별하고 실제 익스플로잇을 실행해 프로덕션 배포 전 취약점을 입증하는 자동화된 보안 테스트 도구다.

## 상세 내용

- 소스 코드 분석 기반의 화이트박스 펜테스팅으로 자동화된 보안 검증
- 실제 익스플로잇 실행으로 취약점의 실질적 영향도 검증 가능

> [!tip] 왜 중요한가
> 개발자가 프로덕션 배포 전 보안 취약점을 사전에 발견하고 수정할 수 있다.
