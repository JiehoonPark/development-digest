## 핵심 개념

> [!abstract]
> Chrome 145에서 column-height와 column-wrap 속성이 도입되어 콘텐츠를 새로운 행으로 래핑하여 가로 스크롤 대신 세로 스크롤을 만들 수 있게 되었다.

## 상세 내용

- column-height와 column-wrap 속성으로 다중 열 레이아웃에서 콘텐츠 흐름 제어 가능
- 수평 스크롤에서 수직 스크롤로 변경하여 사용자 경험 개선

> [!tip] 왜 중요한가
> CSS 레이아웃 옵션이 확대되어 복잡한 반응형 디자인 구현이 더 간단해진다.
