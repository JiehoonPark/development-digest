## 핵심 개념

> [!abstract]
> GitHub 상태 페이지에서 보고된 API 요청 관련 인증 문제입니다. 사용자들이 이메일 또는 문자메시지를 통해 사건 업데이트를 구독할 수 있습니다.

## 상세 내용

- API 요청 인증 관련 서비스 장애 발생
- GitHub 상태 페이지를 통해 실시간 인시던트 모니터링 및 구독 가능

> [!tip] 왜 중요한가
> GitHub API를 사용 중인 개발자들이 인증 문제로 인한 서비스 중단에 주의해야 합니다.

## 전문 번역

죄송합니다만, 제공하신 텍스트가 기술 아티클이 아니라 사용자 인터페이스의 UI 요소들(OTP 재전송, 웹훅 설정, 이메일 구독 등)로 보입니다.

기술 블로그 아티클로 재구성할 수 있는 충분한 내용이 없어서 작업이 어렵습니다.

혹시 다음 중 하나인지 확인해주시겠어요?

1. **전체 기술 아티클**: 더 길고 내용 있는 원문을 제공해주시면 좋겠습니다.
2. **UI 번역**: 이 텍스트를 단순히 한국어로 번역만 해주면 되는 건지
3. **OTP/웹훅 관련 기술 글**: 이 기능들을 설명하는 기술 가이드 작성을 원하시는 건지

원문을 다시 확인하신 후 알려주시면 감사하겠습니다!
