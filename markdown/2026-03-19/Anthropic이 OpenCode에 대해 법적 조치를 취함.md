## 핵심 개념

> [!abstract]
> Anthropic의 법적 요청으로 인해 OpenCode 저장소에서 번들된 플러그인 제거가 이루어졌으나, 여러 사용자가 이를 되돌리는 커밋을 진행하여 프로젝트 내에 논란이 발생했다.

## 상세 내용

- Anthropic의 법적 요청으로 인한 변경사항이 병합되었으나 일부 기여자들이 해당 변경을 즉시 되돌림
- Claude 코드 OAuth 기능에 영향을 미쳤으며 기술 커뮤니티 내에서 법적 분쟁이 지속 중

> [!tip] 왜 중요한가
> 오픈소스 프로젝트에서 발생하는 법적 분쟁이 개발 진행과 기능에 직접적인 영향을 미칠 수 있음을 보여준다.

## 전문 번역

죄송하지만, 제공해주신 내용이 GitHub Pull Request의 활동 로그일 뿐 실제 기술 아티클이 아니어서 재구성할 만한 본문이 없습니다.

번들된 플러그인 제거에 대한 문서 명확화(docs: clarify bundled plugin removal)라는 PR 제목과 여러 커밋 이력, 리버트 기록들만 있을 뿐, 설명할 기술 내용이 포함되어 있지 않습니다.

재구성할 기술 아티클을 제공해주시면 기꺼이 자연스러운 한국어 기술 글로 작성해드리겠습니다.
