## 핵심 개념

> [!abstract]
> AI 분석을 위한 PDF 파서로, PDF 접근성을 자동화하고 오픈소스로 제공하는 Java 기반 프로젝트입니다.

## 상세 내용

- PDF 파일에서 AI 모델 학습에 용이한 형태로 데이터를 추출하는 자동화 도구
- 오픈소스 프로젝트로 개발자가 자유롭게 활용 및 개선 가능

> [!tip] 왜 중요한가
> 대량의 PDF 문서를 AI 모델에 활용해야 하는 개발자에게 데이터 전처리 과정을 단순화할 수 있는 도구입니다.
