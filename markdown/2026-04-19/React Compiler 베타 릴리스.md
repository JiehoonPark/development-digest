## 핵심 개념

> [!abstract]
> React Conf 2024에서 React Compiler의 실험적 릴리스가 발표되었습니다. 이는 자동 메모이제이션을 통해 React 앱을 최적화하는 빌드 타임 도구입니다.

## 상세 내용

- 자동 메모이제이션을 통한 성능 최적화 - 개발자가 수동으로 useMemo, useCallback 등을 작성할 필요 없이 빌드 시점에 자동으로 최적화합니다.

> [!tip] 왜 중요한가
> React 앱의 성능 최적화를 개발자 개입 없이 자동화하여 더 빠른 애플리케이션을 만들 수 있습니다.
