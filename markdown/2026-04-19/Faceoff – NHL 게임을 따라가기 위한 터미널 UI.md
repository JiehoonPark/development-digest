## 전문 번역

# Faceoff¶

NHL 하키 경기를 실시간으로 따라볼 수 있는 터미널 사용자 인터페이스(TUI) 애플리케이션입니다.

## 주요 기능¶

- **라이브 게임 일정**: NHL 경기를 날짜별로 확인하고 전날/다음날로 쉽게 네비게이션
- **실시간 점수 업데이트**: 진행 중인 경기의 스코어를 자동으로 새로고침
- **경기 상세 정보**: 플레이-바이-플레이, 박스 스코어, 득점 요약 등을 진행 중이거나 종료된 경기에서 확인
- **경기 전 미리보기**: 골키퍼 비교, 스케이터 리더 등 경기 매칭업 데이터 제공
- **리그 순위표**: 현재 NHL 순위를 여러 방식(와일드카드, 디비전, 컨퍼런스, 전체 리그)으로 조회
- **선수 스탯 리더**: 스케이터와 골키퍼의 다양한 통계 카테고리별 상위 선수 확인
- **팀 브라우저**: 모든 NHL 팀을 열람하고 로스터, 팀 일정 확인
- **선수 프로필**: 선수의 상세 정보, 경력 통계, 경기 기록 조회
- **반응형 레이아웃**: 터미널 폭에 따라 경기 카드와 순위표가 자동으로 배열
- **로컬 시간대 표시**: 경기 시간이 당신의 시간대로 표시되며 시간대 약자도 함께 제공

## 빠른 시작¶

### 설치¶

Faceoff를 실행하는 가장 간단한 방법은 uvx를 사용하는 것입니다:

```
uvx faceoff
```

또는 pip으로 설치할 수도 있습니다:

```
pip install faceoff
faceoff
```

## 감사의 말¶

이 프로젝트는 MLB 야구 경기를 팔로우하는 터미널 애플리케이션인 Playball에서 영감을 받았습니다.

## 사용된 기술¶

- **Textual** - Python을 위한 최신 TUI 프레임워크로, 사용자 인터페이스를 구동합니다
- **nhl-stats-api-client** - NHL API 데이터에 접근하기 위한 Python 클라이언트

## 고지사항¶

이 프로젝트는 내셔널 하키 리그(NHL), 그 산하 팀, 관계 기관과 무관하며 이들로부터 공식 승인을 받지 않았습니다. 모든 NHL 로고, 상표, 데이터는 NHL과 그 산하 팀의 소유입니다.

이 애플리케이션은 공개적으로 이용 가능한 NHL API 데이터를 정보 제공 및 교육 목적으로만 사용합니다.
