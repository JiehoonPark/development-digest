## 핵심 개념

> [!abstract]
> React Compiler의 첫 번째 안정 릴리스(v1.0)가 2025년 10월 7일에 출시되었습니다. 이는 자동 메모이제이션을 통한 성능 최적화 기능을 정식으로 제공하는 프로덕션 준비 완료 버전입니다.

## 상세 내용

- 프로덕션 안정 버전 도출 - 베타 테스트를 거쳐 안정성이 검증된 v1.0이 정식 릴리스되었습니다.

> [!tip] 왜 중요한가
> React Compiler를 이제 프로덕션 애플리케이션에서 안심하고 사용할 수 있게 되었습니다.
