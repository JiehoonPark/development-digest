## 핵심 개념

> [!abstract]
> Hermes Agent는 NousResearch에서 개발한 Python 기반의 AI 에이전트 프레임워크입니다. 사용자의 필요에 따라 확장 가능하고 성장할 수 있도록 설계된 프로젝트이며, 에이전트 기반 작업 자동화와 AI 통합을 목표로 합니다.

## 상세 내용

- 사용자 요구사항에 맞게 확장 가능한 에이전트 아키텍처를 제공하여, 간단한 작업부터 복잡한 자동화까지 대응할 수 있도록 설계됨
- Python 기반이므로 데이터 과학, 머신러닝 커뮤니티에서 쉽게 활용 가능하며, 기존 Python 생태계와의 통합이 용이함
- 에이전트가 시간이 지남에 따라 학습하고 개선되는 '성장'을 지향하는 설계 철학을 가지고 있음

> [!tip] 왜 중요한가
> AI 에이전트 개발이 주목받는 가운데, 확장 가능하고 유지보수하기 쉬운 오픈소스 프레임워크는 개발자들이 프로덕션 레벨의 에이전트를 빠르게 구축할 수 있게 해줍니다.
