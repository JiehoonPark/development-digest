## 핵심 개념

> [!abstract]
> Claude Code 플러그인으로, 코딩 세션 중 Claude의 모든 활동을 자동으로 캡처하고 AI로 압축한 후 향후 세션에 관련 컨텍스트를 주입하는 메모리 시스템이다.

## 상세 내용

- 코딩 세션의 지속성을 제공하여 이전 작업 내용을 자동으로 기억하고 활용한다.
- Claude의 에이전트 SDK를 활용한 AI 기반 압축으로 메모리 효율성을 달성한다.
- 개발자가 문맥을 반복 설명할 필요 없이 이어서 작업할 수 있는 환경을 제공한다.

> [!tip] 왜 중요한가
> 장기 코딩 프로젝트에서 AI 어시스턴트가 프로젝트 맥락을 유지하도록 하여 상호작용 효율성을 크게 높일 수 있다.
