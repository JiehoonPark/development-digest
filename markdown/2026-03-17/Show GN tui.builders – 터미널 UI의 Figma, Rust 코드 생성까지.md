## 핵심 개념

> [!abstract]
> Rust CLI/TUI 도구 개발자들을 위한 새로운 도구로, Python의 Textual이나 JavaScript의 Ink 같은 도구는 편리하지만 런타임 의존성과 성능 오버헤드가 있는 반면, ratatui 생태계에서 Figma처럼 UI를 디자인하고 Rust 코드를 자동 생성할 수 있는 솔루션을 제공한다.

## 상세 내용

- Rust의 성능과 단일 바이너리 배포 장점을 활용하면서도 TUI 개발 경험을 개선
- ratatui 기반 생태계에서 시각적 설계 도구를 통해 코드 자동 생성 가능

> [!tip] 왜 중요한가
> Rust 기반 CLI/TUI 개발 시 개발 생산성을 크게 향상시킬 수 있는 도구이며, 런타임 의존성 없이 고성능을 유지하려는 개발자들에게 유용하다.
