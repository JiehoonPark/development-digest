## 핵심 개념

> [!abstract]
> Codex, Claude, Cursor, Gemini 등 다양한 AI 코딩 도구의 API 사용량 한도와 리셋 시간을 macOS 메뉴 바에서 한눈에 확인할 수 있는 오픈소스 앱입니다. 16개 이상의 프로바이더를 지원하며 세션·주간 사용량 미터와 리셋 카운트다운을 표시합니다.

## 상세 내용

- 16개 이상의 AI 코딩 도구 프로바이더 지원
- 메뉴 바에서 즉시 사용량과 리셋 시간 확인 가능

> [!tip] 왜 중요한가
> 여러 AI 코딩 도구를 사용하는 개발자들이 API 할당량을 효율적으로 관리할 수 있습니다.
