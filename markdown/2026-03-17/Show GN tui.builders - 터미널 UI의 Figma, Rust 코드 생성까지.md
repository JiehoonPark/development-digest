## 핵심 개념

> [!abstract]
> Rust로 CLI/TUI 도구를 개발할 때 Figma와 같은 비주얼 에디터로 UI를 디자인하고 자동으로 Rust 코드를 생성할 수 있는 tui.builders 프로젝트. Python의 Textual이나 JS의 Ink와 달리 런타임 의존성 없이 Rust의 성능 이점을 활용한다.

## 상세 내용

- 비주얼 UI 디자인 도구에서 즉시 Rust 코드 생성 가능
- ratatui 기반으로 Rust의 성능과 단일 바이너리 배포 이점 활용

> [!tip] 왜 중요한가
> Rust 개발자들이 TUI 개발을 더 빠르고 간편하게 할 수 있어 CLI 도구 개발 생산성이 증가한다.
