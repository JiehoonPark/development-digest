## 핵심 개념

> [!abstract]
> Puppeteer는 Chrome과 Firefox를 제어하기 위한 고수준의 JavaScript API를 제공하는 Node.js 라이브러리입니다. TypeScript로 작성되었으며, 자동화된 브라우저 테스트, 성능 측정, 웹 스크래핑 등 다양한 용도로 사용됩니다.

## 상세 내용

- Chrome 및 Firefox 브라우저 자동화: Puppeteer는 DevTools 프로토콜을 통해 Chrome과 Firefox를 프로그래매틱하게 제어할 수 있으며, 클릭, 입력, 스크롤 등의 사용자 인터랙션을 시뮬레이션할 수 있습니다.
- TypeScript 기반 구현: Puppeteer는 TypeScript로 작성되어 있어 타입 안정성을 제공하고, 자동완성과 런타임 오류 감소의 이점을 제공합니다.
- 다양한 사용 사례: 자동화된 테스트, 성능 프로파일링, 스크린샷/PDF 생성, 웹 스크래핑, Single Page Application(SPA) 테스트 등 여러 용도에 활용될 수 있습니다.

> [!tip] 왜 중요한가
> Puppeteer는 헤드리스 브라우저 자동화의 업계 표준 도구로, E2E 테스트와 자동화 작업을 수행하는 많은 개발팀에서 필수적인 의존성입니다.
