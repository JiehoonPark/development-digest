## 핵심 개념

> [!abstract]
> GitNexus는 브라우저에서 완전히 작동하는 클라이언트 사이드 지식 그래프 생성 도구로, GitHub 저장소나 ZIP 파일을 업로드하면 Graph RAG 에이전트를 통해 인터랙티브한 지식 그래프를 생성합니다.

## 상세 내용

- 클라이언트 사이드 실행으로 인프라 설정 불필요
- Graph RAG 에이전트로 고급 코드 탐색 기능 제공

> [!tip] 왜 중요한가
> 개발자가 서버 없이 로컬에서 코드베이스의 구조와 관계를 시각화하고 이해할 수 있습니다.
