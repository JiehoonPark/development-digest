## 핵심 개념

> [!abstract]
> Inkscape 1.4.4는 2026년 5월 6일 출시된 유지보수 및 버그픽스 릴리스로, 20개의 크래시 수정, 약 20개의 버그 수정, 6개의 성능 개선을 포함합니다.

## 상세 내용

- Inkscape가 시작되지 않는 심각한 3가지 크래시 수정 포함된 총 20개 크래시 버그 해결
- 성능 개선으로 많은 경로가 있는 문서 줌 시 속도 향상, 대량 객체 복사 및 클립보드 관리 성능 최적화
- Inkscape 1.5의 새로운 다중 페이지 포맷과 호환성을 위한 브릿지 릴리스로, 이전 버전과의 변환 지원

> [!tip] 왜 중요한가
> 벡터 그래픽 디자인 작업의 안정성과 성능을 크게 향상시키는 중요한 유지보수 릴리스입니다.

## 전문 번역

# Inkscape 1.4.4 릴리스 노트

2026년 5월 6일 공개

## 변경사항 및 버그 수정

### 주요 특징

Inkscape 1.4.4는 유지보수와 버그 수정에 중점을 둔 릴리스입니다. 이번 업데이트를 통해 다음과 같은 개선사항들이 포함되었어요:

- **20개의 크래시 수정** — 그중 3개는 Inkscape가 실행되지 않는 심각한 문제들
- **약 20개의 버그 수정**
- **6개의 성능 개선**
- 새로운 팔레트 추가
- 별과 다각형을 중립 위치로 회전시키는 새로운 버튼
- 27개의 인터페이스 번역 업데이트
- 15개의 문서 번역 업데이트
- Windows ARM 설치 파일 추가

**파일 포맷 호환성 안내**: Inkscape 1.4.4는 이전 버전처럼 향후 출시될 Inkscape 1.5의 새로운 다중 페이지 파일 포맷을 이전 포맷으로 변환할 수 있습니다. 만약 1.4.3 이전 버전을 사용 중이라면, Inkscape 1.5 이상에서 만든 페이지를 읽을 수 없다는 점을 참고하세요. 1.4.3에서 문서를 열어 저장하면 기존 페이지 포맷으로 변환됩니다(MR #7608).

이렇게 호환성을 유지하는 이유가 있습니다. 기존 Inkscape의 페이지 포맷은 Inkscape 전용이었지만, 새로운 포맷은 표준화된 SVG의 `svg:view` 엘리먼트를 사용하게 됩니다. 덕분에 다른 SVG 뷰어에서도 호환되는 장점이 있죠(MR #7525).

### 크래시 수정

이번 버전에서는 다음과 같은 실행 중단 문제들을 해결했습니다:

- 최근 파일 목록에 같은 파일 항목이 두 개 있을 때 Inkscape 시작 불가 (Bug 6002, MR #7750)
- Windows에서 최근 열었던 두 파일이 다른 드라이브의 같은 폴더에 있을 때 실행 불가 (Bug #6028, MR #7754)
- 그래픽 태블릿이 연결된 상태에서 Inkscape 시작 (Bug #5618, MR #7665)
- 새 페이지 생성 시 (Bug #5904, MR #7417)
- 새 문서 페이지 생성 취소 시 (Bug #6060, MR #7787)
- 유니코드 문자 대화상자 종료 시 (Bug #6014, MR #7853)
- 특정 PDF 파일 열 때 (Bug #6021, MR #7690)
- 손상된 마커가 있는 SVG 파일 열 때 (Bug #6040, MR #7768)
- 내보내기 대화상자가 열린 상태에서 특정 SVG 파일 열고 닫을 때 (Bug #5141, MR #7139)
- Inkscape 1.2 이전 버전에서 만든 손상된 Rotate Copies Live Path Effect가 있는 SVG 파일 열 때 (Bug #5991, MR #7782)
- 가져온 래스터 이미지의 너비/높이 비율이 잠긴 상태에서 크기를 0 또는 빈값으로 설정할 때 (Bug #5428, MR #7139)
- 선택된 객체에 빈 텍스트 객체가 있을 때 Tweak 도구 사용 (Bug #5918, MR #7474)
- XML 편집기로 Connector 도구로 만든 경로의 데이터('d')를 삭제할 때 (MR #7772)
- 특정 상황에서 Connector 도구로 만든 선의 취소 (Bug #5949, MR #7510)
- 겹치는 두 객체를 Connector 도구로 연결하려 할 때 (Bug #5314, MR #7535)
- PowerStroke Live Path Effect가 적용된 객체 선택 시 (Bug #5997, MR #7723)
- 그룹에 Corners LPE를 적용하려 할 때 (Bug #5984, MR #7845)
- XML 편집기에서 그래디언트의 중지점을 제거한 후 편집하려 할 때 (Bug #1299, MR #7532)
- 복잡한 경로에 Break Apart 적용 시 (Bug #6067, MR #7800)
- 최대 개수의 추적으로 래스터 이미지를 추적할 때 (Bug #5475, MR #7880)

### 일반 사용자 인터페이스

**캔버스**

레이어 및 객체 대화상자에서 객체를 그룹 사이로 이동할 때 캔버스의 바운딩 박스 크기와 위치가 올바르게 표시되도록 수정했습니다(Bug #5996, MR #7704).

많은 경로가 포함된 문서를 확대할 때 성능이 크게 향상되었습니다(Bug #6110, MR #7876).

**기타**

새 릴리스에 맞춰 버전 번호(1.4.4)가 포함된 그래픽을 업데이트했습니다(MR #7855).

**클리핑**

Inverse Clip(Live Path Effect)을 해제할 때 클립 경로가 보이지 않는 문제를 해결했습니다(Bug #5966, MR #7521).

**복사/붙여넣기**

그래디언트가 많은 객체들을 복사하여 붙여넣을 때 속도가 개선되었습니다(Bug #2403, MR #7856).

클립보드에 무언가를 복사할 때 특정 클립보드 매니저나 PovRay를 사용하는 환경에서 Inkscape가 느려지거나 붙여넣은 콘텐츠의 형식이 잘못되는 문제를 수정했습니다(MR #7858).

### 대화상자

**채우기 및 획 대화상자**

색상 슬라이더의 더하기/빼기 버튼을 누르거나 텍스트 필드에 색상을 입력할 때 각 단계마다 실행 취소 기록이 생기는 문제를 없앴습니다(MR #7854).

**레이어 및 객체 대화상자**

많은 객체가 선택된 상태에서 대화상자를 열 때의 성능이 개선되었습니다.
