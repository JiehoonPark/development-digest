## 핵심 개념

> [!abstract]
> 코딩 에이전트 개발에서 실제 병목 지점을 분석한 글로, 코드 작성 자체보다 다른 요소들이 개발 속도를 제한할 수 있음을 제시합니다.

## 상세 내용

- 코드 생성은 개발 프로세스의 일부일 뿐, 실제 병목은 문제 정의나 요구사항 분석 등 다른 단계에 있을 수 있음
- AI 에이전트의 효율성 향상은 코드 생성 능력보다 전체 워크플로우 최적화에 초점을 맞춰야 함

> [!tip] 왜 중요한가
> AI 기반 개발 도구의 진정한 가치를 평가할 때 코드 생성 능력만 고려하는 함정을 피할 수 있습니다.
