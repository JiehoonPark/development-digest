## 핵심 개념

> [!abstract]
> 확산 모델의 샘플링 과정을 신경망이 직접 예측하는 'flow maps'를 통해 샘플링 속도를 향상시키는 방법을 설명합니다.

## 상세 내용

- 확산 모델은 각 단계에서 탄젠트 방향을 추정하는 반복 과정인데, flow maps는 경로 상의 임의의 점을 다른 점에서 직접 예측 가능
- Flow maps는 더 빠른 샘플링, 효율적인 보상 기반 학습, 개선된 샘플링 제어 가능성 제공
- Vector calculus 개념이 필요하며, 확산 모델에 대한 이해가 선행되어야 함

> [!tip] 왜 중요한가
> 생성형 AI 모델의 샘플링 효율성을 크게 향상시킬 수 있는 고급 기법으로, 실무에서 생성 모델의 성능과 비용을 개선할 수 있습니다.

# 플로우 맵으로 확산 모델 샘플링 가속하기

확산 모델에서 샘플을 생성하는 과정은 반복적입니다. 각 단계마다 노이즈 제거 모델이 입력 공간을 지나는 경로의 접선 방향을 추정하고, 이 방향을 따라 작은 스텝을 계속 밟으면서 노이즈 레벨 전체에 걸친 적분을 계산합니다. 결국 단순한 노이즈 분포의 샘플을 목표 분포의 샘플로 점진적으로 변환하는 경로를 그려내는 거죠. 그렇다면 신경망을 훈련해서 이 적분값을 직접 예측할 수는 없을까요? 충분히 가능합니다. 플로우 맵의 세계에 오신 것을 환영합니다!

## 확산 모델의 샘플링을 더 빠르게

확산 모델이 등장한 이후로, 사람들은 계속 샘플링 속도를 개선하려고 노력해왔습니다. 약 2년 전 저는 확산 증류(diffusion distillation)에 관해 썼는데, 이것이 고품질 샘플을 얻기 위한 스텝 수를 줄이는 주요 방법 중 하나입니다. 기본 원리는 크게 바뀌지 않았지만, 그 사이 정말 많은 변형 방법들이 나타났어요.

이번 글에서는 플로우 맵을 자세히 살펴보겠습니다. 확산 모델이 경로상 각 지점의 접선 방향을 예측해서 노이즈와 데이터 사이의 경로를 설명한다면, 플로우 맵은 같은 경로 위의 어떤 점에서 다른 점으로 직접 이동할 수 있습니다. 더 빠른 샘플링에 사용될 수 있을 뿐만 아니라, 보상 기반 학습을 더 효율적으로 하고 샘플링 제어성을 높이는 등 다른 여러 장점도 있어요. 최근에 매우 주목받고 있는 기술입니다.

## 플로우 맵의 정의와 다양한 구현

플로우 맵이 정확히 무엇인지는 비교적 간단히 정의할 수 있지만, 이를 구성하고 훈련하는 방법은 매우 다양합니다. 그리고 확산 모델처럼 문헌마다 용어와 형식이 제각각이라서, 전체 그림을 파악하기가 혼란스러울 수 있죠. 주로 Boffi 외의 분류 체계를 기반으로 해서 정리해보겠습니다.

플로우 맵은 확산 모델의 아이디어 위에 세워진 개념이므로, 확산 모델에 대한 기본 이해가 있다면 좋습니다. 벡터 미적분학에 익숙하면 훈련 방식을 이해하는 데 도움이 되지만, 없다 해도 다른 부분들은 충분히 흥미로울 거예요. 필요하면 제가 쓴 이전 글들(예: "확산 모델의 관점들")을 다시 읽어보셔도 좋습니다. 또는 최근에 나온 Chieh-Hsin Lai와 동료들의 확산 모델 저작이 수학적 엄밀함과 직관적 설명을 잘 결합한 추천 자료입니다.

## 노이즈에서 데이터로 가는 경로 그리기

플로우 맵을 이해하려면 확산 모델을 노이즈와 데이터 사이의 전단사 함수로 보는 관점이 중요합니다. 각 샘플 쌍을 연결하는 유일한 경로들이 서로 교차하지 않는다는 특성이 있거든요. 먼저 확산 모델의 샘플링 알고리즘을 자세히 들여다본 후, 거기서부터 플로우 맵으로 나아가겠습니다.

### 확산 모델에서 샘플링하기

오늘날 확산 모델을 위한 샘플링 알고리즘은 많지만, 모두 확률적(stochastic) 방식과 결정적(deterministic) 방식 둘 중 하나에 속합니다. 결정적 샘플링의 원리는 이전에 다룬 적이 있지만, 플로우 맵 개발의 기초이므로 다시 정리해볼 가치가 있습니다.

핵심은 이렇습니다. 노이즈가 섞인 관찰값 $\mathbf{x}_t$가 주어졌을 때, 깨끗한 원본 데이터의 예측값 $\hat{\mathbf{x}}_0 = \mathbb{E}\left[ \mathbf{x}_0 \mid \mathbf{x}_t \right]$을 예측하는 노이즈 제거 모델이 있다면, 두 가지 서로 다른 반복적 생성 절차를 만들 수 있습니다.

**확률적 방식**이 더 직관적입니다. 각 반복 단계마다 현재 노이즈가 섞인 샘플이 주어졌을 때, 좀 더 깨끗한 샘플들의 조건부 분포 $p(\mathbf{x}_{t-1} \mid \mathbf{x}_t)$에서 샘플을 뽑습니다. 이렇게 하면 손상 과정을 한 번에 한 스텝씩 역으로 진행할 수 있죠. 다행히 노이즈 제거 모델의 예측값을 사용해서 이 분포를 근사할 수 있습니다. 시간 $t$와 $t-1$ 사이의 간격이 작을수록 근사가 더 정확해집니다. 많은 반복을 거친 후 노이즈가 사라지면서 $t=0$에서 깨끗한 데이터 분포의 샘플을 얻게 됩니다. 이것이 원래의 DDPM 알고리즘이 작동하는 방식입니다. 확산 모델의 확률 미분방정식(SDE) 형식을 기반으로 한 샘플링 알고리즘들도 입력 공간에서 비슷한 확률적 궤적을 만듭니다.

**결정적 절차**는 초기 단계를 제외하고 어느 시점에도 난수를 뽑지 않습니다.
