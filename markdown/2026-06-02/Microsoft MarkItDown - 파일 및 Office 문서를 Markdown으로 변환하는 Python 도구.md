## 핵심 개념

> [!abstract]
> Microsoft에서 개발한 Python 도구로 다양한 파일 형식과 Office 문서를 Markdown으로 변환할 수 있습니다. 기술 문서 작성이나 콘텐츠 변환 작업을 자동화하는 데 활용됩니다.

## 상세 내용

- 다양한 파일 형식(Office, PDF 등)을 Markdown으로 일괄 변환 지원
- Python 기반 도구로 개발 워크플로우에 쉽게 통합 가능

> [!tip] 왜 중요한가
> 기술 문서화나 LLM 학습 데이터 준비 시 파일 형식 변환을 자동화할 수 있어 개발 생산성을 향상시킵니다.
