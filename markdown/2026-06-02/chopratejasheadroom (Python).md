## 핵심 개념

> [!abstract]
> LLM에 도달하기 전에 컴파일러 출력, 로그, 파일, RAG 청크를 압축하여 토큰 사용량을 60-95% 줄이면서 동일한 답변을 유지하는 도구이다.

## 상세 내용

- 라이브러리, 프록시, MCP 서버로 제공되며 다양한 환경에서 사용 가능
- LLM 비용 절감과 처리 속도 향상으로 실용적 가치 제공

> [!tip] 왜 중요한가
> LLM 기반 애플리케이션 개발 시 토큰 비용을 크게 절감할 수 있는 실용적 솔루션이다.
