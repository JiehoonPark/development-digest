## 핵심 개념

> [!abstract]
> 저시력 사용자가 Kagi 유료 검색 엔진으로 전환하여 광고와 시각적 혼란이 없는 검색 경험을 얻었으며, 개인 맞춤 필터와 접근성 기능을 통해 검색 효율을 크게 향상시켰습니다.

## 상세 내용

- Kagi는 구독 기반의 광고 없는 검색으로 깔끔한 결과 페이지 제공 (월 $5~$25)
- Lenses(저장된 필터), 도메인 커스터마이징, Bangs(사이트 직접 검색) 등 개인화 기능 풍부
- 자동 재생 콘텐츠와 SEO 스팸이 없어 저시력 사용자의 시각 피로 감소

> [!tip] 왜 중요한가
> 접근성이 필요한 개발자들이 더 효율적으로 정보를 검색하고 생산성을 높일 수 있습니다.

# 저시력 사용자를 위한 Kagi 검색 엔진 활용 가이드

## 검색 엔진의 시각적 피로, 이제 그만

강의 자료를 찾기 위해 검색 엔진을 사용하던 중 뭔가 이상함을 느꼈어요. 예전보다 훨씬 자주 눈을 쉬게 되고, 시각적 피로를 많이 느꼈거든요. 원인은 명확했는데요. 검색 결과 페이지가 너무 복잡했던 거예요. AI 요약, 저품질 콘텐츠, 광고, 자동재생 영상까지 뒤섞여 있으니 각 결과를 읽기가 정말 힘들었습니다. 쓸데없는 콘텐츠를 보느라 에너지를 다 써버리니 실제로 필요한 정보에 집중하기 어려웠어요.

음성 변환 같은 기능을 써봐도 여전히 문제는 남아있었습니다. 그러다가 Kagi라는 유료 검색 엔진을 써봤는데, 정말 달라질 정도더라고요. 광고가 없고 다양한 커스터마이징과 접근성 기능을 제공하는 사용자 중심의 검색 경험을 제공하거든요.

검색 결과 페이지의 시각적 복잡함이 얼마나 스트레스였는지 그걸 없앨 때 비로소 깨달았습니다. 그 이후로 주변 사람들에게 Kagi를 계속 추천하고 다니고 있어요.

**공지: 이 글은 Kagi의 지원을 받지 않았으며, 회사와 아무런 관계도 없습니다. 저는 개인적으로 Kagi 구독료를 내고 있습니다. 흥미롭게도 글을 작성하던 중 누군가 제 이름을 Kagi의 Small Web 추천 목록에 올려둔 걸 발견했네요!**

## Kagi는 어떤 서비스인가?

Kagi는 광고주가 아닌 사용자 구독료로만 운영되는 유료 검색 엔진입니다. 따라서 검색 결과는 SEO 스팸이나 광고료가 아닌 순수한 품질로만 순위가 결정돼요. 광고가 없고 추적 기반 수익화도 없으니 결과 페이지가 훨씬 깔끔합니다. 자연스레 관련 정보를 찾기가 더 쉬워지고, 사용자도 자신의 검색 경험을 더 많이 통제할 수 있죠. Google이나 Bing의 대안으로 자주 언급되는 서비스입니다.

## 가격 플랜

Kagi는 무료 체험과 여러 유료 플랜을 제공합니다:

- **체험판 (무료)**: 총 100번 검색 + 기본 AI 접근
- **Starter ($5/월)**: 월 300회 검색 + Kagi Assistant (Quick mode)
- **Professional ($10/월)**: 무제한 검색 + 더 큰 AI 할당량 (Quick mode)
- **Ultimate ($25/월)**: Professional의 모든 기능 + Kagi Assistant (Research mode) + 최신 AI 모델 접근

가정용과 조직용 Family, Team 플랜도 있습니다.

연간 구독 할인도 있고, "Fair Pricing" 정책이 있어서 한 달간 Kagi를 사용하지 않으면 그달 비용이 계정에 환불됩니다. 플랜을 내릴 때도 차액이 환불되죠.

## 검색 결과를 내 방식대로 커스터마이징하기

Kagi 검색창은 다른 검색 엔진과 비슷하지만, 결과 페이지는 훨씬 쓰기 편합니다. 디스플레이 광고도, 자동재생 미디어도, 스폰서 결과도 없어요. 위에 과장된 AI 요약이 떠있지도 않고요. 깔끔하게 정리된 링크들만 있습니다. Quick Answer라는 선택적 요약 기능이 있지만 기본적으로 숨겨져 있어요.

저는 검색 결과를 개인화하는 다양한 도구들을 애용하고 있습니다:

**Lenses (렌즈)**
특정 유형의 소스로 결과를 좁혀주는 저장된 필터예요. 기본 렌즈에는 Forums(온라인 커뮤니티, Reddit), Academic(학술 기관), Programming(코딩 사이트, 개발 포럼) 등이 있습니다. 특정 웹사이트, 지역, 기타 조건을 포함한 커스텀 렌즈도 만들 수 있는데, 저는 저시력과 보조기술 관련 렌즈들을 직접 만들어서 사용합니다. Settings > Search > Lenses에서 설정할 수 있어요.

**도메인 제어 (차단, 낮추기, 올리기, 고정)**
각 검색 결과에는 방패 아이콘이 있어서 특정 사이트를 완전히 차단하거나, 순위를 낮추거나, 올리거나, 맨 위에 고정할 수 있습니다. 이런 개인화 설정은 당신의 Kagi 계정에만 저장되고 다른 곳에 공유되지 않아요. Settings > Search > Personalized Results에서 관리합니다.

**Bangs (뱅)**
특정 웹사이트로 검색을 리다이렉트하는 단축어입니다. 예를 들어 !w를 입력하면 Wikipedia로 검색이 넘어가죠. 자주 방문하는 사이트들의 커스텀 뱅도 만들 수 있어요. 저는 개인 블로그, 대학 웹사이트, 자주 찾는 사이트들에 대한 뱅을 만들어서 빠르게 검색합니다. Settings > Search에서 설정할 수 있고, 커스텀 뱅은 직접 추가할 수 있습니다.
