## 핵심 개념

> [!abstract]
> Chrome DevTools 기능을 코딩 에이전트에서 사용할 수 있도록 MCP 인터페이스로 제공하는 TypeScript 프로젝트다.

## 상세 내용

- AI 코딩 에이전트가 Chrome DevTools의 디버깅, 성능 분석 등의 기능을 활용할 수 있도록 표준화된 MCP 프로토콜로 통합한다.

> [!tip] 왜 중요한가
> AI 에이전트 기반 개발 환경에서 브라우저 디버깅 도구에 접근할 수 있어 자동화된 웹 애플리케이션 개발과 테스트 효율성을 높인다.
