## 핵심 개념

> [!abstract]
> Context.dev는 웹사이트에서 구조화된 데이터를 추출하는 API로, Mintlify, daily.dev, Propane 등 여러 회사가 자동화된 문서 생성, 데이터 검증, 온보딩 개선에 활용하고 있습니다. 기업들이 몇 분의 통합 시간으로 브랜드 데이터를 자동으로 처리하고 활성화율을 높이는 데 성공했습니다.

## 상세 내용

- 웹사이트에서 자동으로 구조화된 데이터를 추출하여 LLM 파이프라인에 활용 가능
- GitHub repo를 문서 사이트로 변환, 회사 검증, 온보딩 자동화 등 다양한 사용 사례 제공
- 10분 이내의 짧은 통합 시간으로 빠른 구현 가능

> [!tip] 왜 중요한가
> 웹 데이터 추출이 필요한 AI 파이프라인 개발자에게 적분 시간을 단축시켜줍니다.
