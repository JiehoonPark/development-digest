## 핵심 개념

> [!abstract]
> Lazypi.org는 Python 프로젝트 설정을 위한 의견 기반(opinionated) 구성 도구입니다. 개발자 선택지를 최소화하고 권장 설정을 자동 적용합니다.

## 상세 내용

- 의견 기반 설정으로 초기 프로젝트 구성 시간 단축
- Python 개발 커뮤니티의 베스트 프랙티스 자동 적용

> [!tip] 왜 중요한가
> 새로운 Python 프로젝트 시작 시 설정 결정을 줄이고 빠르게 개발을 시작할 수 있습니다.
