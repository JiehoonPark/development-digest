## 핵심 개념

> [!abstract]
> Prisma는 PostgreSQL, MySQL, MariaDB, SQL Server, SQLite, MongoDB, CockroachDB를 지원하는 Node.js 및 TypeScript용 최신 ORM 솔루션입니다.

## 상세 내용

- 다양한 데이터베이스(SQL, NoSQL)에 대한 통일된 인터페이스 제공
- TypeScript 네이티브 지원으로 타입 안정성과 개발 생산성 향상

> [!tip] 왜 중요한가
> TypeScript 기반 백엔드 개발에서 데이터베이스 접근을 간편하고 안전하게 처리할 수 있게 해줍니다.
