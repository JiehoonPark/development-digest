## 핵심 개념

> [!abstract]
> Kyutai Labs의 Pocket TTS는 CPU 리소스만으로 동작하며 휴대 가능한 경량 음성 합성 모델입니다. 별도의 GPU나 클라우드 서비스 없이 로컬에서 실행할 수 있도록 설계되었습니다.

## 상세 내용

- CPU 기반 실행으로 별도 하드웨어 투자 없이 음성 합성 가능
- 경량화로 모바일이나 엣지 디바이스에 배포 가능

> [!tip] 왜 중요한가
> 개발자가 클라우드 API 의존 없이 독립적으로 TTS 기능을 구현할 수 있으며, 비용과 지연 시간을 크게 줄일 수 있습니다.
