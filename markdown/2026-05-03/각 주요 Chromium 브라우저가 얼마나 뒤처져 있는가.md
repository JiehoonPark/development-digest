## 핵심 개념

> [!abstract]
> Chromium 버전 지연이 사용자 보안에 미치는 영향을 분석한 대시보드입니다. 오래된 Chromium 버전을 사용하는 브라우저는 이미 패치된 보안 취약점에 노출되어 있습니다.

## 상세 내용

- 구형 Chromium 버전을 사용하는 브라우저 사용자는 공개된 보안 패치가 아직 적용되지 않은 알려진 취약점에 노출됩니다.
- 공격자들은 Chromium 소스에서 공개된 패치를 바탕으로 아직 업데이트되지 않은 브라우저를 적극적으로 공격합니다.

> [!tip] 왜 중요한가
> 브라우저 업데이트 관리가 사용자 보안에 직접적인 영향을 미치므로 개발자와 사용자 모두 Chromium 버전 추적의 중요성을 인식해야 합니다.

## 전문 번역

# Chromium 버전 격차가 중요한 이유

브라우저가 오래된 Chromium 버전을 탑재하고 있다면, 해당 브라우저 사용자들은 이미 알려져 있고 패치된 보안 취약점에 노출되게 됩니다. 공격자들은 이런 취약점을 적극적으로 악용하는데요. Chromium 소스에는 패치가 공개되어 있지만, 아직 구형 브라우저 사용자들에게 배포되지 않았기 때문입니다.
