## 핵심 개념

> [!abstract]
> ruflo는 Claude 기반의 멀티 에이전트 시스템을 배포하고 조율하는 플랫폼입니다. 분산 에이전트 스웜, RAG 통합, Claude Code 연동 등 엔터프라이즈급 기능을 제공합니다.

## 상세 내용

- 멀티 에이전트 스웜과 자율 워크플로우 조율 기능으로 복잡한 AI 작업을 자동화
- Claude Code와 네이티브 통합으로 코딩 작업 자동화 지원

> [!tip] 왜 중요한가
> Claude를 활용한 멀티 에이전트 시스템을 프로덕션 환경에서 쉽게 배포할 수 있는 기반을 제공합니다.
