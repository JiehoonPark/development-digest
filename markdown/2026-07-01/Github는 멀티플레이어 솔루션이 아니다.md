## 핵심 개념

> [!abstract]
> Github이 협업 도구로서의 한계를 다루는 영상으로, 실제 팀 협업 환경에서 발생하는 문제점들을 분석합니다.

## 상세 내용

- Github은 코드 호스팅과 버전 관리에 특화되어 있으나 실시간 협업 기능은 제한적입니다.

> [!tip] 왜 중요한가
> 팀 협업 도구 선택 시 Github의 한계를 이해하고 추가 도구 도입 필요성을 판단하는 데 도움이 됩니다.
