## 핵심 개념

> [!abstract]
> Common User Passwords Profiler(CUPP)는 사용자 정보를 기반으로 일반적인 비밀번호를 프로파일링하는 Python 도구입니다.

## 상세 내용

- 사용자 정보로부터 가능성 높은 비밀번호 조합을 생성합니다.
- 보안 테스트 및 침투 테스트에 활용되는 도구입니다.

> [!tip] 왜 중요한가
> 보안 평가와 비밀번호 강도 테스트를 수행하는 개발자에게 유용한 프로파일링 도구입니다.
