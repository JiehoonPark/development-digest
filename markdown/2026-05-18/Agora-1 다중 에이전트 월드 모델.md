## 핵심 개념

> [!abstract]
> Odyssey가 공개한 Agora-1은 여러 참여자(인간 또는 AI)가 실시간으로 같은 세계 시뮬레이션을 공유하고 상호작용할 수 있는 첫 번째 다중 에이전트 월드 모델이다. GoldenEye를 기반으로 최대 4명의 플레이어가 동시에 생성된 환경에서 상호작용할 수 있으며, 시뮬레이션과 렌더링을 분리한 아키텍처로 일관성 있는 공유 월드 상태를 유지한다.

## 상세 내용

- 월드 모델의 진화: 기존 월드 모델이 단일 참여자로 제한된 반면, Agora-1은 최대 4명이 동시에 같은 생성 환경에서 상호작용하는 멀티플레이 데스매치 시뮬레이션 지원. 모든 콘텐츠가 실시간으로 생성되며, 플레이어의 액션에 따라 공유 월드 상태를 유지하고 각 플레이어에게 동시에 렌더링된 픽셀 스트리밍.
- 아키텍처의 혁신: 기존 접근법(Multiverse의 분할 화면 방식, Solaris의 시퀀스 연결)과 달리, Agora-1은 시뮬레이션과 렌더링을 명확히 분리. 이를 통해 여러 독립적 시점에서 같은 시뮬레이션된 세계의 일관된 뷰를 생성 가능하며, 플레이어 수 증가에 따른 컨텍스트 성장 문제 해결.
- 두 가지 핵심 학습 함수: 첫째, 플레이어 상호작용에 응답하여 월드 상태가 시간에 따라 어떻게 진화하는지 학습하는 상태 모델(게임 내부 상태 직접 학습). 둘째, 공유 상태를 시각적으로 렌더링하는 방법을 학습하는 렌더링 모델(DiT 기반, 프롬프트가 아닌 공유 게임 상태로 직접 조건화).
- 기술적 도전과 해결: 이산적 게임 상태는 연속적 시각 도메인과 구조적으로 다르므로 게임플레이 상태 모델링에 특화된 아키텍처와 대량의 구조화된 학습 데이터 필요. 렌더링 모델은 같은 공유 상태를 여러 시점에서 동시에 생성하면서도 시각적 일관성 유지해야 함.
- 게임 엔진 유사성: 현대 게임 엔진의 구조와 유사하게 시뮬레이션과 렌더링을 분리했으나, 두 컴포넌트 모두 완전히 학습된 시스템으로 하드코딩된 게임플레이 로직이나 렌더링 규칙에 의존하지 않음. 이로 인해 기반이 되는 게임 상태를 직접 조작하여 게임플레이 역학을 보존하면서 완전히 새로운 레벨 생성 가능.
- 현재 상태와 향후 전망: Agora-1의 상태 모델은 현재 상대적으로 단순하지만, 이는 근본적 제한이 아니며 원칙적으로 내부 상태 표현을 임의로 확장하여 더 복잡한 시뮬레이션과 게임플레이 역학 가능. 향후 이들 시스템이 규칙과 상태 표현을 초월하여 일반화되면, 사용자 상호작용에서 직접 생성된 완전히 새로운 경험 창출 가능.
- 연구 방향: 다중 에이전트 상호작용이 기초 월드 모델로 확장되면서도 개방형 행동이나 일반성을 손상시키지 않는 방법 이해에 집중. 명시적으로 손으로 작성된 조정 메커니즘보다 학습된 시스템을 통해 달성 가능하다고 판단하며, Agora-1 같은 연구 환경이 제어된 설정에서 이를 탐색하는 데 제공.

> [!tip] 왜 중요한가
> 멀티플레이 게임, 로봇 시뮬레이션, 방위 응용, 교육 등 여러 분야에서 다중 에이전트가 협력하는 실시간 시뮬레이션이 필요한 상황에서 혁신적인 기술을 제시하며, 월드 모델 연구의 새로운 방향을 제시한다.

## 전문 번역

# Agora-1: 다중 에이전트 월드 모델

Agora-1은 인간과 AI 같은 여러 참여자가 같은 세계 시뮬레이션을 실시간으로 공유하고 상호작용할 수 있게 해줍니다.

오늘 우리는 Agora-1을 공개합니다. 게임, 로봇, 방위, 교육, 파운데이션 모델 등 다양한 분야에서 새로운 공유 경험을 어떻게 만들 수 있을지 탐구하는 다중 에이전트 월드 모델의 첫 번째 작품입니다. 월드 모델은 임의의 환경에 대한 고충실도 시뮬레이션을 생성하는 강력한 도구인데, 지금까지는 한 번에 단 하나의 참여자만 지원했거든요. Agora-1은 여기서 한 걸음 더 나아가 여러 에이전트가 함께 움직이는 세계를 시뮬레이션합니다.

## GoldenEye를 통한 다중 에이전트 월드 모델 탐구

다중 에이전트 월드 모델을 연구하기 위해 팀원들이 어렸을 때 즐겨 했던 GoldenEye를 선택했습니다. 게임은 AI 연구에 오랫동안 좋은 환경이 되어줬으니까요. Atari부터 Minecraft, StarCraft을 거쳐 이제 GoldenEye까지 다양한 게임이 AI 학습의 장이 되고 있습니다.

Agora-1은 최대 4명의 플레이어가 생성된 같은 세계에서 실시간으로 상호작용하게 해줍니다. 플레이어들은 공유 데스매치 시뮬레이션에 매칭되는데, 모든 참여자가 동시에 같은 생성 세계와 상호작용합니다. 당신이 경험하는 모든 것은 Agora-1이 실시간으로 생성한 것입니다. 모델은 플레이어의 행동으로부터 상호작용을 시뮬레이션하고, 참여자 간 공유 월드 상태를 유지하며, 생성된 화면을 모든 플레이어에게 동시에 스트리밍합니다. 다시 말해 Agora-1은 학습된 게임 엔진처럼 작동하는 거죠.

## 기존 접근법의 한계와 새로운 방향

전통적인 월드 모델은 시뮬레이션 동역학과 렌더링을 하나의 모델 안에 결합합니다. 지금까지 Multiverse, Solaris, MultiGen 같은 여러 접근법들이 월드 모델에서 다중 에이전트 상호작용을 탐구했는데요.

**Multiverse**는 에이전트 상태를 하나의 "스플릿 스크린" 표현으로 연결하는데, 결국 여러 플레이어를 하나의 월드 상태처럼 다루는 셈입니다.

**Solaris**는 다른 방식입니다. 각 참여자를 하나의 자동회귀 확산 트랜스포머의 시퀀스 차원으로 연결해서 더욱 견고한 공유 시뮬레이션을 만듭니다. 하지만 플레이어 수가 늘어남에 따라 모델 컨텍스트가 증가하기 때문에 선형적으로 확장되지 않습니다. 게다가 Multiverse와 Solaris 모두 플레이어들이 서로를 볼 수 없을 때 일관성을 유지하기 어려워합니다.

Agora-1은 다른 방향을 택했습니다. **시뮬레이션과 렌더링을 분리**한 것이죠. MultiGen처럼 Agora-1도 참여자들 사이에 명시적인 공유 월드 상태를 유지합니다. 다만 그 공유 상태에서 시뮬레이션 동역학과 렌더링을 모델링하는 방식이 다릅니다. 이 두 기능을 분리함으로써 Agora-1은 같은 시뮬레이션 세계를 여러 독립적인 시점에서 일관되게 표현할 수 있습니다. 이는 멀티플레이어 게임, 로봇공학, 다중 시점 시뮬레이션 같은 응용을 가능하게 합니다.

## Agora-1의 핵심: 두 가지 학습 함수

Agora-1은 두 가지 뚜렷한 기능을 학습합니다.

먼저 월드 상태가 플레이어의 상호작용에 반응해 시간에 따라 어떻게 진화하는지 학습합니다. 이를 위해 하나 이상의 게임—우리의 경우 GoldenEye—의 내부 상태에서 직접 모델을 학습시킵니다. 이 모델은 기저 게임 동역학과 플레이어 행동으로부터 상태가 어떻게 전이되는지를 배웁니다.

다음으로 Agora-1은 그 공유 상태를 시각적으로 렌더링하는 방법을 학습합니다. 이는 프롬프트나 이미지 같은 전통적인 조건 신호가 아니라 공유 게임 상태에 직접 조건화된 DiT 기반 월드 모델을 사용해 이루어집니다.

이 분리를 현대적 게임 엔진의 구조에 비유할 수 있습니다. 차이점은 두 컴포넌트 모두 완전히 학습된 시스템이라는 것입니다. 하드코딩된 게임플레이 로직이나 렌더링 규칙에 의존하지 않고, 데이터에서 직접 학습합니다.

## 설계의 과제와 장점

두 모델 모두 고유한 연구 과제를 제시합니다. 이산 게임 상태는 대부분의 DiT 기반 월드 모델이 다루는 연속적인 시각 영역과 구조적으로 다릅니다. 따라서 게임플레이 상태 모델링을 위해 특별히 설계된 아키텍처와 많은 양의 구조화된 훈련 데이터가 필요합니다. 동시에 렌더링 모델은 같은 공유 상태를 여러 시점에서 동시에 일관되게 시각화하는 법을 배워야 합니다.

이런 아키텍처의 결과로 기저 게임 상태를 직접 조작할 수 있게 됩니다. Agora-1은 원본 게임의 게임플레이 동역학을 유지하면서 완전히 새로운 레벨을 생성할 수 있다는 뜻입니다.

## 현재의 한계와 미래의 가능성

현재 Agora-1의 상태 모델은 상대적으로 단순합니다. 하지만 이것이 근본적인 제약은 아닙니다. 원칙적으로 내부 상태 표현은 임의로 확장될 수 있으며, 이는 점점 더 복잡한 시뮬레이션과 게임플레이 동역학을 가능하게 합니다. 시간이 지나면서 이런 시스템들은 규칙과 상태 표현 전반에 걸쳐 일반화되어, 사용자와의 직접적인 상호작용으로부터 완전히 새로운 경험을 생성할 수 있을 거라 예상합니다.

## 더 큰 그림: 파운데이션 월드 모델로의 확장

우리의 더 광범위한 연구 목표는 다중 에이전트 상호작용이 파운데이션 월드 모델로 어떻게 확장될 수 있으면서도 개방성과 일반성을 잃지 않을 수 있는지 이해하는 것입니다. 우리는 명시적으로 손으로 작성된 조정 메커니즘이 아닌 학습된 시스템을 통해 이것이 가능하다고 믿습니다. Agora-1 같은 연구 환경은 이런 개념들을 통제된 설정에서 탐구할 수 있는 기반을 제공합니다.
