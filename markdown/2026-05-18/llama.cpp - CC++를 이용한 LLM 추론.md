## 핵심 개념

> [!abstract]
> llama.cpp는 C/C++로 구현된 경량 LLM 추론 엔진으로, 로컬 환경에서 효율적으로 대규모 언어 모델을 실행할 수 있게 해준다.

## 상세 내용

- C/C++ 구현으로 메모리 효율성과 빠른 추론 속도 제공. CPU만으로도 모델 실행 가능하며, GPU 가속 옵션 지원.
- 다양한 양자화 포맷 지원으로 모델 크기를 크게 줄이면서도 성능 유지 가능.

> [!tip] 왜 중요한가
> 로컬 환경에서 LLM을 저비용으로 실행해야 하는 개발자와 연구자에게 필수적인 도구다.
