## 핵심 개념

> [!abstract]
> CopilotKit은 React, Angular, 모바일 및 Slack 등 다양한 플랫폼에서 AI 에이전트와 생성형 UI를 구축하기 위한 종합적인 프론트엔드 스택입니다. AG-UI Protocol의 개발사로서 LLM 기반 인터페이스와 에이전트 통합을 위한 도구와 컴포넌트를 제공합니다. TypeScript 기반으로 개발되었으며, 다양한 개발 환경에서 에이전트 기능을 쉽게 통합할 수 있도록 설계되었습니다.

## 상세 내용

- React, Angular, 모바일 앱, Slack 등 여러 플랫폼을 지원하는 통합 프론트엔드 솔루션으로, 단일 스택으로 다양한 환경에 에이전트 기능을 배포할 수 있습니다.
- AG-UI Protocol을 개발한 팀이 만든 공식 구현으로, LLM 기반 사용자 인터페이스와 에이전트 상호작용을 위한 표준화된 방식을 제시합니다.
- 생성형 UI 기능을 통해 AI가 동적으로 사용자 인터페이스를 생성하고 조작할 수 있으며, 이를 통해 더 자연스러운 사용자 경험을 구현할 수 있습니다.
- TypeScript 기반의 타입 안전한 구현으로 개발 과정에서의 오류를 줄이고, IDE 자동완성 지원으로 개발 생산성을 향상시킵니다.
- GitHub 저장소를 통해 오픈소스로 공개되어 있으며, 커뮤니티 기여와 확장이 가능합니다.

> [!tip] 왜 중요한가
> AI 에이전트와 생성형 UI 기술이 빠르게 발전하는 상황에서, 여러 플랫폼을 아우르는 통합 솔루션을 제공하므로 개발자는 각 플랫폼별로 다른 도구를 배울 필요 없이 일관된 방식으로 AI 기능을 구현할 수 있습니다.
