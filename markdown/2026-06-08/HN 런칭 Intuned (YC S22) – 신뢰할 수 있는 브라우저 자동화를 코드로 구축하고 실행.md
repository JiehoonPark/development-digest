## 핵심 개념

> [!abstract]
> Intuned는 프롬프트 설명만으로 Playwright 기반의 프로덕션급 브라우저 자동화 코드를 AI가 생성하고 배포하며, 웹사이트 변화에 따라 자동으로 수정해주는 서비스입니다. 스크래핑, 크롤링, RPA, AI 자동화 등 다양한 유스케이스를 지원하며 TypeScript/Python으로 작성된 코드를 직접 제어할 수 있습니다.

## 상세 내용

- Intuned Agent가 자연어 프롬프트와 스키마로부터 Playwright 스크래퍼를 생성하고, 사이트 변경 시 AI로 자동 수정
- anti-detection, captcha 해결, 인증 처리, 스케줄링 등이 내장되어 인프라 관리 없이 배포 가능
- TypeScript/Python 코드 작성 자유도와 함께 Anthropic/OpenAI 등 다양한 AI 모델과 통합 지원

> [!tip] 왜 중요한가
> 개발자가 복잡한 브라우저 자동화 로직을 직접 작성하지 않고도 생성/관리할 수 있어 개발 시간을 대폭 단축할 수 있습니다.

## 전문 번역

# 브라우저 자동화 코드 작성은 이제 그만하세요

데이터를 추출하거나 웹 자동화를 구현해야 한다면, 직접 코드를 짤 필요가 없습니다. Intuned Agent가 Playwright 코드를 생성하고 배포한 뒤, 사이트가 변경되면 자동으로 수정해줍니다. 개발자는 코드 작성의 수고 없이 안정적인 자동화만 얻을 수 있죠.

## 웹 크롤링 및 데이터 수집

### 웹 스크래퍼

이커머스 사이트, 정부 포털, 구인 게시판 등 어떤 웹사이트에서도 데이터를 추출할 수 있습니다. Intuned Agent를 활용하면 Playwright 코드를 자동 생성하고 지속적으로 유지보수할 수 있어요. 기본으로 탑재된 은폐 기능, 인증 처리, 일정 실행, 대규모 확장을 통해 안정적으로 운영하세요.

**주요 기능:**
- **Intuned Agent** — 프롬프트와 데이터 스키마만 입력하면 스크래퍼 자동 생성, 사이트 변경 시 AI가 자동 수정
- **내 코드, 내 방식** — TypeScript 또는 Python으로 Playwright나 호환 라이브러리 사용
- **은폐 및 인증** — 탐지 회피, CAPTCHA 해결, 로그인 처리 모두 내장
- **정기 실행 및 모니터링** — 반복 실행, 완전한 로그 및 세션 기록
- **자동 확장** — 1대 또는 수백 대의 머신, 우리가 관리

### 웹 크롤러

수천 개 페이지를 돌아다니며 데이터를 수집하는 크롤러를 구축하세요. 링크를 따라가고, 사이트맵을 파싱하고, 콘텐츠를 색인하는 크롤러를 만들 수 있습니다. 기본 제공되는 은폐 기능, 정기 실행, 대규모 확장으로 걱정 없이 운영하세요.

**주요 기능:**
- **내 코드, 내 방식** — TypeScript 또는 Python으로 Playwright나 호환 크롤링 라이브러리 사용
- **Crawl4AI 지원** — Crawl4AI를 1급 지원, 인프라 관리 없이 크롤러 배포 및 확장
- **은폐 모드** — 탐지 회피, 프록시, CAPTCHA 해결 내장
- **정기 실행 및 모니터링** — 반복 크롤링, 완전한 로그 및 세션 기록
- **확장 가능한 인프라** — 머신 개수 제어, 나머지는 우리가 처리

### RPA (로봇 프로세스 자동화)

사용자를 대신해 웹사이트와 상호작용하는 자동화를 구축하세요. 양식 제출, 데이터 입력, 계정 작업 등 API를 제공하지 않는 서비스와의 통합에 완벽합니다. 기본 제공되는 인증 처리, 은폐 기능, 동시성 제어로 안정적으로 배포하세요.

**주요 기능:**
- **기본 제공 인증** — create.ts와 check.ts (또는 .py) 작성, Intuned가 세션 생명주기 관리 (검증, 재사용, 재생성)
- **내 코드, 내 방식** — TypeScript 또는 Python으로 Playwright 사용, 또는 Anthropic Computer Use, OpenAI CUA, Stagehand, Browser-use, Gemini Computer Use 등 AI 기반 방식
- **UI 액션 또는 네트워크 호출** — 브라우저 상호작용으로 자동화하거나 더 빠른 인브라우저 네트워크 요청으로 자동화
- **은폐 모드** — 탐지 회피, 프록시, CAPTCHA 해결 내장
- **API 기반, 동시성 제어** — API 엔드포인트로 배포, 병렬 실행 제한 설정, 우리가 큐 관리 및 실행

### AI 기반 자동화

AI 에이전트를 활용해 어떤 웹사이트든 탐색, 상호작용, 데이터 추출을 할 수 있는 자동화를 구축하세요. AI를 상품으로 제공하든, 스크래핑과 워크플로우를 자동화하든 기본 제공되는 은폐 기능, 인증, 대규모 확장으로 배포할 수 있습니다.

**주요 기능:**
- **어떤 AI 브라우저 라이브러리든 호환** — Anthropic Computer Use, OpenAI CUA, Stagehand, Browser-use, Gemini Computer Use, Intuned/Browser SDK 등
- **코드와 AI 결합** — 결정론적 Playwright 로직과 AI 기반 단계를 단일 자동화에서 혼합
- **은폐 모드** — 탐지 회피, 프록시, CAPTCHA 해결 내장
- **기본 제공 인증** — create.ts와 check.ts (또는 .py) 작성, Intuned가 세션 생명주기 관리, AI로도 처리 가능
- **AI 사용량 제어** — 자동화 전체에서 AI 크레딧 사용량 모니터링 및 제한
- **확장 가능한 인프라** — API로 배포하거나 정기 작업 스케줄, 동시성 제어, 우리가 처리

### 관리형 스크래핑

수백 개 또는 수천 개의 스크래퍼가 필요한가요? Intuned Agent를 기반으로 한 솔루션 엔지니어 팀이 대신 구축하고 유지보수해줍니다. 지금까지 수천 개의 프로덕션 스크래퍼를 전달했습니다. 필요한 데이터와 사이트를 말씀해주면 프로덕션 레벨의 스크래퍼를 제공해드립니다.

**주요 기능:**
- **우리가 구축, 당신이 배포** — 사이트와 데이터 스키마만 알려주면 프로덕션 스크래퍼 완성
- **비용 효율적** — 솔루션 엔지니어 + Intuned Agent로 빠르고 저렴하게 구축
- **지속적인 유지보수** — 사이트가 변경되면 우리가 업데이트해서 계속 작동하도록 관리
- **당신의 코드, 종속성 없음** — 코드는 당신이 소유, 원할 때 셀프 서비스로 전환 가능, 재판매나 블랙박스 없음
- **유연한 가격** — 소스당 정액 또는 맞춤 계약
