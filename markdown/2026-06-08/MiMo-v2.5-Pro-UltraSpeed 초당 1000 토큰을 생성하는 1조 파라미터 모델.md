## 핵심 개념

> [!abstract]
> Xiaomi와 TileRT가 협력하여 1조 파라미터 모델에서 초당 1000 토큰(1000 tps) 생성 속도를 달성했습니다. FP4 양자화와 효율적인 추측적 디코딩(DFlash) 등 모델-시스템 코디자인을 통해 일반 GPU 8개 노드에서 이 성능을 구현했습니다.

## 상세 내용

- 1T 규모 모델에서 초당 1000 토큰 생성은 병렬 추론 경로 실행, 자동 검증과 같은 새로운 AI 애플리케이션 패러다임을 가능하게 함
- FP4 양자화와 DFlash 기반 추측적 디코딩으로 일반 상용 GPU에서 전문 하드웨어 수준의 성능 달성
- 6월 9-23일 제한된 기간 동안 신청 기반으로 UltraSpeed API 접근 제공 (가격은 표준 모델의 3배이지만 약 10배 빠른 속도)

> [!tip] 왜 중요한가
> 대규모 언어 모델의 극한 속도는 실시간 의사결정 루프와 고빈도 트레이딩, 의료 영상 분석 등 시간이 중요한 분야에서 AI의 실용성을 근본적으로 변화시킵니다.

## 전문 번역

# AI 추론 속도의 한계를 넘다: Xiaomi MiMo-V2.5-Pro-UltraSpeed 출시

인류는 예부터 속도를 추구해왔습니다. 내연기관의 시대 첫 레이싱카부터 음속을 깨뜨린 음향 폭발음까지, 우리의 속도에 대한 갈증은 DNA에 새겨져 있거든요. AI 추론 속도도 마찬가지입니다. 그것이 바로 지능의 경계를 결정하니까요.

모델이 충분히 빠르면 어떤 일이 일어날까요? 단순한 대기 도구에서 벗어나 당신의 사고 과정 자체가 됩니다. 실시간으로 응답하고, 순간적으로 반복하고, 마찰 없이 협업하게 되는 거죠.

오늘 우리는 기쁜 소식을 전합니다. Xiaomi와 TileRT가 함께 개발한 **MiMo-V2.5-Pro-UltraSpeed**를 출시했고, 1조 매개변수 규모의 모델에서 **초당 1000 토큰 이상의 생성 속도**를 세계 최초로 달성했습니다!

## 가격 대비 성능: 3배의 가격, 10배의 속도

MiMo-V2.5-Pro-UltraSpeed API는 제한된 시간 동안 특별 가격으로 제공됩니다.

- **가격**: MiMo-V2.5-Pro 대비 약 3배
- **성능**: 약 10배의 생성 속도
- **지원 형식**: API만 제공 (토큰 플랜 미지원)

## 접근 제한: 신청 기반 한정 운영

고속 추론 자원의 한계로 인해 UltraSpeed는 신청 심사 방식으로 운영됩니다.

**시험 운영 기간**: 2026년 6월 9일 ~ 6월 23일 23:59 (베이징 시간, UTC+8)

## 1000 tps가 변화시키는 것

1조 규모에서 초당 1000토큰을 달성한다는 건 단순히 더 빠른 타자기라는 의미를 넘어섭니다. 이는 AI 애플리케이션의 패러다임 자체를 바꿉니다.

**첫째, 속도가 곧 지능이 됩니다.** 이전까지는 어려운 문제에 마주쳤을 때 "한 번의 답변을 기다려야만" 했습니다. 하지만 이제는 같은 시간 내에 모델이 수십 개의 추론 경로를 병렬로 실행할 수 있습니다(Best-of-N, Tree Search). 백그라운드에서 자동으로 검증하고 자가 수정하면서 순수한 속도로 깊이 있는 사고를 만들어내는 거죠. 결과적으로 추론 품질이 근본적으로 향상됩니다.

**둘째, Coding Agent의 생산성 한계가 완전히 풀립니다.** 예전에는 AI로 코드를 작성하게 하면 개발자가 화면 앞에서 추론 지연으로 인해 고통받아야 했습니다. 초당 1000토큰에서는 코드 생성 속도와 생산 효율이 패러다임 수준의 도약을 이루게 됩니다.

**셋째, 그리고 가장 중요하게, 1조 모델이 실시간 의사결정 루프에 진입합니다.** 밀리초 단위의 "사고-응답" 사이클이 가능해지면서, 대규모 모델들이 시간이 중요한 상황에 즉시 투입될 수 있습니다. 고주파 정량적 거래 신호 생성, 즉각적인 부정거래 탐지, 지능형 입찰, 실시간 상호작용 대화 — 이 모든 게 가능해진다는 뜻입니다.

수술 보조와 의료 영상 분석처럼 생사가 갈리는 상황으로 내려가면 어떻게 될까요? 더 이상 AI 속도는 단순한 효율 지표가 아닙니다. 그것은 죽음과의 경쟁에서 의료진을 돕는 무기가 됩니다. 수술실에서 AI가 병변 분석과 위험 예측을 1초 빨리 완료하면, 외과의는 그만큼 더 많은 자유도를 갖게 됩니다. 이것이 우리를 확신하게 합니다 — 속도의 궁극적 의미는 단순한 생산성 향상이 아니라, 기술이 인류의 더 나은 삶을 돕도록 하는 것입니다.

## 기술적 성취: 모델과 시스템의 극단적 협력

초당 1000+ 토큰을 1조 규모 모델에서 달성한 것은 어느 한 기술의 돌파가 아닙니다. MiMo 모델팀과 TileRT 시스템팀의 깊은 협력과 극단적 코-설계(Codesign)의 결실입니다.

업계는 유사한 극한 속도를 위해 보통 특화된 하드웨어를 씁니다. Cerebras의 웨이퍼 스케일 통합이나 Groq의 온칩 SRAM 커스텀 아키텍처 같은 것이죠. 우리는 다른 길을 택했습니다: **일반 GPU만으로도 더 인상적인 추론 속도를 달성하는 것**, 모델과 시스템 코-설계를 통해서 말입니다.

**모델 레벨에서는:**

- **FP4 양자화**를 일반 하드웨어의 대역폭 병목에 맞게 적용했고, 모델 크기를 극적으로 축소하고 메모리 접근 오버헤드를 줄였습니다.
- **DFlash**라는 효율적인 추론 가속 방식을 도입했습니다. 이는 블록 레벨의 마스크된 병렬 예측 기반 추론으로, 검증 단계당 수용되는 토큰 길이를 대폭 늘렸습니다.

**시스템 레벨에서는:**

TileRT가 이러한 알고리즘의 동적 특성에 완벽하게 적응하면서, 새로운 양자화와 추론 가속 파이프라인에 최적화된 맞춤형 컴파일 엔진과 연산 커널을 제공했습니다.

이러한 극단적 코-설계를 통해, **단 8개 GPU 하나로 구성된 일반 서버에서 1조 규모 모델의 1000+ 토큰/초 출력을 달성했습니다.**

1조 규모에서 전통적인 8비트 양자화는 더 이상 선택지가 아닙니다. 여기서부터는 혁신적 기법들이 필요하고, 우리가 바로 그것을 만들어냈습니다.
