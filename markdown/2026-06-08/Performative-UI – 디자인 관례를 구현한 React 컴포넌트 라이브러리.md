## 핵심 개념

> [!abstract]
> Performative-UI는 일반적인 UI/UX 디자인 패턴과 관례를 React 컴포넌트로 구현한 라이브러리입니다. 로딩 애니메이션, 스켈레톤 스크린, 피드백 표시 등 사용자 경험을 향상시키는 시각적 요소들을 재사용 가능한 컴포넌트 형태로 제공합니다. 개발자가 처음부터 이러한 UI 요소를 만드는 대신 검증된 디자인 패턴을 바로 적용할 수 있도록 설계되었습니다.

## 상세 내용

- React 기반의 재사용 가능한 컴포넌트 라이브러리로, 웹 애플리케이션에서 자주 사용되는 UI 패턴들을 미리 구현하여 제공합니다.
- 로딩 상태, 에러 표시, 성공 피드백 등 일반적인 사용자 상호작용 상황에 대한 시각적 표현을 포함하고 있습니다.
- 스켈레톤 로더나 프로그레시브 로딩 같은 현대적인 UX 패턴을 적용하여 사용자 경험을 개선할 수 있습니다.
- 개발자가 디자인 결정을 반복할 필요 없이 업계 표준 관례를 따르는 컴포넌트를 즉시 사용할 수 있습니다.
- 공식 문서(https://vorpus.github.io/performativeUI/)에서 각 컴포넌트의 사용 예시와 시각적 데모를 확인할 수 있습니다.

> [!tip] 왜 중요한가
> 개발자가 일반적인 UI 패턴을 매번 새로 구현하는 시간을 절약하고, 검증된 디자인 관례를 일관되게 적용하여 사용자 경험을 표준화할 수 있습니다.
