## 핵심 개념

> [!abstract]
> Sherlock은 주어진 사용자명으로 여러 소셜 네트워크에서 계정을 찾아내는 Python 기반 도구이다. 300개 이상의 소셜 미디어 플랫폼을 지원하며 사용자 정보 조사에 활용될 수 있다.

## 상세 내용

- Python으로 작성된 오픈소스 도구로 사용자명 기반 소셜 계정 검색 기능 제공
- 다양한 소셜 네트워크 플랫폼을 병렬로 검색하여 빠른 결과 제공

> [!tip] 왜 중요한가
> 웹 스크래핑과 API 통합 기술을 학습할 수 있으며, 보안 조사 및 사용자 검증 도구로 활용 가능하다.
