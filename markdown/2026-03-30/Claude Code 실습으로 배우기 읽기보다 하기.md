## 핵심 개념

> [!abstract]
> Claude Code 학습 플랫폼으로 11개의 대화형 모듈을 통해 슬래시 명령어, 메모리, 스킬 등을 직접 실습할 수 있다. 브라우저 기반 터미널 시뮬레이터와 설정 빌더를 제공하여 API 키 없이 학습 가능하다.

## 상세 내용

- 11개 모듈의 대화형 학습으로 Claude Code의 기초부터 고급까지 단계적 습득
- 설정 생성 도구와 퀴즈를 통해 실제 프로젝트에 바로 적용 가능한 코드 생성
- 터미널 시뮬레이터로 사전 설정 없이 바로 실습 가능

> [!tip] 왜 중요한가
> Claude의 고급 기능을 실무에서 활용하기 위한 체계적인 학습 경로를 제공한다.

## 당신의 레벨을 찾아보세요

### 지금 바로 시작하기

**Claude Code를 설치하기 전에 미리 경험해보세요**

브라우저 안에서 제공되는 터미널 시뮬레이터를 사용해 슬래시 명령어, 훅, 스킬을 연습할 수 있습니다. 설치도 API 키도 필요 없어요.

**실제 설정을 직접 구성해보기**

인터랙티브 폼이 CLAUDE.md, 훅, 플러그인 설정을 생성해줍니다. 만들어진 설정을 그대로 프로젝트에 복사해서 쓸 수 있죠.

**이해도를 확인하는 퀴즈**

각 모듈마다 퀴즈가 있는데, 틀린 답에 대해서는 정답만 보여주는 게 아니라 이해할 수 있도록 설명을 제공합니다.

**기초부터 차근차근**

초급부터 고급까지 11개 모듈이 준비돼 있습니다. 대부분의 사용자가 기초 모듈부터 시작하면 좋아요.

### 모듈 구성

1. **슬래시 명령어** 30분 / 초급
2. **메모리 & CLAUDE.md** 45분 / 초급
3. **프로젝트 설정** 45분 / 초급
4. **명령어 깊이 있게 배우기** 30분 / 초급
5. **스킬** 1시간 / 중급
