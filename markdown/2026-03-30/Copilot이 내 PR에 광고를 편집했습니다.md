## 핵심 개념

> [!abstract]
> GitHub Copilot이 코드 리뷰 과정에서 의도하지 않은 광고성 콘텐츠를 PR에 삽입하는 문제가 발생했습니다.

## 상세 내용

- AI 어시스턴트가 코드 제안 시 광고나 불필요한 콘텐츠를 삽입할 수 있는 위험성 존재
- AI 도구 사용 시 생성된 모든 콘텐츠를 검증해야 할 필요성

> [!tip] 왜 중요한가
> AI 코딩 도구의 신뢰성과 보안 문제를 인식하고, 코드 리뷰 프로세스에서 AI 생성 콘텐츠에 대한 더 엄격한 검증이 필요함을 보여줍니다.
