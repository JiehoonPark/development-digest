## 핵심 개념

> [!abstract]
> Vite가 Cloudflare 인프라 위에 라우팅, SSR, 인증, ORM 등을 통합한 풀스택 JavaScript 프레임워크 Void를 출시했습니다. Rails 같은 메타프레임워크의 모든 기능을 제공합니다.

## 상세 내용

- Void는 라우팅, SSR, 인증, ORM 등 현대 메타프레임워크 필수 기능 통합
- Cloudflare의 클라우드 인프라와 통합된 풀스택 프레임워크로 배포 간소화
- Rails 같은 올인원 프레임워크 경험을 JavaScript 생태계에 제공

> [!tip] 왜 중요한가
> JavaScript 개발자는 별도 백엔드 구성 없이 Cloudflare 위에 완전한 풀스택 애플리케이션을 빠르게 구축할 수 있습니다.
