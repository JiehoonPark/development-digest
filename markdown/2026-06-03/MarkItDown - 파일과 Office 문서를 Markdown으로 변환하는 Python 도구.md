## 핵심 개념

> [!abstract]
> Microsoft의 MarkItDown은 다양한 파일 형식(Office 문서, PDF, 이미지 등)을 Markdown 형식으로 변환하는 Python 유틸리티입니다. LLM 모델에 전달하거나 문서화 작업에 활용하기 위해 파일 콘텐츠를 텍스트 기반으로 변환할 때 유용합니다.

## 상세 내용

- 광범위한 파일 형식 지원: Word, Excel, PowerPoint, PDF 등 주요 Office 문서와 이미지, 웹페이지 등 다양한 형식을 Markdown으로 변환합니다.
- LLM 통합 친화적: 변환된 Markdown 형식을 대언어모델(LLM)에 입력하기 용이하여 문서 분석, 요약, 질의응답 작업에 활용할 수 있습니다.

> [!tip] 왜 중요한가
> 개발자가 다양한 문서 형식을 손쉽게 텍스트 기반으로 변환하여 LLM 기반 애플리케이션 개발과 문서 자동화 작업의 진입장벽을 낮춥니다.
