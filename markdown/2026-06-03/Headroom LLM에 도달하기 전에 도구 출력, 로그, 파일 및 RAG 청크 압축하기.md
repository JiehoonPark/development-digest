## 핵심 개념

> [!abstract]
> Headroom은 도구 출력, 로그, 파일, RAG 청크를 LLM에 전송하기 전에 압축하여 토큰 사용량을 60-95% 줄이는 라이브러리, 프록시, MCP 서버입니다.

## 상세 내용

- 컨텍스트 압축으로 같은 품질의 답변 유지하면서 토큰 사용량 60-95% 감소
- 라이브러리, 프록시, MCP 서버 등 다양한 배포 형태 지원으로 기존 시스템에 유연하게 통합 가능

> [!tip] 왜 중요한가
> LLM API 비용을 절감하고 토큰 한계 내에서 더 많은 컨텍스트를 활용할 수 있습니다.
