## 핵심 개념

> [!abstract]
> Aqua Security의 Trivy는 컨테이너, Kubernetes, 코드 저장소, 클라우드 환경에서 보안 취약점, 설정 오류, 노출된 비밀 정보, 소프트웨어 부품 명세서(SBOM)를 종합적으로 스캔하는 오픈소스 보안 도구입니다.

## 상세 내용

- 다층 보안 스캔: 컨테이너 이미지, Kubernetes 설정, 소스 코드, 클라우드 자원에 걸쳐 일관된 취약점 검사를 수행하는 통합 도구입니다.
- 개발 파이프라인 통합: CI/CD 파이프라인에 쉽게 통합되어 배포 전 보안 검증을 자동화할 수 있습니다.

> [!tip] 왜 중요한가
> 개발자가 코드 작성부터 클라우드 배포까지 전 단계에서 보안 취약점을 조기에 발견하고 수정하여 프로덕션 환경의 보안 침해 위험을 크게 줄일 수 있습니다.
