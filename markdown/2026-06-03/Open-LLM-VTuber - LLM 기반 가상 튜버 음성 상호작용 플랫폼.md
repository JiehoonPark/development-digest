> [!abstract]
> Open-LLM-VTuber는 LLM과 음성 상호작용, 음성 중단, Live2D 페이스 트래킹을 로컬에서 크로스플랫폼으로 지원합니다.

- 핸즈프리 음성 인터페이스로 자연스러운 대화 가능
- Live2D 얼굴 추적으로 실시간 애니메이션 생성
