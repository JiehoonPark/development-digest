1960년대 UNIVAC 컴퓨터에서 Minecraft 서버 등을 실행하기

Running a Minecraft Server and More on a 1960s Univac Computer

📅 2026-04-21📰 Hacker News (Top)
general

요약

1960년대 UNIVAC 1219B 컴퓨터(250kHz, 90KB RAM)에서 Minecraft 서버, NES 에뮬레이터, OCaml, 웹서버 등을 성공적으로 실행했습니다. RISC-V 에뮬레이터와 LLVM을 활용하여 8개월의 개발 기간을 거쳐 완성되었습니다.

핵심 포인트

  • 18비트 워드, ones' complement 산술, 제한된 레지스터와 뱅크 메모리 등 극도로 이질적인 아키텍처 극복
  • Rust 기반 UNIVAC 어셈블러/에뮬레이터 개발 후 RISC-V 에뮬레이터와 LLVM 통합으로 C 컴파일 가능
  • 250kHz 컴퓨터에서 복잡한 프로그램 실행으로 극한 환경에서의 코드 최적화 사례 제시

왜 중요한가

💡
극도로 제한된 하드웨어에서 현대 소프트웨어를 실행하기 위한 창의적인 엔지니어링 접근법을 보여줍니다.
