RuView: WiFi 신호로 실시간 인체 자세 추정 (Rust)

ruvnet/RuView (Rust)

📅 2026-04-19 📰 GitHub Trending ⬆ 47.5k

요약

RuView는 WiFi 신호를 활용해 비디오 없이 실시간 인체 자세 추정, 생체신호 모니터링, 존재 감지를 수행하는 기술이다.

핵심 포인트

  • 카메라나 영상 데이터 없이 WiFi 신호만으로 인체 자세와 생체신호를 감지하는 무시야선 기술을 제공한다.
  • 프라이버시 보호와 저비용 구현이 가능한 대안 기술이다.

왜 중요한가

스마트홈, 보안 감시, 건강 모니터링 등 다양한 응용 분야에서 프라이버시 친화적인 센싱 솔루션을 제공한다.
