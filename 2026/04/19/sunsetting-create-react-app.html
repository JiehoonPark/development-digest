요약

React 팀이 새로운 앱 개발을 위해 Create React App을 공식 지원 중단하고, 기존 앱들은 프레임워크(Next.js 등) 또는 Vite, Parcel, RSBuild 같은 빌드 도구로 마이그레이션할 것을 권장하고 있습니다.

핵심 포인트

  • 새 프로젝트의 Create React App 사용 중단 공식 발표
  • Vite, Parcel, RSBuild 등 현대적 빌드 도구로의 마이그레이션 권장
  • Next.js 등의 프레임워크 채택을 우선하는 방향으로 전환

왜 중요한가

React 생태계의 공식 권장사항 변화로, 새로운 프로젝트 시작 시 빌드 도구 선택 기준을 조정해야 합니다.
