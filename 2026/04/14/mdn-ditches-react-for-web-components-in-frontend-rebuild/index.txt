<h3>흥미로운 기술 글과 영상</h3>
<p><strong>Firefox 확장 프로그램 전수 조사</strong></p>
<p>개발자가 JavaScript로 Firefox 확장 프로그램 생태계를 탐험한 이야기인데, 정말 재미있습니다. 개발자를 위한 '이상한 나라의 앨리스' 같은 느낌이에요. 이런 식의 깊이 있는 탐사 글이 더 많아지면 좋겠습니다.</p>
<p><strong>중첩된 Promise의 쓸모</strong></p>
<p>James가 2013년 Promises/A+ 논쟁을 다시 꺼냈는데, 실제로 마주친 동시성 문제 때문에 생각이 바뀌었다고 합니다. 난이도가 높지만 읽을 가치가 충분합니다.</p>
<p><strong>Promise는 취소할 수 없다 (가끔은 가능하지만)</strong></p>
<p>Promise를 완전히 취소할 순 없지만, async 함수가 영원히 해결되지 않는 Promise를 기다리게 만들면 함수 실행을 멈출 수 있습니다. 함수는 조용히 멈추고 가비지 컬렉션이 뒷정리를 합니다.</p>
<p><strong>Parse, Don't Validate (TypeScript로 유효성 검증 증명하기)</strong></p>
<p>같은 검증 코드를 여러 파일에 반복 작성하느라 고민이신가요? Branded types와 discriminated unions을 사용하면 TypeScript가 그 증명을 대신해줍니다.</p>
<p><strong>Intl API: 당신이 모르는 최고의 브라우저 API</strong></p>
<p>코드 예제가 풍부한 Intl API 입문서입니다. 다국어 처리에 정말 유용하거든요.</p>
