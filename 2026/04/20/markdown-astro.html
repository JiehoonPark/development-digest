Markdown + Astro = ❤️

📅 2026-04-20 📰 CSS-Tricks

요약

Astro의 기본 Markdown 지원을 MDX와 함께 활용하여 더 나은 경험을 제공하는 방법을 설명합니다. Markdown 기능을 향상시키는 실용적인 접근 방식을 제시합니다.

핵심 포인트

  • Astro의 내장 .md 지원만으로도 충분하지만, MDX 활용으로 더 풍부한 콘텐츠 작성 가능
  • MDX를 통해 Markdown 파일 내에 React 컴포넌트를 직접 포함시켜 상호작용성 향상

왜 중요한가

Astro 사용자가 Markdown 기반 콘텐츠를 더욱 동적이고 대화형으로 만들어 사용자 경험을 개선할 수 있습니다.
