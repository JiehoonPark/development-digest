Claude Opus 4.7 공개

Claude Opus 4.7

📅 2026-04-16📰 Hacker News (Top)⬆ 1.4k
ai

요약

Anthropic이 고급 소프트웨어 엔지니어링에서 주목할 만한 개선을 이룬 Claude Opus 4.7을 일반 공개했다. 이 모델은 복잡한 장기 실행 작업을 정밀하게 처리하고, 자체 출력을 검증하는 능력을 보여주며, 이전에 밀접한 감시가 필요했던 어려운 코딩 작업을 자신감 있게 수행할 수 있다. 더 높은 해상도의 이미지 처리, 전문적 작업의 품질 향상, 여러 벤치마크에서 Opus 4.6보다 우수한 성능을 보이며, 사이버 보안 사용 사례에 대해서는 차등적으로 기능을 제한하는 안전장치가 포함되어 있다. 가격은 Opus 4.6과 동일하게 입력 토큰 100만 개당 $5, 출력 토큰 100만 개당 $25이다.

핵심 포인트

  • 소프트웨어 엔지니어링 성능 향상: Opus 4.7은 복잡한 장기 실행 작업에서 엄밀성과 일관성을 보여주며, 정확한 지침 이행과 자체 출력 검증 능력을 갖추고 있다. 사용자들이 이전에는 밀접한 감시가 필요했던 가장 어려운 코딩 작업을 Opus 4.7에 자신감 있게 위임할 수 있다고 보고했다.
  • 멀티모달 비전 능력 대폭 개선: 더 높은 해상도의 이미지를 처리할 수 있으며, 전문적인 작업에서 더 세련되고 창의적이어서 고품질의 인터페이스, 슬라이드, 문서를 생성한다.
  • 벤치마크 성능: Claude Mythos Preview보다는 덜 강력하지만 Opus 4.6을 여러 벤치마크에서 초과했다. 개발자들의 코딩 벤치마크(93개 작업)에서 Opus 4.6 대비 13% 해상도 향상을 기록했고, 4개의 작업에서는 Opus 4.6과 Sonnet 4.6 모두 풀지 못한 문제를 해결했다.
  • 다단계 작업 효율성: 연구 에이전트 벤치마크에서 가장 강력한 효율성 기준선을 보였으며, 6개 모듈 중 최고 종합 점수인 0.715를 기록했다. 금융(General Finance) 모듈에서 Opus 4.6 대비 0.767에서 0.813으로 향상되었고, 장기 컨텍스트 성능이 가장 일관성 있었다.
  • 사이버 보안 기능 제한 및 검증 프로그램: Claude Mythos Preview만큼 사이버 기능이 고급이 아니며, 훈련 중 차등적으로 이 기능을 감소시키는 실험을 진행했다. 금지된 고위험 사이버 보안 사용을 감지하고 차단하는 안전장치가 자동으로 포함되며, 합법적인 보안 연구와 침투 테스트를 위해서는 새로운 사이버 검증 프로그램에 참여할 수 있다.
  • 지연 시간과 지시 이행: 빠른 중앙값 지연 시간과 엄격한 명령 이행으로 복잡한 장기 실행 코딩 워크플로우에 특히 의미 있으며, 개발자들이 다단계 작업의 마찰을 줄여 개발 흐름에 집중할 수 있도록 한다.
  • 출시 및 접근성: 모든 Claude 제품, API, Amazon Bedrock, Google Cloud의 Vertex AI, Microsoft Foundry에서 사용 가능하며, Claude API를 통해 claude-opus-4-7로 접근할 수 있다.

왜 중요한가

💡
복잡한 소프트웨어 엔지니어링 작업과 다단계 워크플로우를 자동화해야 하는 개발자들에게 이전 모델보다 높은 정확도와 효율성으로 생산성을 크게 향상시킬 수 있는 도구를 제공한다.

Claude Opus 4.7 출시 안내

2026년 4월 16일

Claude의 최신 모델인 Opus 4.7이 이제 일반 공개됩니다.

성능 개선 사항

Opus 4.7은 Opus 4.6 대비 소프트웨어 엔지니어링 분야에서 눈에 띄는 성능 향상을 이루었습니다. 특히 난제 해결에서 큰 진전을 보였는데요. 개발자들은 이제 예전에는 세밀한 감시가 필요했던 어려운 코딩 작업을 Opus 4.7에 자신감 있게 맡길 수 있습니다.

복잡하고 오래 걸리는 작업을 일관성 있게 처리하며, 지시사항을 정확히 따르고, 결과물을 자체 검증한 뒤 보고하는 방식으로 작동합니다.

비전 성능도 대폭 향상되었습니다. 더 높은 해상도의 이미지를 처리할 수 있으며, 전문적인 작업—슬라이드, 문서, UI 디자인—에서 더 세련되고 창의적인 결과물을 만들어냅니다.

사이버보안 안내사항

지난주 Project Glasswing을 통해 AI 모델의 사이버보안 위험성과 이점을 공개했습니다. 당시 Claude Mythos Preview의 출시를 제한하고, 더 능력이 낮은 모델부터 새로운 사이버 보안 조치를 테스트하겠다고 밝혔는데요. Opus 4.7이 바로 그 첫 번째 모델입니다.

Opus 4.7의 사이버 능력은 Mythos Preview보다 낮게 조정되었습니다. 학습 과정에서 의도적으로 이러한 능력을 차별적으로 제한하는 실험을 진행했거든요. Opus 4.7은 금지되거나 고위험 사이버보안 용도로 보이는 요청을 자동으로 감지하고 차단하는 보안 조치와 함께 공개됩니다.

정당한 사이버보안 목적(취약점 연구, 침투 테스트, 레드팀 활동)으로 Opus 4.7을 사용하고 싶은 보안 전문가들은 새로운 Cyber Verification Program 참여를 권장합니다.

출시 및 가격

Opus 4.7은 모든 Claude 제품과 API, Amazon Bedrock, Google Cloud의 Vertex AI, Microsoft Foundry에서 사용 가능합니다. 가격은 Opus 4.6과 동일하게 유지됩니다.

  • 입력 토큰: 100만 개당 $5
  • 출력 토큰: 100만 개당 $25

Claude API를 통해서는 claude-opus-4-7로 사용할 수 있습니다.

얼리 액세스 테스터들의 평가

금융 기술 플랫폼

> Opus 4.7로 개발자들이 비약적인 도약을 이룰 수 있을 거라 예상됩니다. 계획 단계에서 스스로 논리적 오류를 포착하고, 이전 Claude 모델들보다 훨씬 빠르게 실행을 가속화합니다. 수백만 명의 소비자와 기업을 대규모로 서빙하는 금융기술 플랫폼으로서, 이런 속도와 정확성의 조합은 게임 체인저가 될 수 있습니다. 고객들이 믿고 의존하는 금융 솔루션을 더 빠르게 제공할 수 있게 되니까요.

엔지니어링 팀

> Anthropic은 이미 코딩 모델의 표준을 세웠고, Opus 4.7은 현존 최고 성능 모델로서 그 기준을 한층 높입니다. 내부 평가에서 눈에 띄는 점은 단순한 성능만이 아닙니다. 실제 비동기 워크플로우—자동화, CI/CD, 장기 실행 작업—를 얼마나 잘 처리하는지가 중요했거든요. 또한 문제를 더 깊이 있게 생각하고 사용자의 요청에 무조건 동의하지 않고 자신의 관점을 제시합니다.

데이터 분석 플랫폼

> Hex가 평가한 모델 중 가장 강력합니다. 그럴듯하지만 틀린 대체값을 제시하는 대신 데이터가 누락되었다고 올바르게 보고하며, Opus 4.6도 빠지던 데이터 충돌 함정을 회피합니다. Opus 4.7은 더 지능적이고 효율적인 Opus 4.6입니다. 저 정도의 노력으로 Opus 4.7이 중간 정도의 노력을 들인 Opus 4.6 수준의 결과를 냅니다.

코딩 워크플로우 전문가

> 93개 과제 코딩 벤치마크에서 Opus 4.7은 Opus 4.6 대비 13% 향상을 달성했습니다. 이 중 Opus 4.6과 Sonnet 4.6 모두 풀지 못한 과제가 4개 포함되어 있습니다. 더 빠른 중앙값 지연 시간과 엄격한 지시사항 준수와 결합하면, 복잡하고 장기 실행되는 코딩 워크플로우에서 특히 의미 있습니다. 다단계 작업의 마찰을 줄여서 개발자들이 집중력을 유지하고 빌딩에만 집중할 수 있게 합니다.

연구 에이전트 평가

> 내부 연구 에이전트 벤치마크를 바탕으로, Opus 4.7은 다단계 작업을 위한 가장 강력한 효율성 기준을 보입니다. 6개 모듈 전체에서 0.715점으로 최고 점수와 동점이며, 테스트한 모든 모델 중 가장 일관된 장문맥 성능을 제공합니다. 금융 분야 최대 모듈인 General Finance에서도 Opus 4.6(0.767)에서 0.813으로 의미 있는 개선을 이루었으며, 데이터 공시와 규율 면에서도 가장 우수합니다. 추론 논리 분야에서 Opus 4.6이 어려움을 겪었다면, Opus 4.7은 견고합니다.

AI 에이전트 관점

> Claude Opus 4.7은 모델이 조사하고 작업을 완료할 수 있는 범위를 확장합니다. Anthropic이 장기 지속적 추론에 최적화했다는 것이 명확히 드러나며, 시장 최고 성능으로 증명됩니다. 엔지니어들이 에이전트와 1:1 작업에서 여러 에이전트를 병렬로 관리하는 방식으로 전환하면서, 이런 프론티어 수준의 능력이 새로운 워크플로우를 열어줍니다.

Opus 4.7의 멀티모달 이해 능력도 큰 폭으로 개선되었습니다. 화학 구조식 해석부터 시작해 다양한 시각 자료 처리에서 눈에 띄는 진전을 보이고 있습니다.
