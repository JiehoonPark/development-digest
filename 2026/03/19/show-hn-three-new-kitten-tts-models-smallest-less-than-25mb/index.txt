14:T1d88,<h2>Kitten TTS</h2>
<p><strong>새소식:</strong> Kitten TTS v0.8 출시 -- 1,500만, 4,000만, 8,000만 파라미터 모델이 이제 사용 가능합니다.</p>
<p>Kitten TTS는 ONNX 기반의 오픈소스 경량 음성 합성 라이브러리입니다. 1,500만부터 8,000만 파라미터 범위(디스크 용량 25~80 MB)의 모델을 제공하며, GPU 없이 CPU만으로도 고품질 음성 합성을 구현할 수 있습니다.</p>
<p><strong>현재 상태:</strong> 개발자 프리뷰 -- 릴리스 사이에 API가 변경될 수 있습니다.</p>
<p>상용 지원을 제공합니다. 통합 지원, 커스텀 음성, 엔터프라이즈 라이선스가 필요하다면 저희에게 연락해주세요.</p>

<h3>특징</h3>
<p><strong>초경량</strong> – 25 MB(int8)부터 80 MB까지의 모델 크기로 엣지 배포에 최적화되어 있습니다.</p>
<p><strong>CPU 최적화</strong> – ONNX 기반 추론이 GPU 없이도 효율적으로 동작합니다.</p>
<p><strong>8가지 내장 음성</strong> – Bella, Jasper, Luna, Bruno, Rosie, Hugo, Kiki, Leo를 지원합니다.</p>
<p><strong>조절 가능한 음성 속도</strong> – speed 파라미터로 재생 속도를 제어할 수 있습니다.</p>
<p><strong>텍스트 전처리</strong> – 숫자, 통화, 단위 등을 자동으로 처리하는 파이프라인이 내장되어 있습니다.</p>
<p><strong>24 kHz 출력</strong> – 표준 샘플레이트의 고품질 오디오를 생성합니다.</p>
<h3>사용 가능한 모델</h3>
<div class="table-wrapper"><table><thead><tr><th>모델</th><th>파라미터</th><th>크기</th><th>다운로드</th></tr></thead><tbody><tr><td>kitten-tts-mini</td><td>80M</td><td>80 MB</td><td>KittenML/kitten-tts-mini-0.8</td></tr><tr><td>kitten-tts-micro</td><td>40M</td><td>41 MB</td><td>KittenML/kitten-tts-micro-0.8</td></tr><tr><td>kitten-tts-nano</td><td>15M</td><td>56 MB</td><td>KittenML/kitten-tts-nano-0.8</td></tr><tr><td>kitten-tts-nano (int8)</td><td>15M</td><td>25 MB</td><td>KittenML/kitten-tts-nano-0.8-int8</td></tr></tbody></table></div>
<p><strong>참고:</strong> kitten-tts-nano-0.8-int8 모델에서 문제를 보고한 사용자들이 있습니다. 문제가 발생하면 이슈를 열어주세요.</p>
<h3>데모</h3>
<p>브라우저에서 직접 Kitten TTS를 체험해보세요. Hugging Face Spaces에서 온라인으로 시도할 수 있습니다.</p>
<h3>빠른 시작</h3>
<h4>사전 요구사항</h4>
<ul>
<li>Python 3.8 이상</li>
<li>pip</li>
</ul>
<h4>설치</h4>
<pre><code class="language-bash">pip install https://github.com/KittenML/KittenTTS/releases/download/0.8.1/kittentts-0.8.1-py3-none-any.whl</code></pre>
<h4>기본 사용법</h4>
<pre><code class="language-python">from kittentts import KittenTTS

model = KittenTTS(&quot;KittenML/kitten-tts-mini-0.8&quot;)
audio = model.generate(&quot;This high-quality TTS model runs without a GPU.&quot;, voice=&quot;Jasper&quot;)

import soundfile as sf
sf.write(&quot;output.wav&quot;, audio, 24000)</code></pre>
<h4>고급 사용법</h4>
<pre><code class="language-python"># 음성 속도 조절 (기본값: 1.0)
audio = model.generate(&quot;Hello, world.&quot;, voice=&quot;Luna&quot;, speed=1.2)

# 파일로 직접 저장
model.generate_to_file(&quot;Hello, world.&quot;, &quot;output.wav&quot;, voice=&quot;Bruno&quot;, speed=0.9)

# 사용 가능한 음성 목록 조회
print(model.available_voices)
# ['Bella', 'Jasper', 'Luna', 'Bruno', 'Rosie', 'Hugo', 'Kiki', 'Leo']</code></pre>
<h3>API 레퍼런스</h3>
<h4>KittenTTS(model_name, cache_dir=None)</h4>
<p>Hugging Face Hub에서 모델을 로드합니다.</p>
<div class="table-wrapper"><table><thead><tr><th>파라미터</th><th>타입</th><th>기본값</th><th>설명</th></tr></thead><tbody><tr><td>model_name</td><td>str</td><td>&quot;KittenML/kitten-tts-nano-0.8&quot;</td><td>Hugging Face 리포지토리 ID</td></tr><tr><td>cache_dir</td><td>str</td><td>None</td><td>다운로드한 모델 파일을 캐싱할 로컬 디렉토리</td></tr></tbody></table></div>
<h4>model.generate(text, voice, speed, clean_text)</h4>
<p>텍스트로부터 음성을 합성하고 24 kHz 오디오 샘플의 NumPy 배열을 반환합니다.</p>
<div class="table-wrapper"><table><thead><tr><th>파라미터</th><th>타입</th><th>기본값</th><th>설명</th></tr></thead><tbody><tr><td>text</td><td>str</td><td>--</td><td>합성할 입력 텍스트</td></tr><tr><td>voice</td><td>str</td><td>&quot;expr-voice-5-m&quot;</td><td>음성 이름 (사용 가능한 음성 참조)</td></tr><tr><td>speed</td><td>float</td><td>1.0</td><td>음성 속도 배수</td></tr><tr><td>clean_text</td><td>bool</td><td>False</td><td>텍스트 전처리 여부 (숫자, 통화 등 확장)</td></tr></tbody></table></div>
<h4>model.generate_to_file(text, output_path, voice, speed, sample_rate, clean_text)</h4>
<p>텍스트로부터 음성을 합성하여 오디오 파일로 직접 저장합니다.</p>
<div class="table-wrapper"><table><thead><tr><th>파라미터</th><th>타입</th><th>기본값</th><th>설명</th></tr></thead><tbody><tr><td>text</td><td>str</td><td>--</td><td>합성할 입력 텍스트</td></tr><tr><td>output_path</td><td>str</td><td>--</td><td>오디오 파일을 저장할 경로</td></tr><tr><td>voice</td><td>str</td><td>&quot;expr-voice-5-m&quot;</td><td>음성 이름</td></tr><tr><td>speed</td><td>float</td><td>1.0</td><td>음성 속도 배수</td></tr><tr><td>sample_rate</td><td>int</td><td>24000</td><td>Hz 단위의 오디오 샘플레이트</td></tr><tr><td>clean_text</td><td>bool</td><td>True</td><td>텍스트 전처리 여부 (숫자, 통화 등 확장)</td></tr></tbody></table></div>
<h4>model.available_voices</h4>
<p>사용 가능한 음성 이름 목록을 반환합니다: ['Bella', 'Jasper', 'Luna', 'Bruno', 'Rosie', 'Hugo', 'Kiki', 'Leo']</p>
