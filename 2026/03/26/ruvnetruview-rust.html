RuView - WiFi 신호를 이용한 실시간 인체 포즈 추정 및 생체신호 모니터링

ruvnet/RuView (Rust)

📅 2026-03-26 📰 GitHub Trending ⬆ 43.1k

요약

RuView는 WiFi 신호를 활용하여 비디오 없이 실시간 인체 포즈 추정, 생체신호 모니터링, 존재감 감지를 수행하는 Rust 프로젝트입니다.

핵심 포인트

  • 카메라나 비디오 없이 WiFi DensePose 기술로 인체 포즈 추정 가능
  • 개인 정보 보호를 최우선으로 하면서도 실시간 생체신호 모니터링 구현

왜 중요한가

비전 기술 대신 무선 신호를 활용하는 대안을 제공하므로 개인 정보 보호가 중요한 응용 분야에 유용합니다.
