27:Tcfe,<h2>Cliamp - 레트로 감성의 터미널 음악 플레이어</h2>
<p>Winamp에서 영감을 받아 만든 터미널 기반 음악 플레이어입니다. 로컬 파일은 물론이고, 스트리밍, 팟캐스트, YouTube, SoundCloud, Bilibili, Spotify, 小宇宙, Navidrome 등 다양한 소스를 지원합니다. 게다가 스펙트럼 비주얼라이저, 파라메트릭 EQ, 플레이리스트 관리 기능까지 갖추고 있네요.</p>
<p>Bubbletea, Lip Gloss, Beep, go-librespot 같은 도구들로 구축되었습니다.</p>
<h3>라디오 스트리밍</h3>
<p>우리 라디오 채널을 즐길 수 있습니다:</p>
<pre><code>cliamp https://radio.cliamp.stream/lofi/stream.pls</code></pre>
<p>자신의 라디오 스테이션을 추가하고 싶다면 <code>~/.config/cliamp/radios.toml</code> 파일을 편집하면 됩니다. 자세한 내용은 <code>docs/configuration.md</code>를 참고하세요.</p>
<p>직접 라디오 서버를 운영하고 싶으신 분들은 cliamp-server를 확인해보세요.</p>
<h3>설치</h3>
<h4>curl로 설치</h4>
<pre><code class="language-bash">curl -fsSL https://raw.githubusercontent.com/bjarneo/cliamp/HEAD/install.sh | sh</code></pre>
<h4>Homebrew</h4>
<pre><code class="language-bash">brew install bjarneo/cliamp/cliamp</code></pre>
<h4>Arch Linux (AUR)</h4>
<pre><code class="language-bash">yay -S cliamp</code></pre>
<h4>미리 컴파일된 바이너리</h4>
<p>GitHub Releases에서 다운로드할 수 있습니다.</p>
<h4>소스코드에서 빌드</h4>
<pre><code class="language-bash">git clone https://github.com/bjarneo/cliamp.git &amp;&amp; cd cliamp &amp;&amp; go build -o cliamp .</code></pre>
<h3>빠른 시작</h3>
<p>디렉토리의 모든 음악을 재생하려면:</p>
<pre><code class="language-bash">cliamp ~/Music</code></pre>
<p>특정 파일들을 재생하려면:</p>
<pre><code class="language-bash">cliamp *.mp3 *.flac</code></pre>
<p>URL을 통해 스트리밍하려면:</p>
<pre><code class="language-bash">cliamp https://example.com/stream</code></pre>
<p><code>Ctrl+K</code>를 누르면 모든 키바인딩을 확인할 수 있습니다.</p>
