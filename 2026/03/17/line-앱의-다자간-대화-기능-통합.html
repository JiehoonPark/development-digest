LINE 앱의 다자간 대화 기능 통합

📅 2026-03-17 📰 LINE Engineering

요약

LINE의 임시 대화용 '여러 명과의 대화'와 장기 대화용 '그룹' 기능을 하나의 '그룹 대화'로 통합했습니다. 이중 읽기(dual read)를 활용한 점진적 마이그레이션과 초대 메커니즘 개선을 통해 사용성 향상과 서버 리소스 절약을 달성했습니다.

핵심 포인트

  • 기존에는 사용자가 기능 차이를 인식하지 못해 여러 명과의 대화에서 필요한 기능이 없으면 새로운 그룹을 만들었고, 동일 구성원의 중복 대화방이 15%까지 쌓여 있었습니다.
  • 이중 읽기 방식으로 구 API와 신규 API 호환성을 유지하며 점진적으로 전환했고, 배치 처리로 기존 그룹 데이터를 마이그레이션했습니다.
  • 통합 후 중복 대화방 비율이 15%에서 0.78%로 감소했으며, 모든 대화에서 앨범, 노트 등 모든 기능을 사용할 수 있게 되었습니다.

왜 중요한가

레거시 시스템의 기능 분산으로 인한 사용성 저하를 이중 읽기와 점진적 마이그레이션으로 해결한 사례로, 대규모 서비스 통합 시 데이터 손실 없이 무중단 전환하는 방법을 보여줍니다.
