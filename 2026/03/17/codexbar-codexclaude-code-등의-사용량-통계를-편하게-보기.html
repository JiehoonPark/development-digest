CodexBar - AI 코딩 도구 사용량 통계 앱

CodexBar - Codex/Claude Code 등의 사용량 통계를 편하게 보기

📅 2026-03-17 📰 GeekNews

요약

CodexBar는 macOS 메뉴 바 앱으로, Codex, Claude, Cursor, Gemini 등 16개 이상의 AI 코딩 도구의 사용량 한도와 리셋 시간을 한눈에 확인할 수 있다.

핵심 포인트

  • 16개 이상의 AI 코딩 도구 프로바이더 지원
  • 세션 및 주간 사용량 미터와 리셋 카운트다운을 메뉴 바에 표시
  • 여러 AI 도구 사용 시 한도 관리를 간편하게 지원

왜 중요한가

다양한 AI 코딩 도구를 사용하는 개발자가 API 사용량 한도를 효율적으로 추적하고 관리할 수 있다.
