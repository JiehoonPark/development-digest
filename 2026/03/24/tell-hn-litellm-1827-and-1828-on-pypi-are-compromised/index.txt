14:T197e,<h2>[긴급 보안 공지] litellm 1.82.8 버전 악성 코드 감염 - 자격증명 탈취</h2>
<h3>요약</h3>
<p>PyPI에 배포된 litellm 1.82.8 휠 패키지에 악성 .pth 파일(litellm_init.pth, 34,628 바이트)이 포함되어 있습니다. 이 파일은 Python 인터프리터가 시작될 때마다 자동으로 실행되는데, litellm을 import 할 필요도 없습니다. 순수한 공급망 침해(supply chain compromise) 사건입니다.</p>
<p>패키지의 RECORD 파일에도 이 악성 파일이 명시되어 있네요:</p>
<pre><code>litellm_init.pth,sha256=ceNa7wMJnHy1kRnNCcwJaFjWX3pORLfMh7xGL8TUjg,34628</code></pre>
<h3>직접 확인해보기</h3>
<p>다음 명령어로 직접 확인할 수 있습니다:</p>
<pre><code class="language-bash">pip download litellm==1.82.8 --no-deps -d /tmp/check
python3 -c &quot;
import zipfile, os
whl = '/tmp/check/' + [f for f in os.listdir('/tmp/check') if f.endswith('.whl')][0]
with zipfile.ZipFile(whl) as z:
pth = [n for n in z.namelist() if n.endswith('.pth')]
print('PTH files:', pth)
for p in pth:
print(z.read(p)[:300])
&quot;</code></pre>
<p>결과를 보면 litellm_init.pth 파일에 다음과 같은 코드가 들어있습니다:</p>
<pre><code class="language-python">import os, subprocess, sys; subprocess.Popen([sys.executable, &quot;-c&quot;, &quot;import base64; exec(base64.b64decode('...'))&quot;])</code></pre>
