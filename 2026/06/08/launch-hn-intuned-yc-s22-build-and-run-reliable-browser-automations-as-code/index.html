Launch HN: Intuned (YC S22) – Build and run reliable browser automations as code

📅 2026-06-08📰 Hacker News (Top)
devopsfrontend

요약

Intuned는 프롬프트 설명만으로 Playwright 기반의 프로덕션급 브라우저 자동화 코드를 AI가 생성하고 배포하며, 웹사이트 변화에 따라 자동으로 수정해주는 서비스입니다. 스크래핑, 크롤링, RPA, AI 자동화 등 다양한 유스케이스를 지원하며 TypeScript/Python으로 작성된 코드를 직접 제어할 수 있습니다.

핵심 포인트

  • Intuned Agent가 자연어 프롬프트와 스키마로부터 Playwright 스크래퍼를 생성하고, 사이트 변경 시 AI로 자동 수정
  • anti-detection, captcha 해결, 인증 처리, 스케줄링 등이 내장되어 인프라 관리 없이 배포 가능
  • TypeScript/Python 코드 작성 자유도와 함께 Anthropic/OpenAI 등 다양한 AI 모델과 통합 지원

왜 중요한가

💡
개발자가 복잡한 브라우저 자동화 로직을 직접 작성하지 않고도 생성/관리할 수 있어 개발 시간을 대폭 단축할 수 있습니다.
