IBM는 Microsoft가 Tab 키로 대화 상자 필드를 이동하는 것을 원하지 않았다

IBM didn't want Microsoft to use the Tab key to move between dialog fields

📅 2026-05-05📰 Hacker News (Top)
general

요약

Microsoft의 초기 개발 과정에서 IBM이 Tab 키를 사용하여 대화 상자의 필드 간 이동을 구현하는 것에 반대했던 역사적 에피소드를 다룬다.

핵심 포인트

  • IBM과 Microsoft 간의 기술 결정 과정에서 UI 상호작용 방식에 대한 의견 차이가 있었다.
  • 이는 초기 PC 시대의 기술 표준 수립 과정에서 벌어진 실제 사건이다.

왜 중요한가

💡
현재 당연하게 사용하는 UI/UX 패턴들이 역사적으로 어떻게 결정되었는지 이해하는 데 도움이 된다.
